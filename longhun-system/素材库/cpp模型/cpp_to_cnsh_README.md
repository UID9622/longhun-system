   - 代码开源，逻辑透明
   - 不做任何隐藏操作
   - 可审计、可验证

## 🚀 快速使用

### 编译编译器

```bash
g++ cpp_to_cnsh_compiler.cpp -o cpp_to_cnsh
```

### 使用编译器

```bash
./cpp_to_cnsh your_file.cpp
```

生成的CNSH文件会保存为 `your_file.cnsh`

---

## 📊 支持的转换

### 基本语法

| C++语法 | CNSH语法 |
|---------|----------|
| void functionName() | 定义 functionName 为 函数() |
| int x = 10; | 定义 x 为 整数 = 10 |
| double y = 3.14; | 定义 y 为 小数 = 3.14 |
| if (condition) | 如果 condition 那么 |
| else | 否则 |
| while (condition) | 当 condition |
| return value; | 返回 value |

### 数据类型

- `int` → 整数
- `double` / `float` → 小数
- `string` → 文本
- `bool` → 布尔
- `vector` → 列表

### 输入输出

- `cout << x` → 打印(x)
- `cin >> x` → 输入(x)

## 💡 使用示例

### 输入（C++代码）

```cpp
void calculateRisk() {
    double C = 0.75;
    double T = 0.40;
    double R = C * C - T;
    
    if (R > 0.1) {
        cout << "高风险" << endl;
    } else {
        cout << "低风险" << endl;
    }
    
    return;
}
```

### 输出（CNSH代码）

```cnsh
定义 calculateRisk 为 函数() {
    定义 C 为 小数 = 0.75
    定义 T 为 小数 = 0.40
    定义 R 为 小数 = C * C - T
    
    如果 R > 0.1 那么 {
        打印("高风险")
    } 否则 {
        打印("低风险")
    }
    
    返回
}
```

## ⚠️ 当前限制

**v1.0版本的已知限制：**

1. **复杂表达式**：可能需要手动调整
2. **模板代码**：暂不支持C++模板
3. **面向对象**：类和对象转换需要优化
4. **STL容器**：部分STL功能需要手动映射

**这些都可以改进，但v1.0版本已经可用。**

## 🔧 如何改进

如果你想改进这个工具，可以：

1. **扩展映射表**
   - 在 `cpp_to_cnsh` map中添加更多映射
   
2. **优化正则表达式**
   - 改进 `translateLine()` 函数
   
3. **添加语义分析**
   - 增强对复杂语句的识别
