# 🧬 数据结构

## 📌 一句话（初中能懂）
东西怎么摆放，才能好找好用

## 🧠 本质（f(x)=x 不动点）
「存」和「取」的平衡。再复杂的结构，本质都逃不掉这个。

## 🔧 最小实现（能跑）
```

data.append(4)  # 存
print(data[0])  # 取
```

数据量大→列表慢，需要排序→要换结构

# 🧬 哈希/加密

## 📌 一句话（初中能懂）
把内容变成指纹

## 🧠 本质（f(x)=x 不动点）
唯一映射。同一输入→同一输出，不同输入→基本不同输出。

## 🔧 最小实现（能跑）
```
import hashlib
data = "hello"
hash_value = hashlib.sha256(data.encode()).hexdigest()
print(hash_value)
```

## ⚠️ 边界（什么时候不能用）
哈希≠加密（不能还原），不能用于隐藏数据
