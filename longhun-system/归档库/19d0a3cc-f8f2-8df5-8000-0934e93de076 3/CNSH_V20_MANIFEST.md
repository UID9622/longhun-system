## 🚀 快速开始

### 1. 启动系统

```bash
# 一键启动
python3 start_cnsh_v20.py

# 或导入使用
from cnsh_master_v20 import CNSHMasterSystemV20
cnsh = CNSHMasterSystemV20()
cnsh.initialize_ancestor()
cnsh.start()
```

### 2. 部署社区

```bash
# 一键安装Discuz!插件
chmod +x install_cnsh_community.sh
./install_cnsh_community.sh
```

### 3. 启动API服务

```bash
cd longhun_api
chmod +x start.sh
./start.sh
```
