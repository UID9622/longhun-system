# 🐉 CNSH Vault

> iCloud Drive 知识库根目录

## 目录结构

```
vault/
├── inbox/          # 收件箱 - 新笔记入口
├── projects/       # 成品项目 - 已完成的项目
└── archive/        # 归档 - 历史资料
```

## 使用流程

1. **收集** → 在 inbox/ 创建新笔记
2. **整理** → 完善内容，添加标签
3. **归档** → 完成后移动到 projects/ 或 archive/

## DNA 编码

每篇笔记都有唯一的 DNA 追溯码：

```
#ZHUGEXIN⚡️-CNSH-NOTE-{序号}
```
