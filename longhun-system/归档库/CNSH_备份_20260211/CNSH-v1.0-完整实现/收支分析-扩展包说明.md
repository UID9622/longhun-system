## 五、代码结构说明

### 5.1 核心功能模块

```cnsh
# 模块1：数据输入
项目名称 = 「手工包子」
收入金额 = 500
手工成本 = 200
机器成本 = 50

# 模块2：自动计算
总成本 = 手工成本 + 机器成本
利润 = 收入金额 - 总成本
手工占比 = (手工成本 * 100) / 总成本
利润率 = (利润 * 100) / 收入金额

# 模块3：智能建议
如果【利润率 > 40】{
    打印「✅ 项目很赚钱」
    打印「💡 继续保持」
}

如果【手工占比 > 70】{
    打印「💪 靠手艺赚钱」
    打印「💡 可以定价高一些」
}
```
