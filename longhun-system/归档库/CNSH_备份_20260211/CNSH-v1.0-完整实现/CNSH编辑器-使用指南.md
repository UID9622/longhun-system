## 🚀 3步启动（无需编程）

### 步骤1：打开编辑器
直接双击 **`CNSH编辑器.html`** 文件，会自动用浏览器打开

**苹果用户：**

- 或者右键 → 打开方式 → Safari

### 步骤2：选择模板或开始编程

**方式1：使用模板（推荐新手）**
- 点击右侧的模板卡片
- 会自动加载示例代码

**方式2：自己编写**
- 在左侧编辑区直接输入代码
- 使用快速插入按钮帮助输入

1. 点击「🔄 编译C代码」按钮
2. 查看编译输出
3. 点击「💾 保存文件」下载 `.cnsh` 文件
4. 用命令行编译运行

## 📋 CNSH语法速查

### 基本语法

```cnsh
# 打印
打印「内容」

# 变量
整数 数字 = 123
文本 文字 = 「你好」

# 计算
整数 利润 = 售价 - 成本

# 条件语句
如果【利润 > 0】{
    打印「有利润」
}

# 循环
循环【5】{
    打印「循环执行」
}
```

### 函数语法

```cnsh
# 函数调用
打印(拼接(「收入：」, 转文本(123)))

# 注意：嵌套调用需要括号
打印(拼接(...))  # ✅ 正确
打印拼接(...)      # ❌ 错误
```

## ❓ 常见问题

### Q1：双击打不开？
**A：**
- 苹果电脑：用Safari浏览器打开
- 右键 → 打开方式 → Safari

### Q2：编译后如何运行？
**A：**
1. 保存 `.cnsh` 文件
2. 在终端运行：`bash 编译运行.sh 文件名.cnsh`

### Q3：代码有错误怎么办？
**A：**
- 检查括号是否匹配
- 检查是否使用了正确语法
- 查看编译输出区的错误提示

### Q4：快速插入按钮不工作？
**A：**
- 确保编辑器有焦点
- 点击后代码会插入到光标位置

### Q5：如何清空代码？
**A：**
- 点击「🗑️ 清空」按钮
- 会弹出确认对话框
