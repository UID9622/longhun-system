# 🇨🇳 CNSH一键运行使用说明

## ✅ 快速开始

### 方法1：使用脚本自动编译运行（推荐）

```bash
# 进入项目目录
cd "/Users/zuimeidedeyihan/Desktop/打包待命/CNSH 军人的编辑器/CNSH-v1.0-完整实现/"

# 运行hello.cnsh
bash 编译运行.sh hello.cnsh

# 运行个体户收支分析
bash 编译运行.sh 个体户收支分析.cnsh
```

### 方法2：手动编译运行

```bash
# 1. 编译CNSH代码为C代码
node cnsh-compiler.js hello.cnsh

# 2. 编译C代码为可执行文件
gcc hello.c -o hello

# 3. 运行程序
./hello
```

## ⚙️ 系统要求

- Node.js (运行编译器)
- gcc (编译C代码)

检查是否已安装：
```bash
node --version
gcc --version
```

## ❓ 常见问题

**Q: 提示 `command not found: node`**
- 安装Node.js：https://nodejs.org/

**Q: 提示 `command not found: gcc`**
- 安装Xcode命令行工具：`xcode-select --install`

**Q: 提示 `no such file or directory`**
- 确保文件名正确，区分大小写
- 使用tab键自动补全文件名
