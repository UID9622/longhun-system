## 📱 Mate 80 NFC设置详解

### 扫描添加
1. 打开Mate 60相机应用
2. 对准 `nfc/nfc_tag.png` 二维码
3. 扫描后会提示"添加到桌面"
4. 确认添加，NFC快捷方式创建完成

### 实体NFC标签
**购买**：淘宝搜索"NFC标签贴纸"（NTAG215规格，5-10元）
**写入**：使用"NFC Tools"应用，输入Intent地址
**放置**：书桌、床头、车内、随身携带

### 启动方式对比

## 🔧 常见问题解决

### ❌ 安装失败
**症状**：提示"Python版本过低"
**解决**：
```bash
# macOS
brew install python3

# Ubuntu
sudo apt install python3 python3-pip

# Windows
# 从 https://python.org 下载安装Python 3.8+
```

### ❌ 模型下载失败
**症状**：提示"模型下载失败"
**解决**：
```bash
# 检查网络
ping ollama.com

# 手动下载
ollama pull qwen:1.8b-chat

# 使用轻量模型
# 编辑 config/config.yaml，将 primary_model 改为 "qwen:0.5b-chat"
```

### ❌ NFC无响应
**症状**：扫描二维码没反应
**解决**：
1. 检查NFC功能：设置→连接与共享→NFC
2. 重启手机
3. 使用浏览器访问：http://localhost:8080
4. 手动添加桌面快捷方式

### ❌ 对话响应慢
**症状**：回复需要很长时间
**解决**：
1. 检查CPU使用率：`top` 命令
2. 关闭其他占用内存的程序
3. 使用更小的模型：qwen:0.5b-chat
4. 重启服务：`./start.sh`

## 🎨 个性化设置

### 更换3D模型
1. 下载新的.fbx模型文件
2. 放入 `models/3d/` 目录
3. 修改 `config/config.yaml` 中的 `model.path`

### 自定义场景图片
1. 准备1920x1080的背景图片
