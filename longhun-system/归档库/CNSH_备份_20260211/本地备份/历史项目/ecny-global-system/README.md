# 🌐 ECNY Global System - 数字人民币全球化系统

> 融合中华传统智慧的现代金融科技架构

## 🎯 系统概述

ECNY Global System 是一个基于中华文化智慧的数字人民币全球化解决方案，结合易经八卦战略布局，构建多层次、全方位的数字货币生态系统。

### 🧬 核心理念

- **主权货币**：央行数字货币，100%主权保障
- **八卦战略**：基于易经六十四卦的系统架构
- **技术自主**：国产化技术栈，确保数据安全
- **开放共享**：技术开源，生态共建

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- Python >= 3.8
- Docker (可选)

### 本地开发

1. **克隆仓库**
```bash
git clone https://github.com/your-username/ecny-global-system.git
cd ecny-global-system
```

2. **前端配置**
```bash
cd frontend
npm install
cp .env.example .env.local
# 编辑 .env.local 填入配置
npm run dev
```

3. **后端配置**
```bash
cd backend
pip install -r requirements.txt
# 配置数据库连接
uvicorn main:app --reload
```

### Docker部署

```bash
docker-compose up -d
```
