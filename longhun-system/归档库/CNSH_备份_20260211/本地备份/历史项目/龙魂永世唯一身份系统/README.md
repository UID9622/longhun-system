## 🔐 使用说明

### 生成龍魂ID

1. 运行主程序
2. 选择模式 `1. 生成新身份`
3. 输入身份证号
4. 输入国家代码（默认CN）
5. 系统自动生成龍魂ID
6. 选择是否导出证书
7. 可选择立即验证

### 验证龍魂ID

1. 运行主程序
2. 选择模式 `2. 验证现有身份`
3. 输入龍魂ID
4. 输入身份证号
5. 系统验证有效性

### 评估委员会

系统内置评估委员会，可处理身份争议：

```python
from core.龍魂评估委员会 import 龍魂评估委员会

委员会 = 龍魂评估委员会()

# 提交案例
案例 = {
    "case_id": "ARB-2025-001",
    "type": "身份盗用",
    "龍魂ID": "LONGHUN-CN-乾-坤-屯-蒙...",
    "public_consent": "完全公开"
}

# 生成评估报告
报告 = 委员会.生成报告(案例)
```
