# 龍魂·多平台同步中心

## 功能说明

自动同步你在各个平台的对话到本地数据库，统一DNA追溯，让CodeBuddy记住你的所有历史。

## 支持平台

- ✅ Notion（自动监听）
- ✅ 微信（监听导出文件）
- ✅ 飞书（监听导出文件）
- 🔄 Slack（待实现）

## 🎯 核心价值

### 数据主权
- 所有对话数据保存在本地SQLite数据库
- 平台被封？数据依然在本地
- 用户100%数据控制权

### DNA追溯
每条对话自动生成唯一DNA码：
```
#CONV-notion-UID9622-20251214-103045-ab12c34d
#CONV-wechat-UID9622-20251214-103045-bd56e78f
```

### 跨平台记忆
让CodeBuddy记住你所有平台的历史对话，形成"数字分身"。

## 🚀 快速开始

### 第1步：配置

编辑 `config/sync.env` 文件：
```bash
# Notion配置
NOTION_TOKEN=你的Notion集成Token
NOTION_DATABASE_ID=你的数据库ID

# 微信导出目录
WECHAT_EXPORT_DIR=~/Downloads/微信导出

# 飞书导出目录  
FEISHU_EXPORT_DIR=~/Downloads/飞书导出
```

### 第2步：启动

双击 `启动同步中心.command` 或运行：
```bash
cd ~/龍魂同步中枢
python3 sync_center.py
```

### 第3步：让CodeBuddy记住你

在CodeBuddy中输入：
```
/LU-LOAD-MEMORY-FILE ~/龍魂同步中枢/logs/记忆摘要_UID9622.md
```

## 📊 系统架构

```
平台层（微信/飞书/Notion）
    ↓ 自动抓取
同步中枢（监听API/导出文件）
    ↓ 统一格式  
本地聚合库（SQLite数据库）
    ↓ DNA锚定
记忆加载器（CodeBuddy随时调用）
    ↓
你的本地AI大脑（Ollama + CodeBuddy）
```

## 🔒 安全特性

### 本地存储
- 所有数据保存在本地SQLite数据库
- 备份到JSON文件，无需网络
- 完全私有，不依赖第三方云服务

### DNA追溯
- 每条对话生成唯一DNA码
- 跨平台内容关联识别
- 完整的数据溯源链条

### 自动备份
- 每小时自动备份数据库
- 保留30天备份历史
- 一键恢复功能
