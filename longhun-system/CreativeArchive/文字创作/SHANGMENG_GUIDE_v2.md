## 💡 核心功能详解

### 功能1：LER算法可视化

```yaml
什么是LER:
  LER = Logical Entropy Reduction
  逻辑熵减 = 从混乱到有序的度量

可视化效果:
  ✅ 全息粒子球实时动画
  ✅ 颜色变化：红（噪声）→ 黄（过渡）→ 金（信号）
  ✅ 数值显示：LER值实时更新
  ✅ DNA追溯码：每一步都可追溯

应用场景:
  - 信号检测
  - 数据分析
  - 算法演示
  - 技术讲解视频
```

## 🐛 常见问题

### Q1: 程序启动报错？

```yaml
错误: ModuleNotFoundError: No module named 'PyQt6'

解决:
  双击运行: install_shangmeng.bat
  或手动: pip install PyQt6
```

### Q2: 没有语音？

```yaml
原因:
  pyttsx3未安装或系统无中文语音

解决:
  1. pip install pyttsx3
  2. Windows: 自带中文语音（无需额外安装）
  3. 如果还是没声音，取消勾选"语音播报"
     （不影响主功能）
```

### Q3: 录制功能灰色不可用？

```yaml
原因:
  opencv或mss未安装

解决:
  pip install opencv-python mss

提示:
  如果安装失败，可以用OBS等第三方软件录制
```

### Q4: 录制的视频打不开？

```yaml
原因:
  可能是编码器问题

解决:
  用VLC播放器打开（支持所有格式）
  下载: https://www.videolan.org/
  
  或用格式工厂转换为标准MP4
```

### Q5: 想打包成exe？

```bash
# 安装pyinstaller
pip install pyinstaller

# 打包成单文件exe
pyinstaller --onefile --windowed shangmeng_console_v2_complete.py

# 生成的exe在dist/目录
# 可以发给别人直接运行（无需Python环境）
```
