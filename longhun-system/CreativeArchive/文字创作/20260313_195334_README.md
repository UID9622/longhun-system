## 用途

监控系统的 CPU 使用率、内存消耗量、磁盘占用率、工作区磁盘 I/O 性能，并在超过设定的阈值时在状态栏中高亮显示，低于阈值时会从状态栏消失。

## 设置项

- `resourcesExplorer.cpu.thresholds`: CPU 使用率的阈值。
- `resourcesExplorer.memory.thresholds`: 内存消耗的阈值。
- `resourcesExplorer.disk.thresholds`: 磁盘占用的阈值。
- `resourcesExplorer.updateFrequencyms`: 查询系统信息的频率。最小值为 5000 毫秒，以防止意外更新过快导致机器冻结。
- `resourcesExplorer.diskUpdateFrequencyms`: 查询磁盘信息的频率。最小值为 60000 毫秒。
