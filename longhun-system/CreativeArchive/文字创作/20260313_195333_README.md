修改品牌名和图标
- 在customLogo\customLogo_vscode_pangu_doer目录下
- 用待替换的图片替换custom-logo文件夹下的图片
其中custom-logo\activeBarLogo-external.png是插件侧边栏chat模式图标
custom-logo\agentActiveBarLogo.png是插件侧边栏agent模式图标
custom-logo\logo.png是chat模式主页上的小图标
custom-logo\posters.png是chat模式主页上的海报图标
custom-logo\logo_opaque.png是插件市场插件的图标
custom-logo\dark\login.woff是ide暗色主题下状态栏上登录状态的图标
custom-logo\dark\connecting.woff是ide暗色主题下状态栏上连接状态的图标
custom-logo\dark\logout.woff是ide暗色主题下状态栏上未登录状态的图标
custom-logo\light\login.woff是ide浅色主题下状态栏上登录状态的图标
custom-logo\light\connecting.woff是ide浅色主题下状态栏上连接状态的图标
custom-logo\light\logout.woff是ide浅色主题下状态栏上未登录状态的图标
- 本目录下，右键选择在终端打开，在终端执行（需自行修改路径和品牌名）
```
.\switchBrandName.bat ".\vscode-codebot-25.1.0.vsix" "C:\Program Files\7-Zip\7z.exe" HaiCode
```
-- 其中第二个参数是插件包名
-- 其中第三个参数是7z工具路径
-- 其中第四个参数是新的品牌名
