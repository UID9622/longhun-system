# CodeArts IDE Product Icons Theme

## Preview 预览

This extension replaces the CodeArts IDE default icon with the codearts icon.

此扩展将CodeArts IDE默认图标替换为codearts图标。

![image.png](https://bbs-img.huaweicloud.com/blogs/img/20250212/1739325011836400449.png)

## How to switch 如何切换

1. The product icons can be activated via command palette:

    产品图标可通过命令面板激活：

   ![image.png](https://bbs-img.huaweicloud.com/blogs/img/20250212/1739325082317167584.png)

   ![image.png](https://bbs-img.huaweicloud.com/blogs/img/20250212/1739325105768305730.png)

2. Click on the Settings menu

    单击“设置”菜单

    ![image.png](https://bbs-img.huaweicloud.com/blogs/img/20250212/1739325135400849929.png)
