## 🚀 使用方法

### 第1步：解压

```bash
# Linux/macOS
tar -xzf longhun-os-v1.0-universal.tar.gz
cd longhun-os-v1.0-universal

# Windows
使用7-Zip或WinRAR解压
进入解压后的目录
```

### 第2步：启动

```bash
# 方式1：一键启动（推荐）
./start.sh              # Linux/macOS
start.bat               # Windows

# 方式2：直接打开Web界面
双击 cnsh-editor-v2.html       # CNSH编程
双击 dual-auth-demo.html       # 双重认证
双击 longhun-os-console.html   # 统一控制台

# 方式3：运行Python系统
python3 longhun_police_system.py
python3 longhun_dual_auth.py
python3 longhun_os.py
```

### 第3步：查看文档

```bash
# 打开主文档
cat README.md

# 或使用文本编辑器打开
```
