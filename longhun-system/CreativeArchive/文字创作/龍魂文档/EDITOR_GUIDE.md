## 🚀 快速开始（3步）

### 第1步：打开编辑器

```bash
# 方法1：双击
# 直接双击 cnsh-editor.html

# 方法2：浏览器打开
firefox cnsh-editor.html
# 或
chrome cnsh-editor.html
```

### 第2步：编写代码

**方式1：从模板开始**
- 点击左侧边栏的模板（如"Hello World"）
- 代码自动插入到编辑器

**方式2：手动编写**
- 直接在编辑器中输入CNSH代码

### 第3步：编译运行

1. 点击顶部"⚡ 编译"按钮
2. 查看右侧"📝 编译日志"
3. 切换到"🔧 C代码"查看生成的C代码
4. 点击"💾 下载C代码"保存

## 🎯 核心功能详解

### 1. 📚 代码模板系统

**内置6个模板：**

```yaml
👋 Hello World:
  - 最简单的CNSH程序
  - 适合初学者

🔧 函数定义:
  - 展示函数定义和调用
  - 包含参数和返回值

🔀 条件判断:
  - if-else语句
  - 嵌套判断

🔄 循环语句:
  - 循环【N】语法
  - 重复执行代码

🔢 斐波那契:
  - 递归函数示例
  - 经典算法

📊 数组操作:
  - 模拟数组操作
  - 累加计算
```

**使用方法：**
- 点击左侧模板名称
- 代码自动插入到编辑器
- 可以直接运行或修改

### 2. 🧬 DNA追溯系统

**一键插入DNA追溯码：**

```cnsh
# 点击"🧬 插入DNA"按钮后自动生成：

# DNA追溯码：#龍芯⚡️2026-02-02-CNSH项目-v1.0

函数 主函数() ...
```

**DNA格式：**
```
#龍芯⚡️YYYY-MM-DD-项目名称-vX.X
```

**作用：**
- 标识代码创作者
- 记录创作时间
- 防止代码被盗用
- 可追溯源头

### 3. ⚡ 实时编译系统

**编译流程：**

```yaml
第1步 - 三色审计:
  检查代码内容安全性
  🟢 绿色 = 安全
  🟡 黄色 = 警告
  🔴 红色 = 阻断

第2步 - 词法分析:
  识别关键字、标识符、运算符
  生成Token流

第3步 - 语法分析:
  构建抽象语法树(AST)
  检查语法错误

第4步 - 代码生成:
  将CNSH转译为C代码
  保留中文变量名
```

**查看结果：**
- 右侧"📝 编译日志"显示过程
- 切换到"🔧 C代码"查看生成的C代码
- 切换到"🧬 DNA追溯"查看DNA信息

### 4. 💾 代码下载

**下载C代码：**
1. 编译成功后
2. 点击"💾 下载C代码"
3. 自动下载`output.c`文件
4. 可以用GCC编译运行

```bash
# 下载后编译
gcc output.c -o program
./program
```

### 5. 🌙 主题切换

**支持两种主题：**

```yaml
深色主题（默认）:
  - 护眼黑色背景
  - 金色高亮
  - 适合长时间编程

浅色主题:
  - 白色背景
  - 适合打印
  - 适合演示
```

**切换方法：**
- 点击左侧"🌙 切换主题"
- 或直接调用`toggleTheme()`

### 6. ⌨️ 快捷键支持

```yaml
Ctrl + S:  保存/下载C代码
Ctrl + K:  编译代码
Ctrl + L:  清空编辑器
Tab:       插入2个空格（缩进）
```
