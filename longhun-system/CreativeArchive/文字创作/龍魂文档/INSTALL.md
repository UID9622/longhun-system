# 🐉 龍魂操作系统 v1.0 - 安装说明

**龍魂操作系统 v1.0 支持所有 Linux/Windows/macOS/鸿蒙/iOS 设备 无需安装，解压即用 —— UID9622**

---

## ⚡ 3步安装（30秒完成）

### 第1步：下载

```bash
下载文件：longhun-os-v1.0-universal.tar.gz
大小：60KB
```

### 第2步：解压

**Linux/macOS:**
```bash
tar -xzf longhun-os-v1.0-universal.tar.gz
cd longhun-os-v1.0-universal
```

**Windows:**
```
使用7-Zip或WinRAR解压
双击进入目录
```

**鸿蒙/iOS:**
```
使用系统自带解压工具
进入解压后的目录
```

### 第3步：启动

**方式1：一键启动（推荐）**
```bash
# Linux/macOS
./start.sh

# Windows
start.bat

# 鸿蒙/iOS
双击打开 web/longhun-os-console.html
```

**方式2：直接打开Web界面**
```
双击打开以下任一文件：
- web/longhun-os-console.html (主控制台)
- web/cnsh-editor-v2.html (中文编程)
- web/dual-auth-demo.html (双重认证)
```

## 💡 常见问题

**Q: 需要安装Python吗？**
A: 不需要！Web界面（HTML文件）可以直接使用。只有运行Python核心系统时才需要Python。

**Q: 网络环境要求？**
A: Web界面完全离线可用，无需网络。Python系统也支持离线运行。

**Q: 如何卸载？**
A: 直接删除解压后的目录即可，不会留下任何残留文件。

**Q: 支持中文吗？**
A: 完全支持！所有界面和文档都是中文。
