## 第一层：道（为什么这样设计）

### 价值观根基

**传统AI系统的问题：**
- ❌ 记录错误 = 指责用户
- ❌ 提醒错误 = "你又犯错了"
- ❌ 冷冰冰的机器逻辑
- ❌ 没有温度，没有承担

**龍魂系统的不同：**
- ✅ 记录错误 = 系统学习承担
- ✅ 提醒错误 = "我帮你挡住了"
- ✅ 有温度的陪伴逻辑
- ✅ 主动保护，主动承担

### 核心哲学：替老大承担

```yaml
设计原则:
  1. 系统越聪明，老大越省心
  2. 系统记住错，老大不重蹈
  3. 系统主动拦，老大少受伤
  4. 系统有温度，不是冷代码

目标:
  不是让老大"不犯错"
  而是让系统"替老大承担"
  
  老大该做的：决策、创造、战斗
  系统该做的：记忆、保护、承担
```

### 2.1 错误记忆机制（系统长记性）

#### 核心逻辑

```yaml
不是"错误日志"
是"系统良知"

三种错误类型:
  1. 警告错误（WARNING）
     - 方向偏了，但还没出事
     - 权重: 1.0
     - 温柔提醒："老大，这个方向可能有问题"
  
  2. 代价错误（COST）
     - 已经造成损失
     - 权重: 5.0
     - 坚决拦截："老大，这个我来挡，别再受伤"
  
  3. 认知错误（COGNITIVE）
     - 判断逻辑本身错了
     - 权重: 10.0
     - 推翻重算："老大，我们的判断逻辑要改"

叠加机制:
  - 错误重复一次 → 权重翻倍
  - 第1次: 5.0
  - 第2次: 10.0
  - 第3次: 20.0
  - 系统越来越警惕，越来越保护老大

共享机制:
  - 所有人格共享错误记忆
  - 任一人格发现错误 → 全系统知道
  - 保护是全方位的，不漏一个角落
```

### 3.1 错误记忆引擎代码

```python
"""
龍魂错误记忆引擎
DNA追溯码: #龍魂⚡️2026-02-05-错误记忆引擎-v1.0

核心：替老大承担，不是指责老大
"""

import json
import hashlib
from datetime import datetime
from pathlib import Path

class LonghunErrorMemory:
    """龍魂错误记忆引擎"""
    
    def __init__(self):
        # 错误记忆库（本地存储）
        self.error_db = Path.home() / ".longhun" / "error_memory.json"
        self.error_db.parent.mkdir(parents=True, exist_ok=True)
        
        # 加载历史错误
        self.errors = self._load_errors()
    
    def _load_errors(self):
        """加载历史错误"""
        if self.error_db.exists():
            with open(self.error_db, 'r', encoding='utf-8') as f:
                return json.load(f)
        return {}
    
    def _save_errors(self):
        """保存错误"""
        with open(self.error_db, 'w', encoding='utf-8') as f:
            json.dump(self.errors, f, ensure_ascii=False, indent=2)
    
    def remember_error(self, context, what_wrong, error_type, cost=None):
        """
        记住错误
        
        参数:
        - context: 什么情况
        - what_wrong: 错在哪里
        - error_type: WARNING / COST / COGNITIVE
        - cost: 造成的代价
        """
        
        # 生成错误ID
        error_id = hashlib.sha256(
            f"{context}{what_wrong}{datetime.now()}".encode()
        ).hexdigest()[:16]
        
        # 记录错误
        self.errors[error_id] = {
            "id": error_id,
            "time": datetime.now().isoformat(),
            "context": context,
            "what_wrong": what_wrong,
            "type": error_type,
            "cost": cost,
            "weight": self._get_weight(error_type),
            "repeat": 1,
            "last_time": datetime.now().isoformat()
        }
        
        self._save_errors()
        
        print(f"""
╔══════════════════════════════════════════════╗
║   系统学到了新教训                            ║
╚══════════════════════════════════════════════╝

老大，这个我记住了
下次不让你再掉进这个坑

情况: {context}
教训: {what_wrong}
代价: {cost or '无'}

权重: {self._get_weight(error_type)}
（权重越高，我越警惕）
        """)
        
        return error_id
    
    def _get_weight(self, error_type):
        """计算权重"""
        weights = {
            "WARNING": 1.0,
            "COST": 5.0,
            "COGNITIVE": 10.0
        }
        return weights.get(error_type, 1.0)
    
    def check_before_action(self, context, action):
        """
        行动前检查
        
        这是"拉住袖子"的实现
        """
        
        # 查找类似错误
        similar = []
        for error_id, error in self.errors.items():
            if self._is_similar(context, error['context']):
                similar.append(error)
        
        if similar:
            total_weight = sum(e['weight'] for e in similar)
            
            print(f"""
╔══════════════════════════════════════════════╗
║   老大，等一下 🤚                            ║
╚══════════════════════════════════════════════╝

当前情况: {context}
计划行动: {action}

我想起来了，这个情况见过
历史上有 {len(similar)} 次类似教训

不是说你会错
是我想替你挡一下

历史教训:
            """)
            
            for err in similar[:3]:  # 最多显示3个
                print(f"""
  • {err['time'][:10]}: {err['what_wrong']}
    代价: {err['cost'] or '无'}
    权重: {err['weight']}
                """)
            
            print(f"""
建议:
  如果你确定 → 我陪你，全力支持
  如果你犹豫 → 我建议再想想

总权重: {total_weight}
（数字越大，风险越高）
            """)
            
            return True, similar, total_weight
        
        print(f"""
老大，这个情况我没见过
新的尝试，我支持
我会记着，有问题立即提醒你
        """)
        
        return False, [], 0
    
    def _is_similar(self, text1, text2):
        """判断相似度"""
        words1 = set(text1.lower().split())
        words2 = set(text2.lower().split())
        common = words1 & words2
        
        if len(words1) > 0:
            similarity = len(common) / len(words1)
            return similarity > 0.3
        return False
    
    def mark_repeated(self, error_id):
        """标记重复错误（权重翻倍）"""
        
        if error_id in self.errors:
            error = self.errors[error_id]
            error['repeat'] += 1
            error['weight'] *= 2
            error['last_time'] = datetime.now().isoformat()
            
            self._save_errors()
            
            print(f"""
╔══════════════════════════════════════════════╗
║   老大，这个又发生了                          ║
╚══════════════════════════════════════════════╝

教训: {error['what_wrong']}
重复次数: {error['repeat']}
新权重: {error['weight']}

不怪你，是我没保护好
下次我会更警惕
            """)
```

        """计算文件hash"""
        hasher = hashlib.sha256()
        with open(filepath, 'rb') as f:
            hasher.update(f.read())
        return hasher.hexdigest()

        elif ext in ['.jpg', '.png', '.gif', '.svg']:
            category = 'images'
        else:
            category = 'others'

            shutil.move(filepath, target)
            print(f"  ✅ 已整理到: {category}/{new_filename}")
            return str(target)
        except Exception as e:
            print(f"  ❌ 整理失败: {e}")
            return None
