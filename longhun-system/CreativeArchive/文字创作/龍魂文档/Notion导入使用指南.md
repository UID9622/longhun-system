### 方法1：直接复制粘贴（推荐）

**步骤1：打开文件**
```bash
打开：AI语义标准化系统-Notion完整版.md
```

**步骤2：全选复制**
```bash
Ctrl+A (全选)
Ctrl+C (复制)
```

**步骤3：粘贴到Notion**
```bash
1. 打开Notion
2. 创建新页面（Page）
3. Ctrl+V (粘贴)
4. 完成！
```

**耗时：** < 1分钟  
**Notion会自动识别：**
- Markdown标题（#）→ Notion标题
- 代码块（```）→ Notion代码块
- 表格（|）→ Notion表格
- YAML块 → Notion代码块

**步骤2：调整格式**
```
Notion会自动转换Markdown格式
如果有格式问题，手动微调即可
```

## 🚨 常见问题

### Q1：表格格式乱了怎么办？
```yaml
原因：Notion对Markdown表格支持有限

解决：
  方法1：手动调整表格列宽
  方法2：转换为Database
  方法3：使用Notion内置表格重建
```

### Q2：代码块没有语法高亮？
```yaml
原因：需要指定语言

解决：
  1. 点击代码块
  2. 选择语言（YAML/JSON/Bash等）
  3. 完成
```

### Q3：Toggle折叠不工作？
```yaml
原因：Markdown标题自动转为H1/H2，不是Toggle

解决：
  1. 选中标题和内容
  2. 点击右上角"⋮"
  3. 选择"Turn into toggle"
  4. 完成
```

### Q4：能不能直接导入为Notion模板？
```yaml
可以：
  1. 导入文档
  2. 调整格式
  3. 点击右上角"⋮"
  4. 选择"Duplicate as template"
  5. 保存为团队模板
```
