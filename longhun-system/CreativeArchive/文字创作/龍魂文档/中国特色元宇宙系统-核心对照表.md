```yaml
基础字段：
  - Name（场景名） → 命名遵循：山川河岳、州府县城
  - Type（类型） → 九州分野、五岳名山、江河湖海
  - BBox（边界） → 方圆几里、东西南北
  
中国特色字段（新增）：
  - 地理位置（Region）：华北、东北、华东、中南、西南、西北
  - 文化标签（Culture）：儒家、道家、法家、兵家、墨家
  - 历史朝代（Dynasty）：夏商周秦汉、唐宋元明清
  - 风水布局（FengShui）：龍脉、穴位、水口、明堂
  - 二十四节气（Solar Term）：影响天气和活动
```

### Object（物件）库·万物生灵
```yaml
基础字段：
  - Name → 器物命名：鼎、爵、觚、钟、磬

  - 材质（Material）：金木水火土五行
```
