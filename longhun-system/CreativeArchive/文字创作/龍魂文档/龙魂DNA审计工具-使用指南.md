## 📖 怎么用？（小白版·超详细）

### 第一步：下载工具

**方法1：复制代码**
1. 打开这个文件：`龍魂DNA审计工具.sh`
2. 全选复制（Cmd+A，Cmd+C）

4. 新建文件，粘贴（Cmd+N，Cmd+V）
5. 保存到桌面，文件名：`龍魂DNA审计工具.sh`
6. 格式选"纯文本"（不要选RTF）

**方法2：直接下载**
（如果有下载链接的话，直接下载）

### 第二步：让工具能运行

**Mac用户：**
1. 打开"终端"（应用程序 → 实用工具 → 终端）
2. 输入这个命令（复制粘贴）：
   ```bash
   chmod +x ~/Desktop/龍魂DNA审计工具.sh
   ```

**Windows用户：**

### 第三步：运行工具

**在终端输入：**
```bash
bash ~/Desktop/龍魂DNA审计工具.sh
```

**然后按回车，你会看到菜单：**
```
🐉 龍魂DNA审计工具 v1.0

2. 📝 生成DNA证书（给你的代码打标记）
3. 📋 查看我的DNA（看看我的设备信息）

请输入数字（1-4）：
```

---

### 第四步：选择功能

#### 功能1：检测代码（最常用）

**用途：** 检查一个代码文件有没有被篡改

**操作步骤：**
1. 输入 `1`，按回车
2. 把要检测的文件拖到终端窗口（会自动填入路径）
3. 按回车
4. 看结果：
   - **🟢 有DNA标记** = 这是龍魂系统的文件，正品
   - **🟡 没有DNA标记** = 不是龍魂文件，或者标记被删了

**示例：**
```
请输入要检测的文件路径（拖拽文件到这里）：[把文件拖进来]

文件名：我的代码.py
文件路径：/Users/lucky/Desktop/我的代码.py
文件大小：2.3 KB
文件哈希：abc123def456...

✅ 发现DNA追溯码：#龍芯⚡️2026-01-26-我的项目-v1.0

🟢 这个文件有龍魂DNA标记！
```

---

#### 功能2：生成DNA证书（给代码打标记）

**用途：** 给你自己的代码打上龍魂DNA标记

**操作步骤：**
1. 输入 `2`，按回车
2. 输入项目名称（例如：我的网站）
3. 工具会生成DNA证书，自动打开
4. 复制证书里的DNA追溯码
5. 粘贴到你的代码文件顶部

**示例：**
```
请输入项目名称（例如：我的项目）：我的网站

✅ DNA证书已生成！

📄 证书位置：/Users/lucky/龍魂DNA证书-我的网站.txt

🎯 下一步：
   1. 打开证书文件
   2. 复制DNA追溯码到你的代码文件
   3. 格式：# DNA追溯码：#龍芯⚡️2026-01-26-我的网站-v1.0
```

**然后在你的代码文件第一行加上：**
```python
# DNA追溯码：#龍芯⚡️2026-01-26-我的网站-v1.0

# 下面是你的代码
print("Hello World")
```

### 工作原理

```yaml
设备DNA生成：
  算法：SHA256(hostname + username + CPU架构 + 操作系统)
  特点：同一台设备DNA固定

文件哈希计算：
  算法：SHA256
  用途：检测文件是否被修改

DNA追溯码格式：
  格式：#龍芯⚡️YYYY-MM-DD-项目名-版本
  示例：#龍芯⚡️2026-01-26-龍魂系统-v1.0

检测逻辑：
  1. 读取文件内容
  2. 搜索 #龍芯⚡️ 标记
  3. 如果找到 → 有DNA
  4. 如果找不到 → 无DNA
  5. 计算文件哈希，用于对比
```
