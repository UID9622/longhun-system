## 三、部署步骤

### 步骤1: 准备环境

```bash
# 1. 安装Discuz! X3.5
# 下载: https://www.discuz.net/thread-3847642-1-1.html
# 按官方文档完成安装

# 2. 确认PHP扩展
php -m | grep -E "curl|json|pdo_mysql|mbstring|openssl"

# 3. 确认MySQL
mysql -u root -p -e "SELECT 1"
```

### 步骤2: 部署CNSH护盾

```bash
# 运行护盾安装脚本
bash install_cnsh_shield.sh

# 启动护盾
python3 /opt/cnsh_shield/cnsh_shield_v05_integrated.py
```

### 步骤3: 部署社区插件

```bash
# 运行社区部署脚本
bash install_cnsh_community.sh /var/www/discuz /opt/cnsh_shield discuz root "your-password"

# 参数说明:
# $1: Discuz!安装路径
# $2: CNSH护盾路径
# $3: 数据库名
# $4: 数据库用户名
# $5: 数据库密码
```

### 步骤4: 启用插件

1. 登录Discuz!后台
2. 进入 **应用 → 插件**
3. 找到 **CNSH-64社区系统**
4. 点击 **安装** → **启用**

### 步骤5: 配置桥接

```python
# 编辑 cnsh_community_bridge.py 配置
config = CommunityConfig(
    base_url='http://your-site.com',  # 你的Discuz!站点
    api_key='your-api-key-from-deploy',  # 部署时生成的API密钥
    sync_interval=300  # 同步间隔(秒)
)
```

## 六、70%治理规则详解

### 6.1 投票权重计算

```
投票权重 = 基础权重 + 活跃度加成 + 心种子加成

基础权重 = 信誉分 (0-100)
活跃度加成 = min(发帖数+登录天数) * 0.1, 20)
心种子加成 = 心种子哈希前4位数字和 (0-15)

上限: 100
```

### 6.2 通过条件

```
通过条件: 支持权重 / 总权重 >= 70%

示例:
- 总投票权重: 1000
- 支持权重: 750
- 支持率: 75%
- 结果: 通过 (75% >= 70%)
```

### 6.3 提案类型

| 类型 | 说明 | 执行方式 |
|------|------|----------|
| 社区规则 | 修改社区规章制度 | 自动更新缓存 |
| 版主任免 | 任命/撤销版主 | 自动更新权限 |
| 资金分配 | 社区资金支出 | 自动执行转账 |
