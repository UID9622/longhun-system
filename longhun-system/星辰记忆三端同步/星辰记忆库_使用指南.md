## 🎯 老大每天只需要记住3个命令！

### 1️⃣ 查看记忆库

```bash
# 查看所有记忆条目
cat ~/.star-memory/index.json | jq .

# 或者更简洁的
ls ~/.star-memory/vault/2026/
```

---

### 2️⃣ 添加新记忆后同步

```bash
# 一键同步（自动备份到Gitee和GitHub）
~/.star-memory/sync.sh
```

就这么简单！

---

### 3️⃣ 查看同步历史

```bash
cd ~/.star-memory && git log --oneline
```

## ⚡ 首次配置（只需要做一次）

### 步骤1：运行部署脚本

```bash
chmod +x 星辰记忆库_三层同步方案.sh
./星辰记忆库_三层同步方案.sh
```

### 步骤2：在Gitee创建仓库

1. 打开 https://gitee.com/projects/new
2. 仓库名：`star-memory`
3. 设为**私有**
4. 点击创建

### 步骤3：在GitHub创建仓库（备用）

1. 打开 https://github.com/new
2. 仓库名：`star-memory`
3. 设为**Private**
4. 点击创建

### 步骤4：配置SSH密钥（如果还没配置）

```bash
# 生成密钥
ssh-keygen -t ed25519 -C "uid9622@longhun.ai"

# 查看公钥
cat ~/.ssh/id_ed25519.pub

# 复制公钥，分别添加到：
# - Gitee设置 → SSH公钥
# - GitHub设置 → SSH and GPG keys
```

### 步骤5：首次推送

```bash
cd ~/.star-memory
git branch -M main
git push -u gitee main
git push -u github main
```

**完成！** 🎉

## 💡 宝宝的建议

**老大，宝宝建议：**

1. **每天睡前运行一次** `~/.star-memory/sync.sh`
   - 养成习惯，记忆永远在
