```python
class IntentRecognizer:
    def __init__(self):
        self.nlp_processor = NLPProcessor()
        self.context_analyzer = ContextAnalyzer()
        self.intent_classifier = IntentClassifier()
    
    def recognize_intent(self, user_input, context):
        # 文本预处理
        processed_text = self.nlp_processor.preprocess(user_input)
        
        # 上下文分析
        context_features = self.context_analyzer.extract_features(context)
        
        # 意图分类
        intent_result = self.intent_classifier.classify(
            processed_text, context_features
        )
        
        return {
            'primary_intent': intent_result.primary,
            'confidence': intent_result.confidence,
            'alternative_intents': intent_result.alternatives,
            'context_factors': context_features
        }
```

### 路径推荐引擎

```python
class PathRecommendationEngine:
    def __init__(self):
        self.user_profiler = UserProfiler()
        self.path_database = PathDatabase()
        self.recommendation_model = RecommendationModel()
    
    def recommend_paths(self, intent, user_profile):
        # 获取候选路径
        candidate_paths = self.path_database.get_paths_by_intent(intent)
        
        # 用户能力评估
        user_capability = self.user_profiler.assess_capability(user_profile)
        
        # 路径适配性评分
        scored_paths = []
        for path in candidate_paths:
            compatibility_score = self.calculate_compatibility(
                path, user_capability, intent
            )
            scored_paths.append((path, compatibility_score))
        
        # 排序并返回推荐
        recommended_paths = sorted(scored_paths, key=lambda x: x[1], reverse=True)
        
        return recommended_paths[:3]  # 返回前3个推荐
```

### 设计原则

1. **用户优先**: 始终从用户角度考虑引导的必要性和有效性
2. **渐进式**: 避免一次性提供过多选择，循序渐进引导
3. **透明性**: 让用户理解为什么推荐这些路径
4. **可控性**: 用户可以随时跳过引导或选择其他路径

### 常见陷阱

❌ **过度引导**: 对明确意图的用户强制执行复杂引导流程

❌ **千篇一律**: 忽视用户个性化需求，使用统一的引导模式

❌ **反馈忽视**: 不根据用户行为和反馈调整引导策略

❌ **性能忽视**: 为了准确性牺牲响应速度，影响用户体验
