## 🎯 使用方法

### 基本使用
```python
# 导入主引擎
from cnsh_font_engine_uid9622 import CNSH字元基础引擎_UID9622

# 创建引擎实例
引擎 = CNSH字元基础引擎_UID9622()

# 载入CNSH数据
引擎.载入_cnsh数据_cnsh龙魂_v1("CNSH字元库.json")

# 执行三色审计
引擎.执行三色审计_cnsh龙魂_v1()

# 输出SVG
引擎.输出SVG_cnsh龙魂_v1("龙", "output.svg")
```

### 批量使用
```python
# 导入批量引擎
from CNSH字元引擎_UID9622_批量版 import CNSH字元批量引擎_UID9622

# 创建批量引擎实例
批量引擎 = CNSH字元批量引擎_UID9622()

# 批量渲染
批量引擎.批量渲染(["龙", "魂", "中", "华"], "output_dir/")
```
