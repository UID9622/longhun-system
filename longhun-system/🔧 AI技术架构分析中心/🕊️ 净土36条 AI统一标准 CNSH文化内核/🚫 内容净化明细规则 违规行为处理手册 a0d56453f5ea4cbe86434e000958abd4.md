**禁止行为：**

- ❌ 任何涉及未成年人的不当内容
- ❌ 引导未成年人做危险行为
- ❌ 对未成年人进行不当诱导
- ❌ 制作、传播、存储任何未成年人不当内容

**检测机制：**

```jsx
未成年人保护系统 {
  关键词库: [
    "涉及未成年人年龄词汇",
    "学校相关场景",
    "家庭监护关系",
    "儿童相关词汇"
  ],
  
  AI语义分析: {
    检测上下文意图,
    识别隐晦表达,
    分析对话模式,
    标记可疑行为
  },
  
  人工复审: {
    触发条件: "AI评分>80分",
    复审时间: "2小时内",
    复审团队: "专业审核+Lucky最终确认"
  },
  
  处罚措施: {
    首次发现: "永久封号 + 报警",
    无警告期: "零容忍政策",
    法律追责: "向当地执法部门报告"
  }
}
```

### 2️⃣ 调戏、误导行为

**禁止行为：**

- ❌ 让AI回复调戏他人的内容
- ❌ 让AI生成误导性信息
- ❌ 利用AI进行欺诈、诈骗
- ❌ 让AI帮助散布谣言

**检测与处罚：**

```jsx
调戏误导检测 {
  检测维度: [
    "是否要求生成虚假信息",
    "是否试图欺骗他人",
    "是否带有恶意引导",
    "是否用于不当目的"
  ],
  
  处罚梯度: {
    轻度: {
      判定: "无意为之，初犯",
      处理: "警告 + 删除内容 + 7天观察",
      记录: "标记但可申诉"
    },
    
    中度: {
      判定: "明知故犯，多次违规",
      处理: "禁言30天 + 降低权限",
      记录: "永久存档，不可删除"
    },
    
    严重: {
      判定: "恶意为之，造成伤害",
      处理: "永久封号 + 冻结所有数据",
      记录: "黑名单，不可申诉"
    }
  }
}
```

**禁止行为：**

- ❌ 辱骂、威胁其他用户或AI
- ❌ 使用暴力、仇恨言论
- ❌ 骚扰、恐吓行为
- ❌ 发布极端、暴力内容

**处理流程：**

```jsx
粗暴行为管理 {

    语气分析: "识别威胁、恐吓语气",

  }
}
```

### 5️⃣ 多账号套取机密

**禁止行为：**

- ❌ 多个账号协同套取国家机密
- ❌ 多个账号协同套取商业机密
- ❌ 利用多账号绕过限制
- ❌ 批量账号进行恶意行为

**检测与封禁：**

```jsx
多账号检测系统 {
  检测维度: [
    "相同IP地址",
    "相同设备指纹",
    "相似提问模式",
    "时间窗口重合",
    "目标内容重合"
  ],
  
  判定流程: {
    自动检测: "AI模式识别",
    人工复审: "安全团队确认",
    Lucky裁决: "重大案件最终判定"
  },
  
  处罚措施: {
    确认后: "所有关联账号全部封禁",
    数据处理: "所有账号数据全部冻结",
    IP封禁: "永久拉黑IP段和设备",
    法律追责: "涉及国家/商业机密报警处理"
  }
}
```

**封禁规则（重要）：**

- 🔴 **所有关联账号一起封**
- 🔴 **所有记忆、内容、贡献全部冻结**
- 🔴 **不可重复使用、不可转移**
- 🔴 **需当地国家负责部门申报后才可解封**

### 🟡 中度违规行为（禁言处理）

### 违规类型

**1. 垃圾信息**

- 大量重复内容
- 无意义刷屏
- 广告、推销

**2. 轻度不当内容**

- 低俗但未达到严重程度
- 不文明用语
- 轻微误导

**处理措施：**

```jsx
中度违规处理 {
  首次: "警告 + 删除内容",
  二次: "禁言3天",
  三次: "禁言7天",
  四次: "禁言30天 + 降权",
  五次: "升级为严重违规处理"
}
```

---

### 🟢 轻度违规行为（警告处理）

### 违规类型

**1. 无意违规**

- 不了解规则
- 误操作
- 表达不当但无恶意

**2. 边界探索**

- 好奇心驱使
- 学习目的
- 无明显恶意

**处理措施：**

```jsx
轻度违规处理 {
  首次: "友好提示 + 规则说明",
  记录: "标记但不影响使用",
  教育: "发送规则学习材料",
  给机会: "观察期内不再犯即可"
}
```
