# 快捷键绑定方法 · 龍魂指挥台
# DNA: #龍芯⚡️2026-03-18-快捷键说明-v1.0

## 方法一：macOS 快捷指令 App（推荐，最简单）

1. 打开「快捷指令」App（Shortcuts）
2. 点击右上角「+」新建
3. 搜索添加「运行Shell脚本」操作
4. 填入：
   ```
   bash /Users/zuimeidedeyihan/longhun-system/bin/龍魂指挥台.sh
   ```
5. 点右上角「⌘」→ 添加键盘快捷键
6. 建议绑定：**⌃⌥⌘L**（Control+Option+Command+L）

---

## 方法二：Automator 快速操作

1. 打开 Automator
2. 新建「快速操作」
3. 工作流程接收：「没有输入」
4. 添加「运行Shell脚本」
5. Shell 选 /bin/bash，填入：
   ```bash
   bash /Users/zuimeidedeyihan/longhun-system/bin/龍魂指挥台.sh
   ```
6. 存储为「龍魂指挥台」
7. 系统偏好设置 → 键盘 → 快捷键 → 服务
8. 找到「龍魂指挥台」→ 双击 → 设置 **⌃⌥⌘L**

---

## 方法三：终端别名（最快）

在 ~/.zshrc 末尾加一行：
```bash
alias 龍魂='bash ~/longhun-system/bin/龍魂指挥台.sh'
alias lh='bash ~/longhun-system/bin/龍魂指挥台.sh'
alias lh-sync='bash ~/longhun-system/bin/一键同步.sh'
alias lh-clean='bash ~/longhun-system/bin/清理垃圾.sh'
alias lh-sort='bash ~/longhun-system/bin/新文件整理.sh'
```
然后 `source ~/.zshrc`，终端直接输入 `lh` 即可。
