## 一、快捷指令(Shortcuts) · 现在就能做

打开"快捷指令"App，新建以下指令：

### 指令1：龍魂看板
- 动作：运行Shell脚本
- 脚本：`bash ~/longhun-system/bin/每日看板.sh`
- Siri触发词：**"龍魂看板"**

### 指令2：一键同步
- 动作：运行Shell脚本
- 脚本：`bash ~/longhun-system/bin/一键同步.sh`
- Siri触发词：**"龍魂同步"**

### 指令3：启动宝宝精灵
- 动作：运行Shell脚本
- 脚本：`python3 ~/longhun-system/bin/宝宝精灵.py &`
- Siri触发词：**"叫出宝宝"**

### 指令4：换脸工具
- 动作：运行Shell脚本
- 脚本：`bash ~/longhun-system/私密区/影像库/换脸工具/run_roop.sh`
- Siri触发词：**"开始换脸"**

## 二、Final Cut Pro · 换脸视频后期

换脸出来的MP4可以直接拖进FCP：
1. 导入：`~/longhun-system/私密区/影像库/换脸工具/龍魂换脸工具_roop/我的素材/输出/`
2. 用FCP调色/剪辑/加字幕
3. 用Compressor压缩输出（比ffmpeg质量好）

---

## 三、Logic Pro · 给宝宝精灵配音

1. Logic Pro录制或生成音频
2. 导出为AIFF/MP3
3. 替换宝宝精灵的`say`命令为播放音频文件：
   `afplay ~/longhun-system/assets/宝宝声音.aiff`

## 四、Photomator · 换脸素材处理

换脸前的人脸照片可用Photomator：
- AI自动去背景
- 肤色调整匹配目标视频
- 导出高质量JPEG

## 七、AppleScript自动化示例

```applescript
-- 打开看板并念出来
do shell script "bash ~/longhun-system/bin/每日看板.sh > /tmp/看板.txt"
set 内容 to read file "/tmp/看板.txt"
say 内容 using "Shelley (中文（中国大陆）)"
```
