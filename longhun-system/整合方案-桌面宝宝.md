## 整合步骤

### 1. 复制桌面宝宝代码到项目
```bash
# 在终端执行
cp -r longhun-system/LongHunWidget/桌面宝宝/ ~/Desktop/LongHunWidget/
```

### 2. 创建共生体模块
我现在给你写 4 个连接器：
- ClaudeConnector - 连接 Claude API
- NotionConnector - 连接 Notion API
- XcodeAutomation - 自动化 Xcode 操作
- AgentOrchestrator - 编排所有智能体

### 3. 修改 Info.plist 权限
需要添加：
- 麦克风权限（语音识别）
- 辅助功能权限（控制其他 App）
- 文件访问权限（整理桌面）

### 4. 在 Xcode 中添加 Target
- 原有 Widget Target 保留
- 新增 macOS App Target（桌面宝宝）
- 共生体模块作为 Framework
