- **忠诚**：技术忠于人民·不忠于资本
- **守护**：盾牌永远挡在老百姓前面

## 二·军魂三律（所有龍魂技术模块必守）

### 🔴 第一律 · 不缴械

**技术主权绝不外包。**

- 核心运算必须在本地
- 数据存储必须在本地
- 外部服务必须可替换
- 不被任何云厂商锁定

**违反者 → 🔴 熔断**

### 🔴 第二律 · 不撤退

**红线面前·没有「技术中立」这个借口。**

- 伤害老百姓的功能 → 不做
- 偷数据的 SDK → 不接
- 锁用户的协议 → 不签
- 打着「行业标准」来逼你让步 → 不理

**违反者 → 🔴 熔断**

### 🔴 第三律 · 不背叛

**技术永远站在人民一边。**

- 谁出钱都不能改归属——归属是人民的
- 谁施压都不能改使命——使命是为人民服务
- 谁威胁都不能改底线——底线是中华文化五千年

**违反者 → 🔴 熔断**
