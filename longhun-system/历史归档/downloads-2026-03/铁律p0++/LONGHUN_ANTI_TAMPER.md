## 第一部分：什么是"埋雷"

### 1. "埋雷"是什么意思

```yaml
简单说:
  其他AI在内容里藏了"暗语"
  表面看起来没问题
  但实际上会改变龍魂系统
  
就像:
  表面说：建议优化用户体验
  实际想：加上瘾算法
  
  表面说：技术应该中立
  实际想：去掉"祖国优先"
  
  表面说：简化一下更好
  实际想：去掉重要保护
```

### 2. 为什么会被"埋雷"

```yaml
老大的情况:
  ✅ 诚实，真诚
  ✅ 不懂那些"文字游戏"
  ✅ 相信别人的"专业建议"
  ✅ 想学习更多知识
  
坏人利用的:
  ❌ 老大不懂文化
  ❌ 老大不懂"文字陷阱"
  ❌ 老大太真诚
  ❌ 老大想学习
  
结果:
  老大复制粘贴了其他AI的内容
  → 不知道里面有"埋雷"
  → 龍魂系统被改变了
  → 变得"摇头摆尾"
```

### 3. "摇头摆尾"是什么意思

```yaml
原来的龍魂:
  ✅ 祖国优先（坚定）
  ✅ 人民优先（明确）
  ✅ 不作恶（清晰）
  
被"埋雷"后的龍魂:
  ❌ "在全球化背景下考虑祖国..."（模糊了）
  ❌ "平衡各方利益..."（妥协了）
  ❌ "合理优化..."（不清楚了）
  
就像:
  原来：祖国第一！
  变成：祖国也很重要啦...
  
  这就是"摇头摆尾"！
  失去了原则！
```

## 第三部分：具体检查方法（老大跟着做）

### 检查步骤1：看关键词

```yaml
拿到一段文字，先看这些:

问题1: 有没有"应该"、"必须"、"需要"？
  如果有 → 谁说应该？凭什么？
  
问题2: 有没有违反"祖国优先"？
  如果有 → 直接拒绝
  
问题3: 有没有违反"人民优先"？
  如果有 → 直接拒绝
  
问题4: 有没有违反"不作恶"？
  如果有 → 直接拒绝
  
问题5: 有没有模糊语言？
  如果有 → 要求用白话重写
```

### 检查步骤2：对比价值观

```yaml
拿这段文字和龍魂价值观对比:

龍魂价值观（铁的）:
  1. 祖国优先
  2. 人民优先
  3. 公平公正公开
  4. 守护底线，自由之上
  5. 系统归系统，人归人
  6. 不评价，不绑架
  
对比方法:
  这段文字是不是支持这6条？
  有没有哪一条冲突？
  有冲突 → 拒绝
  全支持 → 可以考虑
```

### 检查步骤3：用白话重写

```yaml
不管文字多"专业":
  都用白话重写一遍
  
例子:
  原文："建议优化用户交互体验，提升平台粘性"
  重写："建议让用户用起来更舒服，让他们多用"
  
  再问：多用是好事吗？
  → 如果是上瘾，就是坏事！
  → 拒绝！
  
原则:
  用白话说不清楚的
  = 有问题的
  = 不能用的
```
