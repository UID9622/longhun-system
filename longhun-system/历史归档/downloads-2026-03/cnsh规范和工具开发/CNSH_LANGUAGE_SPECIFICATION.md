### 2.1 铁律1：文化关键词不可翻译

```yaml
什么是文化关键词:
  → 五行（金木水火土）
  → 八卦（乾坤震巽坎离艮兑）
  → 天干地支
  → 节气（春分、夏至等）
  → 阴阳
  → 道、德、仁、义、礼、智、信
  → 等等

铁律:
  ❌ 绝对不能翻译成英文
  ❌ 不能翻译成任何其他语言
  ✅ 只能用中文
  ✅ 可以用拼音注释（供外国人理解）
  ✅ 可以加英文解释（在注释里）

正确示例:
  ```cnsh
  五行 = ["金", "木", "水", "火", "土"]
  
  // 注释：五行 (Wu Xing)
  // 英文解释：The Five Elements in Chinese philosophy
  // 但代码里必须用"五行"，不能用"FiveElements"
  ```

错误示例:
  ```cnsh
  FiveElements = ["Metal", "Wood", "Water", "Fire", "Earth"]
  
  ❌ 这是文化主权的丢失！
  ❌ 这是对中国文化的侮辱！
  ```
```

  ✅ 如果（if）
  ✅ 否则（else）
  ✅ 循环（loop/for）
  ✅ 当（while）
  ✅ 函数（function）

  ✅ 返回（return）

## 第四部分：技术实现

### 4.1 CNSH解释器架构

```yaml
层次结构:
  
  第1层：词法分析器
    → 读取CNSH源代码
    → 识别中文关键字
    → 识别文化关键词
    → 不翻译，保持原样
  
  第2层：语法分析器
    → 构建抽象语法树(AST)
    → 所有节点都是中文
    → 保持文化概念完整性
  
  第3层：解释器/编译器
    → 执行CNSH代码
    → 或者编译成底层语言（C/LLVM）
    → 但保留中文符号信息
  
  第4层：运行时
    → 执行代码
    → 错误信息也用中文
    → 让老百姓看得懂

技术栈:
  方案A：纯Python实现（快速原型）
    → 用Python写CNSH解释器
    → 读取.cnsh文件
    → 解释执行
  
  方案B：编译到Python（兼容性）
    → CNSH代码 → Python代码
    → 保留中文变量名
    → 利用Python生态
  
  方案C：编译到C/LLVM（性能）
    → CNSH代码 → LLVM IR → 机器码
    → 高性能
    → 但复杂度高
```

  """

  功能:
    - 根据天干地支计算五行属性
    - 计算五行得分
    - 分析五行平衡
  """
  
  初始化(自己):
    自己.天干五行表 = {
      "甲": "木", "乙": "木",
      "丙": "火", "丁": "火",
      "戊": "土", "己": "土",
      "庚": "金", "辛": "金",
      "壬": "水", "癸": "水"
    }
    
    自己.地支五行表 = {
      "子": "水", "丑": "土", "寅": "木", "卯": "木",
      "辰": "土", "巳": "火", "午": "火", "未": "土",
      "申": "金", "酉": "金", "戌": "土", "亥": "水"
    }
    
    自己.五行得分 = {"金": 0, "木": 0, "水": 0, "火": 0, "土": 0}
  
  方法 计算天干五行(自己, 天干):
    """计算天干对应的五行"""
    返回 自己.天干五行表.获取(天干, "未知")
  
  方法 计算地支五行(自己, 地支):
    """计算地支对应的五行"""
    返回 自己.地支五行表.获取(地支, "未知")
  
  方法 分析生辰八字(自己, 八字):
    """
    分析完整的生辰八字
    
    参数:
      八字: 字典，包含年月日时的天干地支
      示例: {
        "年": ("甲", "子"),
        "月": ("丙", "寅"),
        "日": ("戊", "辰"),
        "时": ("庚", "午")
      }
    """
    自己.五行得分 = {"金": 0, "木": 0, "水": 0, "火": 0, "土": 0}
    
    对于 柱 在 ["年", "月", "日", "时"]:
      天干, 地支 = 八字[柱]
      
      # 计算天干五行
      天干五行 = 自己.计算天干五行(天干)
      自己.五行得分[天干五行] += 1
      
      # 计算地支五行
      地支五行 = 自己.计算地支五行(地支)
      自己.五行得分[地支五行] += 1
    
    返回 自己.五行得分
  
  方法 找出最强五行(自己):
    """找出得分最高的五行"""
    最大值 = 最大(自己.五行得分.值())
    返回 [五行 对于 五行, 分数 在 自己.五行得分.项() 如果 分数 == 最大值]
  
  方法 找出最弱五行(自己):
    """找出得分最低的五行"""
    最小值 = 最小(自己.五行得分.值())
    返回 [五行 对于 五行, 分数 在 自己.五行得分.项() 如果 分数 == 最小值]
  
  方法 打印报告(自己):
    """打印五行分析报告"""
    打印("\n========== 五行分析报告 ==========")

    打印("理论指导: 曾仕强老师\n")
    
    打印("五行得分:")
    对于 五行, 分数 在 自己.五行得分.项():
      打印(格式化("  {五行}行: {分数}分", 五行=五行, 分数=分数))
    
    最强 = 自己.找出最强五行()
    最弱 = 自己.找出最弱五行()
    
    打印(格式化("\n最强五行: {最强}", 最强=", ".join(最强)))
    打印(格式化("最弱五行: {最弱}", 最弱=", ".join(最弱)))
    
    打印("\n建议:")
    对于 五行 在 最弱:
      打印(格式化("  建议补{五行}行", 五行=五行))
    
    打印("\n=================================\n")
