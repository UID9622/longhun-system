🧾 一、导出格式说明

| 导出格式 | 文件类型 | 用途说明 | 推荐使用场景 |
| --- | --- | --- | --- |
| PDF | .pdf | 格式清晰、易分享 | 提交成果 / 保存档案 |
| Markdown (.md) | .md | 文本结构清晰，适合开发 | 技术迁移 / 写作 |
| HTML (.zip) | .zip | 离线保存网页结构 | 本地浏览 / 静态存档 |
| CSV（数据库） | .csv | 可导入 Excel 分析数据 | 数据整理 / 迁移外部表格系统 |

---

🧠 二、字段类型总览

| 字段类型 | 是否多选 | 功能用途 | 示例说明 |
| --- | --- | --- | --- |
| 文本 Text | 否 | 输入内容 | 项目标题 / 描述 |
| 富文本 Text | 是 | 可编辑段落内容 | 总结 / 详细记录 |
| 多选 Multi-select | ✅ | 多标签分类 | #AI、#任务、#角色 |
| 单选 Select | ❌ | 选择状态/阶段 | “已完成”、“进行中” |
| 日期 Date | 否 | 时间节点记录 | 2025-08-13 |
| 数字 Number | 否 | 排序 / 打分 | 1～5 |
| URL | 否 | 外链引用 | 网页地址 / 文件路径 |
| 文件 File | 是 | 上传文件 | PDF、截图 |
| 人员 Person | 是 | 协作分配 | Lucky / 系统角色 |
