**问题**: `Cannot find module 'express'`  
**原因**: Node.js模块缺失  
**解决方案**:

```bash
# 进入项目目录
cd /path/to/your/project

# 安装express
npm install express

# 或安装所有依赖
npm install

# 重新启动服务器
node server.js
```

### 服务器启动失败

**可能原因**:
1. 端口被占用
2. 依赖未完全安装
3. 配置文件错误

**解决方案**:

```bash
# 检查端口占用
lsof -i :3000

# 使用不同端口
node server.js --port 3001

# 查看详细错误
node server.js --verbose
```
