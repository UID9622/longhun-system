## 🚀 一键发布流程

### 第一步：配置Gitee仓库（首次使用）

1. **登录Gitee** - https://gitee.com
2. **创建新仓库** - 仓库名：`ai-security-toolkit`
3. **设置为公开仓库**
4. **获取仓库地址** - 例如：`https://gitee.com/你的用户名/ai-security-toolkit.git`

### 第二步：更新配置（可选）

编辑 `publish_config.json`：
```json
{
  "platforms": {
    "gitee": {
      "repo_name": "你的仓库名",
      "repo_url": "你的完整仓库地址"
    }
  }
}
```

### 第三步：执行发布

```bash

# 方式2：直接运行Python脚本
python3 auto_publisher.py
```
