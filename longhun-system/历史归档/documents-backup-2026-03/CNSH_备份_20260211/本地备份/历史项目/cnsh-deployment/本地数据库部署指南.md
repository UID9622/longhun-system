# CNSH 本地数据库部署指南

## 🎯 概述

这个指南将帮助您建立一个完全本地化的CNSH数据库系统，让您的数据和AI人格系统完全自主可控，不需要依赖任何外部服务。

## 🚀 快速开始

### 1. 运行自动部署脚本

```bash
cd cnsh-deployment
chmod +x local-db-setup.sh
./local-db-setup.sh
```

这个脚本将自动：
- 安装必要的数据库组件
- 初始化SQLite和PostgreSQL数据库
- 创建CNSH 12核心人格系统
- 配置自动备份和监控
- 设置本地启动脚本

### 2. 启动本地服务

```bash
./scripts/start-local.sh
```

### 3. 验证部署

```bash
./scripts/monitor.sh
```

## 📊 数据库架构

### 主数据库 (SQLite)

**位置**: `./data/databases/sqlite/cnsh.db`

**核心表结构**:

**功能**:

## 🔧 高级配置

### 环境变量配置

编辑 `./data/.env.local`:

```bash
# 数据库类型选择
DB_TYPE=sqlite          # 或 postgres

# 数据库路径
DB_PATH=./data/databases/sqlite/cnsh.db

# 向量数据库配置
VECTOR_DB_TYPE=sqlite
VECTOR_DB_PATH=./data/vector_db

# CNSH系统配置
CNSH_MODE=local
CNSH_DATA_DIR=./data
CNSH_BACKUP_DIR=./data/backups

# 安全配置
JWT_SECRET=your_secret_key
API_KEY=your_api_key
```

### 性能优化选项

编辑 `database-enhanced.js` 配置:

```javascript
const config = {
    enableWAL: true,        // 启用WAL模式提高并发性能
    enableFTS: true,        // 启用全文搜索
    autoBackup: true,        // 启用自动备份
    backupInterval: 24 * 60 * 60 * 1000,  // 备份间隔(24小时)
    cacheSize: 10000,       // 缓存大小
    mmapSize: 268435456     // 内存映射大小(256MB)
};
```

```javascript
// Redis缓存配置
const redis = require('redis');
const client = redis.createClient({
    url: process.env.REDIS_URL || 'redis://localhost:6379'
});

// 缓存查询结果
async function getCachedResult(key, query) {
    const cached = await client.get(key);
    if (cached) return JSON.parse(cached);
    
    const result = await query();
    await client.setex(key, 3600, JSON.stringify(result));
    return result;
}
```

## 🆘 故障排除

### 常见问题

1. **数据库锁定**
   ```bash
   # 检查锁定状态
   lsof ./data/databases/sqlite/cnsh.db
   
   # 强制解锁
   sqlite3 ./data/databases/sqlite/cnsh.db "PRAGMA wal_checkpoint(TRUNCATE);"
   ```

2. **内存不足**
   ```javascript
   // 减少缓存大小
   await db.run('PRAGMA cache_size = 1000');
   
   // 增加虚拟内存
   await db.run('PRAGMA mmap_size = 67108864'); // 64MB
   ```

3. **备份失败**
   ```bash
   # 检查磁盘空间
   df -h ./data/backups/
   
   # 手动备份
   sqlite3 ./data/databases/sqlite/cnsh.db ".backup ./data/backups/manual_backup.db"
   ```
