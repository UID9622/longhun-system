## 🚀 三步快速上手

### 第一步：准备文件

```bash
# 创建私密层目录
mkdir -p ./私密层

# 把要发布的文件放入 ./私密层
cp -r 你的源代码/* ./私密层/
```

### 第二步：一键发布

```bash
# 给脚本添加执行权限
chmod +x 一键开源发布.sh

# 执行发布（默认版本 v1.0.0）
./一键开源发布.sh v1.0.0
```

### 第三步：Lucky 最终签名

```bash
# 1. 查看发布报告
cat 发布报告-v1.0.0.json

# 2. GPG 签名（必须）
gpg --detach-sign --armor dragonsoul-system-v1.0.0-release.tar.gz

# 3. 发布到 GitHub
gh release create v1.0.0 dragonsoul-system-v1.0.0-release.tar.gz
```

## 🔧 手动执行（可选）

如果你想分步执行，可以使用这些命令：

### 1. 生成公开层

```bash
python3 open_source_release_helper.py \
  generate-public \
  ./私密层 \
  ./公开层 \
  --mapping 私密→公开映射表-v1.0.0.json
```

### 2. 扫描敏感信息

```bash
python3 open_source_release_helper.py \
  scan-sensitive \
  ./公开层 \
  > sensitive_scan_report.json

# 查看报告
cat sensitive_scan_report.json
```

### 3. 运行检查清单

```bash
python3 open_source_release_helper.py \
  checklist \
  ./公开层 \
  > 发布检查报告-v1.0.0.json

# 查看报告
cat 发布检查报告-v1.0.0.json
```

### 4. 注入固定块（5个标准声明块）

```bash
python3 memory_quantum_system.py /开源 > injected_blocks.json

# 会自动注入到 README.md 顶部
```

### 5. 构建发布包

```bash
python3 open_source_release_helper.py \
  build \
  ./公开层 \
  --version v1.0.0 \
  --name dragonsoul-system-v1.0.0-release.tar.gz
```

### 6. 验证发布包

```bash
python3 open_source_release_helper.py \
  verify \
  dragonsoul-system-v1.0.0-release.tar.gz
```
