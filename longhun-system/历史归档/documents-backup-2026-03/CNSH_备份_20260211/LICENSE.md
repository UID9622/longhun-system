# 龍魂开源协议 v1.0

## 最高原则

龍魂系统属于全球公益事业，永不商业化。

### 📜 使用权

✅ **允许**：
- 教育用途（学校、培训、科研）
- 环保公益（生态修复、可持续农业）
- 弱势帮扶（妇女、儿童、残障人士）
- 政策宣传（反战、反歧视、人权）
- 个人学习与开发

❌ **禁止**：
- 商业软件集成
- API服务付费化
- 数据交易或售卖
- 品牌名称修改或声称所有权
- 移除"龍芯"前缀或DNA追溯码

### 🔐 文化锚不可变

以下内容属于龍魂系统的核心身份，任何版本都不得修改：

1. **CNSH语义体系** - 变量、符号、设计理念
2. **易经64卦体系** - 伦理决策框架
3. **甲骨文锚点** - 文化根源标记
4. **#龍芯⚡️前缀** - 所有代码文件的身份证

修改 = 背叛 = 社区监管

### 🌍 全球协作规则

各国可基于龍魂开发本地版本，但必须：

1. **实名认证** - 贡献者真实身份可验证
2. **虚假信息留证** - 所有错误信息自动快照存档
3. **三次违规投票** - 第三次违规后由全球社区投票决定处罚

投票结果：
- 暂停权限（3个月）
- 永久除名
- 品牌声誉降级
- 代码审计重做

### 💝 公益白名单

以下用途永久免费且鼓励使用：

**教育**
- 乡村学校数字化
- 老年人数字扫盲
- 未来领袖培养项目

**环保**
- 生态修复监测
- 可持续农业方案
- 气候变化研究

**弱势帮扶**
- 妇女儿童保护
- 残障人士赋能
- 贫困地区支援

**政策宣传**
- 反战、反歧视
- 人权倡议
- 国际共识推广

### ⚖️ 冲突解决

1. **初级** - 社区讨论与道歉
2. **中级** - 代码回滚与审计
3. **高级** - 全球社区投票决定
