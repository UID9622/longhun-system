- **ESLint + TSLint**：TSLint 已弃用，使用 ESLint
- **多个 AI 助手**：推荐只使用一个，避免冲突

### 解决方案
1. 卸载冲突插件
2. 调整插件优先级
3. 使用插件配置文件管理
