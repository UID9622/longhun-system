## 🎯 系统概述

### 功能说明

UID9622 每周自动清理系统会自动清理以下内容：

1. **Python 缓存**
   - `__pycache__` 目录
   - `.pyc` 字节码文件

2. **临时文件**
   - `.tmp` 临时文件
   - `.DS_Store` 系统文件

3. **旧日志**
   - 超过 7 天的日志文件

4. **pip 缓存**
   - Python 包缓存

---

## ⏰ 自动清理功能

### 默认配置

| 配置项 | 值 |
|--------|-----|
| 执行频率 | 每周日 |
| 执行时间 | 12:00 (中午) |
| 任务名称 | com.lucky.uid9622.weeklyclean |

### 工作原理

系统使用 macOS 的 `launchd` 定时任务服务：

```xml
<key>StartCalendarInterval</key>
<dict>
    <key>Weekday</key>
    <integer>0</integer>  <!-- 0 = 周日 -->
    <key>Hour</key>
    <integer>12</integer>  <!-- 12:00 -->
    <key>Minute</key>
    <integer>0</integer>  <!-- 0 分 -->
</dict>
```

### 自动执行

每个周日上午 12:00，系统会自动：

1. ✅ 运行清理脚本
2. ✅ 记录清理日志
3. ✅ 生成清理统计
4. ✅ 释放磁盘空间

## ❓ 常见问题

### Q1: 清理会影响系统功能吗？

**A:** 不会。清理的内容都是临时文件和缓存，删除后不影响系统功能。

### Q2: 清理需要多长时间？

**A:** 通常 1-3 分钟，取决于需要清理的文件数量。

### Q3: 清理会删除重要数据吗？

**A:** 不会。清理规则非常保守：
- 只删除缓存文件
- 只删除超过 7 天的日志
- 不删除任何代码或配置文件

### Q4: 如何自定义清理规则？

**A:** 可以编辑清理脚本：

```bash
nano /Users/zuimeidedeyihan/LuckyCommandCenter/UID9622-LocalServer/UID9622-Scripts/weekly_clean.sh
```

### Q5: 定时任务没有运行怎么办？

**A:** 检查以下几点：

1. 任务是否已加载：
   ```bash
   launchctl list | grep uid9622
   ```

2. 系统是否在周日上午 12:00 开机
3. 查看错误日志：
   ```bash
   cat /Users/zuimeidedeyihan/LuckyCommandCenter/UID9622-LocalServer/logs/weekly_clean_error.log
   ```
