## 🛠️ 故障排除

### 常见问题

**1. 端口被占用**
```bash
# 查看占用8000端口的进程
lsof -i :8000

# 终止占用进程
kill -9 <PID>

# 或者使用其他端口
python3 start_mvp_server.py --port 8080
```

**2. 依赖包缺失**
```bash
# 安装所有必要依赖
pip3 install pyyaml requests psutil python-dotenv
```

**3. 页面无法显示**
- 检查防火墙设置
- 确认文件路径正确
- 查看服务器日志

### 系统状态检查
```bash
# 健康检查
python3 test_mvp.py

# 查看日志
tail -f ~/UID9622_Workspace/logs/system.log
```
