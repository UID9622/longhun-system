# 快速开始指南 - 朴素的命令行界面

## 安装 PDF 解析库

在终端执行以下命令：

```bash
python3 -m pip install --user PyPDF2 pdfplumber
```

如果失败，尝试：

```bash
pip3 install PyPDF2 pdfplumber
```
