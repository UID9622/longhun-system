# 📱 Mate60热点配置完整指南

## 🎯 **最佳解决方案**

```
柬埔寨流量卡 → Mate60 → 热点 → 苹果电脑
```

**优势：**
- ✅ 零成本：无需购买新卡或设备
- ✅ 稳定性：手机热点技术成熟
- ✅ 简单性：减少设备层级
- ✅ 便携性：Mate60随身携带

## 📋 **详细操作步骤**

### 步骤1: 安装流量卡
1. 关闭Mate60手机
2. 使用卡针打开Mate60的SIM卡托盘
3. 将柬埔寨流量卡插入第二个卡槽
4. 重新安装SIM卡托盘
5. 开启Mate60手机

### 步骤2: 配置移动数据
1. 进入 **设置** → **移动网络**
2. 选择新插入的SIM卡
3. 点击 **启用移动数据**
4. 如需要，配置APN设置（通常自动识别）
5. 设置 **默认移动数据** 为柬埔寨流量卡

### 步骤3: 开启个人热点
1. 进入 **设置** → **个人热点**
2. 开关拨到 **开启**
3. 配置热点设置：
   - **热点名称**: `Lucky-Hotspot`
   - **密码**: 设置8位以上强密码（如：`Lucky2024@88`）
   - **频段**: 选择 **5GHz**（速度更快，干扰更少）
   - **安全性**: **WPA2个人**
   - **最大连接数**: **5台设备**
   - **空闲超时**: **5分钟**

### 步骤4: 电脑连接
1. 在苹果电脑上点击WiFi图标
2. 在网络列表中找到 `Lucky-Hotspot`
3. 点击连接
4. 输入设置的密码
5. 测试网络连接（可访问百度或Google）

## ⚙️ **优化设置建议**

### Mate60热点优化
```
• 频段：5GHz优先（速度更快，干扰少）
• 信道：自动选择（避免冲突）
• 加密：WPA2个人（安全性高）
• 最大连接：5台（性能平衡）
• 空闲超时：5分钟（省电）
• 低数据模式：关闭（保证性能）
```

### 苹果电脑网络设置
```
1. 系统偏好设置 → 网络
2. 将Mate60热点设为最高优先级
3. 设置"自动加入已知网络"
4. 取消"询问是否加入新网络"
5. 保存网络配置
```

## 🛠️ **常见问题解决**

### 连接问题
**问题**: 电脑找不到Mate60热点
**解决**: 
- 确认Mate60热点已开启
- 重启电脑WiFi适配器
- 检查是否选择了正确的频段

**问题**: 连接后无法上网
**解决**:
- 检查Mate60移动数据是否开启
- 确认流量卡有足够余额
- 尝试重启Mate60

### 性能问题
**问题**: 网络速度慢
**解决**:
- 检查流量卡信号强度
- 切换到5GHz频段
- 减少同时连接设备数量

### 稳定性问题
**问题**: 频繁断开连接
**解决**:
- 设置热点空闲超时时间
- 检查Mate60电量（低电量可能影响）
- 重启Mate60热点功能
