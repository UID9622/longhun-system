## ⚠️ 注意事项

1. **编码问题**：输出文件使用 UTF-8 编码
2. **格式丢失**：转换后丢失格式（加粗、表格等），只保留纯文本
3. **大文件处理**：57.47 MB 的 PDF 文件可能需要 10-30 秒
4. **文件权限**：确保有读取源文件和写入输出目录的权限

## 🔍 故障排查

### 问题：缺少 unzip 命令

**错误：** `缺少依赖: unzip`

**解决：**
```bash
brew install unzip
```

### 问题：缺少 PDF 解析库

**错误：** `无法转换 .pdf 文件: 缺少 PDF 解析库`

**解决：**
```bash
pip3 install PyPDF2
# 或
pip3 install pdfplumber
```

### 问题：.doc 文件无法转换

**错误：** `无法转换 .doc 文件: 需要 pandoc`

**解决：**
```bash
brew install pandoc
```
