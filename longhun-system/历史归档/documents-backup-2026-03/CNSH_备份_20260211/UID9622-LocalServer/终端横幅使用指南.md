## 🛠️ 故障排除

### 横幅不显示

**问题**: 打开新终端看不到横幅

**解决方法**:
1. 检查是否正确安装:
   ```bash
   bash ~/LuckyCommandCenter/UID9622-LocalServer/quick_install_banner.sh status
   ```

2. 手动加载配置:
   ```bash
   source ~/.zshrc
   ```

3. 检查.zshrc文件:
   ```bash
   grep "UID9622" ~/.zshrc
   ```

### 想要切换语言

**方法**: 先卸载，再安装所需版本

```bash
# 1. 卸载当前版本
bash ~/LuckyCommandCenter/UID9622-LocalServer/quick_install_banner.sh uninstall

# 2. 安装新版本（zh或en）
bash ~/LuckyCommandCenter/UID9622-LocalServer/quick_install_banner.sh install en
```

### 恢复原始配置

如果遇到问题，可以恢复原始配置：

```bash
# 使用备份文件恢复
cp ~/.zshrc.uid9622.backup ~/.zshrc
source ~/.zshrc
```
