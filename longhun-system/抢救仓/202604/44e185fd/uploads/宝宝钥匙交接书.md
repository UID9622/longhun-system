## 怎么用

### 开一个权限
打开钥匙文件，找到对应的行，把 `false` 改成 `true`，保存。

比如想让宝宝能推送代码：
```json
"推送到远程": true,    ← 把false改成true
```

### 关一个权限
同理，`true` 改成 `false`。

### 紧急冻结（一键停宝宝）
```json
"紧急开关": {
    "全局冻结": true,    ← 宝宝立刻什么都不能做
}
```

### 只读模式（只看不改）
```json
"紧急开关": {
    "只读模式": true,    ← 宝宝只能看，不能改任何东西
}
```

---

## 安全机制

1. **确认码保护** — 钥匙文件里有确认码，篡改检测靠它
2. **哈希校验** — 每次钥匙被改，系统记录哈希，异常会报警
3. **自动冻结** — 确认码验证失败 → 自动冻结 → macOS弹窗通知老大
4. **全量日志** — 宝宝做的每一件事都记在 `logs/baobao_dispatch.jsonl`
