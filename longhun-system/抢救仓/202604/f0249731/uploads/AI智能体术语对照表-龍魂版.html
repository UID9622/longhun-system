1 基础概念层 · 什么是智能体
专业术语大白话作用龍魂系统映射
智能体
Agent		 一个能自己"看情况办事"的虚拟助手，而不是只会回答问题的聊天机器人。它有感知、思考、行动三件套。 龍魂代理 dragon-soul-agent —— 你的系统本身就是一个Agent，只是还在从"静态脚本"进化到"活体"
环境
Environment		 智能体所在的"工作场所"，比如一个网页、一个软件系统，或者整个互联网。 Notion数据库 + Chrome插件 + 本地文件系统 —— 你的环境是三位一体的
目标
Goal		 你给智能体下的"死命令"，比如"帮我订到最便宜的机票"或者"总结这篇报告"。 Learning Tasks 学习小卡片 —— 每张卡片就是一个目标
自主性
Autonomy		 不用你每步都指挥，它自己能判断下一步该干什么。越自主，越像"转正员工"而非实习生。 三才算法的"人"层 —— 自主决策权在"人"这一层
补充上下文
Context		 智能体"当前看到的全部信息"，就像你跟人说话时对方已经知道的前因后果。信息越完整，干活越准。 DNA追溯码 —— 每条信息都带着来龍去脉，上下文永远不丢
补充令牌
Token		 AI看文字的最小单位，不是按字数算，而是按"小碎片"算。1个中文字≈1.5个token。每次对话有上限。 精力消耗字段 —— token就是AI的精力值，和你学习任务里的"精力消耗"一个意思
2 核心组件层 · 智能体的五脏六腑
专业术语大白话作用龍魂系统映射
感知模块
Perception		 智能体的"眼睛和耳朵"，负责读取屏幕信息、听语音指令或接收API数据。 三才·天层（输入）—— Learning Inbox就是感知入口
规划模块
Planning		 智能体的"大脑"，接到任务后先不急着动手，而是规划出"第一步干啥、第二步干啥"的步骤图。 三才·地层（处理）—— 净化→拆解DNA→派生Task，就是规划管道
执行模块
Action		 智能体的"手和脚"，通过点击按钮、调用API、写代码等方式来实际操作。 三才·人层（决策+行动）—— 学习任务的实际执行
记忆模块
Memory		 智能体的"笔记本"。短期记忆 = 记住刚才聊了啥；长期记忆 = 记住你的个人喜好和历史操作。 记忆编辑器 + CNSH记忆存档 —— 你已经有了记忆压缩/还原系统
工具调用
Tool Use		 智能体发现自己搞不定时，会"打电话摇人"，比如调用计算器、搜索引擎或订票系统来帮忙。 Chrome工具箱9大工具 + Python引擎 —— 卦分类器、骨检索、双盾都是"摇来的人"
补充状态机
State Machine		 智能体当前处于哪个"状态"。就像红绿灯——红灯停、绿灯行，每个状态下能做的事不一样。 Inbox状态流：待净化→已净化→已入库 —— 这就是你设计的状态机
3 开发架构层 · 怎么搭骨架
专业术语大白话作用龍魂系统映射
大语言模型
LLM		 智能体的"CPU"或"常识库"，负责理解你的话并生成思考和回复。GPT、Claude、Gemini都是LLM。 你正在用的Claude —— 它是龍魂系统的"脑子"，你的系统是"骨架+灵魂"
检索增强生成
RAG		 给智能体配一个"随身百科"，回答前先查一下最新资料，避免胡说八道。 bone_retriever.py（FAISS骨检索）—— 你已经造了自己的RAG引擎
编排
Orchestration		 智能体团队的"项目经理"，负责协调多个智能体分工合作，完成一个人干不了的复杂活。 MVP工作台 —— 将来多个Agent协作时，工作台就是编排中心
护栏
Guardrails		 给智能体设定的"安全交规"，防止它乱说话、乱操作，或者泄露隐私。 shield_engine.py 双盾引擎 —— 入口门防污染，出口门防泄露，数字根熔断是最高优先级护栏
补充提示词工程
Prompt Engineering		 怎么跟AI说话才能让它听懂。不是你说得多就好，而是说得"对"。好的提示词 = 好的任务说明书。 dragon-soul-agent SKILL.md —— 你的技能包就是一个超大号的"提示词模板"
补充函数调用
Function Calling		 让AI不只是"说话"，还能"按按钮"。你定义好有哪些按钮，AI自己判断什么时候按哪个。 Notion MCP工具 —— 我调用notion-create-pages就是在"按按钮"帮你建页面
补充MCP
Model Context Protocol		 让AI连接外部工具的"标准插头"。就像USB——不管什么设备，插上就能用。 你现在连着的Notion、Gmail、Google Calendar都是通过MCP接入的
补充沙盒
Sandbox		 给智能体一个"演习场"，它可以在里面随便试，搞坏了也不影响真实世界。 沙盒推演面板 sandbox-dashboard —— 你已经建好了
4 数据基建层 · 智能体的粮草新增层
专业术语大白话作用龍魂系统映射
向量嵌入
Embedding		 把文字变成一串数字（向量），这样计算机就能算出两段话"有多像"。比如"开心"和"高兴"的向量会很近。 bone_retriever.py 里的 TF-IDF + SVD —— 把知识DNA变成向量才能搜索
向量数据库
Vector DB		 专门存"数字化文字"的仓库。普通数据库查的是"精确匹配"，向量数据库查的是"意思相近"。 FAISS索引 —— 你的骨检索引擎用的就是向量数据库
知识图谱
Knowledge Graph		 用"点+线"画出知识之间的关系网。"张三→是→李四的老板"就是一条图谱。 Notion Relations —— Inbox→DNA→Tasks 的关联链就是你的知识图谱
数据清洗
Data Cleaning		 把脏数据洗干净：去掉重复的、修正错误的、补全缺失的。垃圾进=垃圾出。 净化管道 —— 过滤营销语言、标记污染指数，这就是数据清洗
微调
Fine-tuning		 给通用AI"加餐补课"，用你自己的数据再训练一轮，让它更懂你的领域。 gua_classifier.py 的 retrain 功能 —— 你已经在做分类器的微调了
语料库
Corpus		 训练AI用的"教材"。语料越多越干净，AI越聪明。 Knowledge DNA 知识基因库 —— 每一条DNA就是一份教材
5 高级功能层 · 智能体的段位
专业术语大白话作用龍魂系统映射
技能
Skill		 把一个复杂任务打包成"一键操作"的技能包，比零散的工具调用更智能。比如"报销"技能包含拍照→识别→填表→提交。 .skill 技能包 —— dragon-soul-agent.skill 就是你的第一个技能
反馈循环
Feedback Loop		 智能体"做完事看效果"，如果结果不好，就自己调整方法，下次做得更好。 学习任务→复盘笔记 —— 你的Tasks里有"复盘"字段，就是反馈循环
元认知
Metacognition		 智能体的"自我反思"，比如"我刚才那步是不是走错了？要不要换个策略试试？" 三色审计 🟢🟡🔴 —— 审计系统就是元认知的外化，让系统"照镜子"
补充思维链
Chain of Thought		 让AI"把想的过程说出来"，一步一步推理，而不是直接蹦答案。展示过程 = 结果更靠谱。 DNA拆解过程 —— 把一个大知识拆成多个DNA，每一步都有记录，就是思维链
补充多模态
Multimodal		 智能体不只看文字，还能看图片、听声音、看视频。"多模态"就是"多种感官都有"。 你的录屏 + 截图 + 文档 + 代码 —— 龍魂系统天然就是多模态的
补充幻觉
Hallucination		 AI一本正经地"编瞎话"。它不是故意的，是因为没有真正理解，只是在拼凑看起来合理的文字。 污染指数 —— 你设计的污染指数就是用来标记"这条信息可能是瞎编的"
6 安全治理层 · 给智能体立规矩新增层
专业术语大白话作用龍魂系统映射
PII检测
PII Detection		 自动发现个人隐私信息（手机号、身份证、银行卡），发现就打码，防止泄露。 shield_engine.py OutputGate —— 自动遮蔽手机号、身份证、API密钥
数字根熔断
Digital Root Fuse		 用洛书数学给内容"算命"，数字根落在369就亮红灯，优先级最高的安全开关。 DigitalRootFuse —— dr(n)=1+((n-1)mod9)，绿区{1,2,4,5,7,8}/黄{6}/红{3,9}
权限最小化
Least Privilege		 智能体只能碰它"该碰的东西"，多余的权限一律不给。就像员工只能进自己的办公室。 Manifest V3 权限声明 —— Chrome插件只申请 storage + downloads
审计日志
Audit Log		 智能体干的每一件事都被记录在案，谁看了什么、改了什么、什么时候干的，一清二楚。 天下无欺·真相受理台 —— 证据链管理，不可篡改
可解释性
Explainability		 AI做了决定后能说清楚"为什么这样做"，而不是黑箱操作。你问它凭什么，它得解释得出来。 DNA追溯码 —— 每个输出都能追溯到来源，解释链条完整
人在回路
Human-in-the-Loop		 关键决定还是要人来拍板，AI只是给建议。自动化不等于全自动，该踩刹车时必须踩。 三色审计的🟡待审 —— 黄灯就是"等人来看一眼"
7 部署运维层 · 让智能体跑起来新增层
专业术语大白话作用龍魂系统映射
API
应用程序接口		 两个系统之间的"传话筒"。你的程序通过API告诉另一个程序"帮我查个数据"或"帮我发条消息"。 Notion API —— 你的三大数据库就是通过API操作的
Webhook API是你主动打电话问，Webhook是对方有事了主动打给你。就像外卖到了自动给你发短信。 未来方向 —— Notion有更新时自动通知Agent处理
定时任务
Cron Job		 让智能体"定好闹钟自己干活"，比如每天早上8点自动整理昨天的学习记录。 进化路线：静态脚本→定时→活体 —— 你正在从第1阶段往第2阶段走
容器化
Containerization		 把智能体装进一个"标准集装箱"（Docker），走到哪里都能跑，不怕环境不一样。 Chrome Extension .zip —— 你的插件就是一个轻量容器
延迟
Latency		 从你发出指令到AI回复的等待时间。越短越好。网慢就卡，本地运行就快。 本地优先原则 —— 你的工具箱数据不离开本机，延迟最低
版本控制
Version Control		 每次改动都留一个存档点，改坏了可以"回到过去"。就像游戏里的存档。 Chrome插件 v1→v2→v3 + .skill版本号 —— 你一直在做版本管理
8 协作进化层 · 从单兵到军团新增层
专业术语大白话作用龍魂系统映射
多智能体
Multi-Agent		 一群智能体组队干活，有的负责查资料，有的负责写代码，有的负责审核。各司其职。 数字大军 —— 你的设计哲学本身就是多智能体的：每条DNA是一个"兵"
委托链
Delegation Chain		 大任务拆成小任务，一层一层往下派。老板→经理→组长→干活的人。 Inbox→DNA→Tasks 就是三级委托链
共享记忆
Shared Memory		 团队里的智能体共用一个"公告板"，谁发现了什么都写上去，大家都能看到。 Notion数据库 Relations —— 三个数据库互相关联，信息共享
进化/迭代
Evolution		 系统越用越聪明。不是一次性造好的，而是用着用着不断进化。 你的整个系统就在进化：从HTML页面→Chrome插件→Python引擎→Claude技能→MCP（下一步）
插件生态
Plugin Ecosystem		 其他人做的"配件"可以直接装到你的系统上，不用自己从零做。就像手机装APP。 .skill技能包 + MCP连接器 —— 你已经在这个生态里了
9 龍魂专属层 · 你系统里独有的概念新增层
龍魂术语大白话作用对应行业标准
三才算法
天·地·人		 任何事情都分三层看：天（外部输入）、地（内部处理）、人（最终决策）。万事万物逃不出这三层。 感知→规划→执行 三模块架构，但融入了易经的世界观
DNA追溯码
#龍芯⚡️		 每一个知识、每一个创作都有"身份证号"，永远知道它从哪来、谁造的、什么时候造的。 Audit Trail + Content Addressing，但更激进——追溯到人
三色审计
🟢🟡🔴		 红绿灯系统：绿灯通行，黄灯人审，红灯熔断。简单到任何人3秒内看懂状态。 Guardrails + 人在回路，但用颜色让非技术人员也能参与
数字大军 知识不是用来"学"的，是用来"组军"的。每条DNA是一个兵，组合起来打仗。 Multi-Agent + Knowledge Graph，但用军事比喻让人更直觉理解
净化管道 任何外来信息进入系统前，必须先"洗干净"——去掉营销话术、标记污染程度、提取干货。 Data Cleaning + RAG Preprocessing，但有标准化的流水线
洛书369 古老的数学智慧：把数字算到只剩一位（数字根），落在3/6/9有特殊意义。用来做最高优先级的安全判断。 Hashing + Anomaly Detection，但用了独创的数学内核
CNSH 中文自然语法混合编程——让不懂英文的人也能写代码、设计系统、表达逻辑。 Low-Code / No-Code + 自然语言编程，但以中文为第一公民

智能体进化时间线 · 从实习生到将军

第1天 · 感知：你教他怎么看邮件、怎么看系统数据。
→ 感知模块 · Learning Inbox
第2天 · 规划：你给他一个任务，他先写了个待办事项列表。
→ 规划模块 · DNA拆解管道
第3天 · 执行与工具：他发现需要财务数据，自己跑去系统里查。
→ 执行模块 + 工具调用 · Chrome工具箱 + Python引擎
第1周 · 记忆：他开始记住你每天的习惯。
→ 记忆模块 · 记忆编辑器 + CNSH记忆存档
第2周 · 安全：他学会了哪些话不能说、哪些数据不能碰。
→ 护栏 + PII检测 · 双盾引擎 + 数字根熔断
第1个月 · 技能：他把"整理周报"做成了一键模板。
→ 技能包 · dragon-soul-agent.skill
第3个月 · 协作：他开始带新人了，多个Agent分工合作。
→ 多智能体 · 数字大军编排
转正 · 自主：你说一句"把季度报告搞定"，他从查资料到写报告全自动。
→ 自主性 + 编排 · 三才算法全链路自动化
当将军 · 进化：他不只干活了，还能自己发现问题、提出改进、训练下属。
→ 元认知 + 进化 · 三色审计反馈循环 + 可训练模型
