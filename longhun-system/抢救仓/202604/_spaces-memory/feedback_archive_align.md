## 三步闭环

### Step 1 · 冻结（立规矩·0分钟）
所有 `~/Documents` / `~/longhun-system` 里的 `.md` / `.html` / `.txt` 原地冻结——**不删·不搬·不改·不重命名**。它们是母本的前世·是UID9622创世纪的考古层。

### Step 2 · 隔离（15分钟·一次性）
新建 `~/Documents/📦 _ARCHIVE_PRE_NOTION_MERGE_20260418/` 文件夹·把今晚以前所有 `.md` / `.html` 拖进去。Swift / Python / TS 等可执行代码**不动**（那是肌肉·不是考古）。

**隔离禁忌**：
- ❌ 不要一次性整理 `_ARCHIVE` 内部结构（陷阱）
- ❌ 不要建索引·不要做 TOC（让它自然沉淀）
- ❌ 不要删重复文件（`README.md` 和 `README 2.md` 都是版本考古证据）

### Step 3 · 按需出土（日常·每次5分钟）
每次遇到一个主题时：
1. **先 Notion 搜** → 有就用 Notion 的·直接收工
2. **Notion 没有** → 去 `_ARCHIVE` 捞对应那份 `.md`
3. **读一眼** → 有价值的精华进 Notion（不是整份·是精华）
4. **提炼完** → `.md` 留在考古层·不删
