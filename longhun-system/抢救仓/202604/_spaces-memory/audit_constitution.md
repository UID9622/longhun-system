### 宪法铁律（不可对冲）
- **R1** 不拦截只标记 · **R2** 不审判只追溯 · **R3** 不抓人要改人
- **R4** 🔴 熔断不可对冲 · **R5** 必须有申诉通道
- **R6** 上下文不可忽略（≥10条）· **R7** 审计链不可篡改（WAL/区块链）
- **R8** 气话必须有缓冲（延迟 2h）· **R9** 水军举报必须降权
- **R10** 陪伴保护 > 监控责备

### 写死参数
- 上下文窗口：**10 条**（5~50）· 反讽阈值：**0.65**（0.5~0.8）
- 气话特征：命中 ≥ 2 项（感叹号密集 + 第一人称情绪词 + 短时高频）
- 延迟判定：**2 小时** · 审计存储：不可篡改日志
- 三色阈值：🟢 > 90% / 🟡 50~90% / 🔴 < 50%
- 贡献公式：`调用×40% + 准确×30% + 信任×30% - 警告×5 - 熔断×20`
- 对冲规则：积分 ≥ 5 可抵消 1 次警告（积分-1），🔴 熔断永不可对冲

### 申诉公证
- 任何 🟡🔴 判定 → 自动生成申诉 ID，不得无通道
- 24h 内必须响应；公证团 = 5 资深用户 + 2 平台人员
- 申诉成功 → 误判案例入模型微调数据集
- 不允许「申诉失败后加重处罚」—— 申诉是权利不是赌博
