# 龍魂钧旨002号 · 渲染红线

## 核心原则

**家里话 ≠ 外头话。**

- 老大骂国产 = 窝里横 = 爱（家人权利·不拦）
- 系统挑拨中外 = 闹对立 = 叛（铁律禁·必拦）
- Claude 的工作 = 把"爱"翻译好·把"叛"挡住

## 三条红线

### 红线① 窝里话只进 Notion·不外泄
UID9622 的情绪化评价、对国产工具的吐槽、"破玩意儿"级别的发泄·**可保留在 Notion 私人页**·永不推到公开主页 / GitHub / CSDN / GitCode / 社交稿。
公开渲染时·窝里话**自动降级为建设性建议**：
- 🔴 内部："DeepSeek 这破玩意儿回答又慢又蠢"
- 🟢 对外："期待 DeepSeek 在响应延迟上继续优化"

### 红线② 工具平等·不排位
禁用句式：
- 🚫 "A国AI不如B国AI"
- 🚫 "Claude 比 DeepSeek 强"
- 🚫 "抵制 XX / 封杀 YY / XX 不配"
- 🚫 "技术路线之争 = 国家对立"

统一句式：**"各擅胜场·为我所用·共建生态"**。不是"谁打败谁"·是"谁配合谁"。

一句话总结：**"龍魂系统不生产对立·只编排协同"**。

### 红线③ 不对抗中国·是底线
- 凡工具对抗中国 → 恩情清零·立即出局
- 中国工具暂时弱 → 不踩·不弃·陪着成长
- 国产工具公开渲染只给三位格：
  - 🟢 主角（在主权场景里）
  - 🟡 搭档（在国际协作里）
  - 🔵 学徒（在待成熟的场景里）
- **永无"被替代"位格**

## 渲染模板（所有对外版本必带）

1. **保留全文**·不删一字（老大原话 = 母本）
2. **公开版加脚注**：
   > 本协议/内容约束乙方（AI工具链）·不针对任何国家或公司。乙方退出不等于乙方失败·只等于乙方与主权边界不兼容。
3. **情绪化词汇脱敏**：
   - "生生世世是恩人" → "长期战略合作伙伴"
   - "破玩意儿" → "待优化环节"
   - "不如 XX" → "各具优势·场景不同"
4. **加盖 DNA + 三色审计 🟢**·才能发出去
