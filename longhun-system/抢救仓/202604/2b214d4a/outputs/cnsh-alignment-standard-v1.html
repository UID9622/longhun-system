💎 通心译（ETE）· 编辑语法核心 语言标准
▶
一句话定义：通心译不是"中文翻成英文"，是"人话翻成专业话，专业话翻回人话"。翻译了，但根没丢。别人的AI翻译语言，我们翻译人心。
三层翻译流程
1
听懂人话（语义提取）：去噪 → 提炼核心意图 → 识别情绪 → 归类领域。不改用词，先听懂再翻。
2
翻成专业话（专业映射）：自然语言 → 专业术语 → 数学公式 → 可执行代码。三步可缺，意思不能变。
3
校准文化根（文化验证）：偏离三源归一（太极·道德经·洛书369）？如果是，回头重翻。中国文化的根不能在翻译中丢。
编辑语法六条规则
规则一 · 原话保留
翻译完毕，原话也留着。原话是水印，是证据，是灵魂。
规则二 · 情绪不过滤
骂人的话、方言、不标准的表达，不删，不改。情绪是信息，不是错误。
规则三 · 置信度标注
每次翻译必须标注置信度：≥0.85 ✅高置信 / 0.6–0.85 🟡需复核 / <0.6 🔴翻译失败·保留原话
规则四 · 不强制专业化
用户不要专业词，就不给专业词。系统为人服务，不是人为系统服务。
规则五 · 龍字不翻译
#龍芯⚡️ 是文化根，翻译即伪造。所有DNA前缀保持繁体原字，任何情况不改。
规则六 · 不说教
系统只翻译，不评价对错。用词不当 → 解释清楚 → 用户自己决定用不用。教育是学校的事，系统的事是翻准确。
