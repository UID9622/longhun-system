# 快速开始 - 3步连接 Notion

## 第1步：获取 API 密钥 🔑

1. 打开浏览器，访问：**https://www.notion.so/my-integrations**
2. 点击 **"+ New integration"** 按钮
3. 填写名称：**龍魂助手**
4. 点击 **Submit**
5. **复制显示的密钥**（格式：`secret_xxxxxxxx...`）

---

## 第2步：给助手授权 ✅

1. 打开你存放 **26张卡片** 的 Notion 页面
2. 点击右上角 **"..."** 菜单
3. 选择 **"Add connections"**
4. 选择 **"龍魂助手"**
5. 点击 **Confirm**

---

## 第3步：获取数据库 ID 🆔

1. 打开 Notion 数据库页面
2. 看浏览器地址栏，网址像这样：
   ```
   https://www.notion.so/xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx?v=...
   ```
3. **复制中间那一长串字母和数字**（32位）
4. 那就是你的数据库 ID

---

## 第4步：填写到代码里 📝

1. 打开 **Notion助手.swift** 文件
2. 找到第 10-13 行：
   ```swift
   private let API密钥 = "在这里粘贴你的 secret_xxxx 密钥"
   private let 数据库ID = "在这里粘贴你的数据库ID"
   ```
3. 把你复制的密钥和 ID 粘贴进去
4. 保存文件

---

## 第5步：测试一下 🚀

在代码最后面运行：
```swift
await 使用示例()
```

如果看到：
```
✅ 成功读取 26 张卡片
```

**恭喜！成功连接 Notion！** 🎉

---

## 遇到问题？

### 问题1：找不到"Add connections"
**解决**：确保你打开的是数据库页面，不是普通页面

### 问题2：显示 HTTP 401 错误
**解决**：
- 检查 API 密钥是否正确
- 确认已经给助手授权（第2步）

### 问题3：显示 HTTP 404 错误
**解决**：数据库 ID 不对，重新复制

### 问题4：我的卡片不在数据库里，在普通页面
**解决**：
- 告诉我你的 Notion 结构
- 或者把页面转成数据库
