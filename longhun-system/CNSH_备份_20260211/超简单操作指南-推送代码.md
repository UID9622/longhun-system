## 📦 如果你想推送 GitHub

### 第一步：配置 GitHub（只需要做一次）

```bash
chmod +x 配置GitHub远程仓库.sh

# 替换成你的 GitHub 仓库地址
./配置GitHub远程仓库.sh https://uid9622:<你的token>@github.com/uid9622/dragonsoul-system.git
```

**如何获取 GitHub Token？**
1. 登录 GitHub
2. 点击头像 → Settings
3. 左侧菜单 → Developer settings
4. Personal access tokens → Tokens (classic)
5. Generate new token → 选择权限（repo）
6. 复制生成的 token

### 第二步：一键推送（每次修改后）

```bash
./一键推送全平台.sh
```

**完成！** 自动推送到 Gitee + GitHub + CSDN

### Q5: 如果推送失败怎么办？

**答：脚本会自动停止并显示错误信息**

常见错误和解决方法：

**错误1：账号密码错误**
```
remote: Invalid username or password
```
**解决方法**：检查 Gitee/GitHub 的用户名和密码/token 是否正确

**错误2：权限不足**
```
remote: Permission denied
```
**解决方法**：检查你是否有该仓库的推送权限

**错误3：网络问题**
```
fatal: unable to access
```
**解决方法**：检查网络连接，稍后重试
