## 🚀 使用流程

### 方式1: 一键发布（推荐）

```bash
# 1. 准备文件
mkdir -p ./私密层
cp -r 你的源代码/* ./私密层/

# 2. 执行发布
chmod +x 一键开源发布.sh
./一键开源发布.sh v1.0.0

# 3. Lucky 最终签名
gpg --detach-sign --armor dragonsoul-system-v1.0.0-release.tar.gz

# 4. 发布到 GitHub
gh release create v1.0.0 dragonsoul-system-v1.0.0-release.tar.gz
```

### 方式2: 分步执行（调试用）

```bash
# 步骤1: 生成公开层
python3 open_source_release_helper.py generate-public ./私密层 ./公开层

# 步骤2: 扫描敏感信息
python3 open_source_release_helper.py scan-sensitive ./公开层

# 步骤3: 运行检查清单
python3 open_source_release_helper.py checklist ./公开层

# 步骤4: 构建发布包
python3 open_source_release_helper.py build ./公开层 --version v1.0.0

# 步骤5: 验证发布包
python3 open_source_release_helper.py verify dragonsoul-system-v1.0.0-release.tar.gz
```
