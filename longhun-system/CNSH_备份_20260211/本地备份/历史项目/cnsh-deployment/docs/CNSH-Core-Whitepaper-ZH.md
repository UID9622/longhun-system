## 三、技术架构

**本地优先架构：**

```
用户设备（本地）
├── Obsidian（知识管理前端）
├── Ollama（本地大语言模型）
├── 加密存储层（AES-256-GCM）
├── DNA身份验证（生物特征+设备指纹）
└── 备份机制（本地+可选加密云端）
```

**核心特点：**

- **不依赖境外云服务**：所有核心功能可完全离线运行
- **端到端加密**：数据传输采用TLS 1.3 + 国密SM2/SM3/SM4
- **可审计性**：所有操作生成DNA确认码，形成不可篡改的审计链
- **可迁移性**：数据格式开放，支持随时导出和平台切换
