# 📚 DNA编码系统 - 使用指南

## 🚀 快速开始

### 环境要求

**系统要求**:
- Python 3.8+
- 操作系统: Linux/macOS/Windows
- 内存: 至少4GB RAM
- 存储: 至少1GB可用空间

**依赖软件**:
- Ollama (用于AI引擎)
- Git (可选，用于版本控制)

### 安装步骤

#### 1. 克隆项目
```bash
git clone <项目地址>
cd dna-system
```

#### 2. 安装依赖
```bash
pip install -r requirements.txt
```

#### 3. 配置系统
```bash
# 编辑配置文件
cp config.example.yaml config.yaml
vim config.yaml
```

#### 4. 启动系统
```bash
# 方式1: 使用启动脚本
./startup_dna_system.sh

# 方式2: 直接运行
python integrated_system.py
```

## 💡 基本使用

### DNA标签生成

```python
from dna_verifier import CNSHDNAVerifier

# 初始化验证器
verifier = CNSHDNAVerifier()

# 生成DNA标签
dna_info = verifier.generate_dna("AIENGINE")
print(f"DNA标签: {dna_info['dna_code']}")
print(f"LU序列: {dna_info['lu_sequence']}")
```

### 记忆存储操作

```python
from dna_memory_system import DNAMemorySystem

# 初始化记忆系统
memory_system = DNAMemorySystem()

# 存储记忆
memory_content = "这是需要存储的重要信息"
result = memory_system.store_memory(
    content=memory_content,
    memory_type="DIALOG",
    priority="P1"
)

print(f"记忆存储成功，LU序列: {result['lu_sequence']}")
```

### 记忆检索操作

```python
# 通过LU序列号检索记忆
lu_sequence = "LU-2025-12-00001-ZHUGEXIN-DIALOG"
memory = memory_system.recall_memory(lu_sequence)

if memory:
    print(f"记忆内容: {memory['content']}")
    print(f"存储时间: {memory['metadata']['timestamp']}")
else:
    print("未找到对应的记忆")
```

## 🔧 高级功能

### 自定义配置

#### 配置文件示例 (config.yaml)

```yaml
# DNA系统配置
dna_system:
  # DNA验证配置
  verifier:
    default_content_type: "MEMORY"
    sequence_start: 1
    hash_algorithm: "sha256"
    
  # 记忆存储配置
  memory:
    storage_backend: "local"  # local, sqlite, mysql
    storage_path: "./memory_storage/"
    retention_days: 365
    
  # 系统集成配置
  integration:
    ollama_endpoint: "http://localhost:11434"
    ollama_model: "qwen2.5:72b"
    
  # 日志配置
  logging:
    level: "INFO"
    file_path: "./logs/dna_system.log"
```

### 多存储后端支持

```python
# 使用SQLite后端
memory_system = DNAMemorySystem(storage_backend="sqlite")

# 使用MySQL后端  
memory_system = DNAMemorySystem(storage_backend="mysql")

# 使用云存储后端
memory_system = DNAMemorySystem(storage_backend="cloud")
```

### 批量操作

```python
# 批量存储记忆
memories = [
    {"content": "记忆1", "type": "DIALOG", "priority": "P1"},
    {"content": "记忆2", "type": "KNOWLEDGE", "priority": "P2"},
    {"content": "记忆3", "type": "TASK", "priority": "P3"}
]

for memory in memories:
    result = memory_system.store_memory(**memory)
    print(f"存储成功: {result['lu_sequence']}")
```

## 🛠️ 命令行工具

### 系统管理命令

```bash
# 启动系统
python integrated_system.py start

# 停止系统
python integrated_system.py stop

# 查看系统状态
python integrated_system.py status

```

### DNA操作命令

```bash
# 生成DNA标签
python dna_verifier.py generate --type AIENGINE

# 验证DNA标签
python dna_verifier.py verify --dna "#CNSH-ZHUGEXIN..."

# 批量验证
python dna_verifier.py batch-verify --file dna_list.txt
```

### 记忆管理命令

```bash
# 存储记忆
python dna_memory_system.py store --content "记忆内容" --type DIALOG

# 检索记忆
python dna_memory_system.py recall --lu LU-2025-12-00001

# 导出记忆
python dna_memory_system.py export --output memories.json

# 清理过期记忆
python dna_memory_system.py cleanup --days 30
```

## 🔍 故障排除

### 常见问题

#### 1. 系统启动失败
```bash
# 检查依赖是否安装
pip list | grep ollama

# 检查Ollama服务状态
ollama list

# 查看日志文件
cat logs/dna_system.log
```

#### 2. DNA验证失败
```bash
# 检查DNA格式
python dna_verifier.py verify --dna "你的DNA标签"

# 查看验证规则
cat config.yaml | grep verifier
```

#### 3. 记忆存储失败
```bash
# 检查存储路径权限
ls -la ./memory_storage/

# 检查磁盘空间
df -h

# 查看存储配置
cat config.yaml | grep storage
```
