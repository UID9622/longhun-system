## 🌟 概述

CNSH (Chinese Natural Semantic Humanity) 是一套基于中文自然语言的编程系统，融合了中华五千年文化元素与现代编程技术。CNSH IDE是这套系统的集成开发环境，提供了完整的编写、编译、调试和执行功能。

## 🎯 核心特性

- **中文语法编程**: 直接使用中文表达编程逻辑，无需学习英文关键词
- **易经64卦系统集成**: 将64卦作为状态机和决策系统
- **CNSH字节码(CNCODE)**: 高效的中文编程字节码执行引擎
- **可视化调试**: 追踪因果链，实时显示变量状态和卦象变化
- **世界编辑器**: 可视化构建元宇宙场景
- **100+中文标准函数**: 涵盖数学、逻辑、文本、列表、存储、世界操作等领域

## 🚀 快速开始

### 1. 启动IDE

```bash
cd CNSH_IDE_and_Compiler
python3 start_ide.py
```

启动成功后，浏览器会自动打开IDE界面。

### 2. 编写CNSH代码

在IDE的代码编辑器中输入以下示例代码：

```cnsh
设 温度 为 35
若 温度 高于 30 则
  输出 "开启空调"
否则
  输出 "温度适宜"
```

### 3. 运行代码

点击"运行代码"按钮，在控制台查看执行结果和因果链。

## 📘 CNSH语法指南

### 基本语法

#### 赋值语句
```cnsh

```

#### 条件判断
```cnsh
若 条件 则
  执行动作
否则
  执行其他动作
```

#### 循环语句

## 🔮 卦象系统

CNSH系统集成了易经64卦，每个卦象都有对应的执行规则：

```json
{
  "䷀": {
    "名称": "乾",
    "卦辞": "元，亨，利，贞。",
    "象辞": "天行健，君子以自强不息。",
    "执行规则": {
      "触发条件": ["遇阻", "遇机"],
      "推荐动作": ["强行突破", "主动出手"]
    }
  }
}
```

在CNSH代码中，可以通过卦象来实现状态判断和决策：

```cnsh
设 当前状态 为 ䷀  # 乾卦
若 当前状态 等于 ䷀ 则
  输出 "天行健，君子以自强不息"
```

## 🌍 世界编辑器

CNSH世界编辑器提供了可视化的元宇宙构建工具：

1. **对象管理**: 创建和管理世界中的对象
2. **状态设置**: 为对象设置卦象状态
3. **事件触发**: 设置对象间的事件关系
4. **关系连接**: 建立对象间的各种关系

通过世界编辑器创建的场景会自动生成对应的CNSH代码。

## 📊 系统架构

CNSH系统采用分层架构：

```
CNSH代码
  ↓ (分词)
Token流
  ↓ (解析)
AST抽象语法树
  ↓ (生成)
CNCODE字节码
  ↓ (执行)
虚拟机执行
  ↓ (输出)
结果 + 因果链 + 卦象变化
```

## 🎯 设计哲学

CNSH系统的设计遵循"不绕、不碎、不跳、不丢"的原则：

- **不绕**: 直接使用中文表达逻辑，无需绕到英文思维
- **不碎**: 系统完整统一，避免碎片化
- **不跳**: 循序渐进，不跳跃步骤
- **不丢**: 全面融合传统文化，不丢失文化根基
