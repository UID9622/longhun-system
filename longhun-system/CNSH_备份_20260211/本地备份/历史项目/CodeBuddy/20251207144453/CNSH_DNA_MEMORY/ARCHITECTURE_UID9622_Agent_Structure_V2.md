1. **主数据库**：派生智能体管理
   - 属性：人格名称、功能描述、权限等级、协作关系、状态

2. **卡片管理数据库**：知识卡片管理
   - 属性：卡片类型、关键词、内容、来源、创建时间、引用计数
   - 视图：按类型分类视图、关键词云视图、引用热度视图

3. **任务管理数据库**：执行任务追踪
   - 属性：任务名称、负责人格、创建时间、状态、结果、反馈
   - 视图：进行中任务视图、已完成任务视图、异常任务视图

4. **系统日志数据库**：操作记录
   - 属性：时间、操作类型、负责人格、详细信息、结果
   - 视图：按操作类型视图、按负责人格视图、异常日志视图
