## 🔍 故障排除

### 端口被占用
- 启动脚本会自动查找可用端口 (8080-8090)
- 如果全部被占用，请手动指定端口

### 模块导入错误
```bash
# 确保安装了所有依赖
pip install -r requirements.txt

# 确保虚拟环境已激活
source venv/bin/activate
```

### API Key 错误
- 使用默认的 API Key: `UID9622-SECRET-KEY`
- 或在代码中修改为您自己的 Key
