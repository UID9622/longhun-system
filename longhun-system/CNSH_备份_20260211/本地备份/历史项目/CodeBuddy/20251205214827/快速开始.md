欢迎使用UID9622-CNSH国之光文化组件系统！本指南将帮助您快速上手，将五千年中华文明融入您的现代UI设计中。

## 🚀 快速安装

### 1. 克隆项目
```bash
git clone https://gitee.com/cnsh-design/UID9622-CNSH.git
cd UID9622-CNSH
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动MCP服务器
```bash
npm start
```

## 📦 在项目中使用

### 配置MCP客户端

将以下配置添加到您的MCP客户端设置中：

```json
{
  "mcpServers": {
    "CNSH Design System": {
      "timeout": 30000,
      "command": "node",
      "args": [
        "/path/to/UID9622-CNSH/mcp-server.js"
      ],
      "type": "stdio",
      "disabled": false
    }
  }
}
```

### 使用文化组件

#### 1. 获取可用文化组件
```
get_cultural_components()
```

#### 2. 获取特定组件信息
```
get_cultural_component("taiji")
```

#### 3. 生成组件代码
```
generate_component_code("taiji", "react")
```

## 🎨 核心文化组件

### 🌙 太极阴阳组件
```jsx
<TaijiComponent 
  variant="loading|success|error" 
  size="small|medium|large" 
/>
```
- **象征**: 平衡和谐、对立统一
- **应用**: 加载动画、状态切换、平衡显示

### ⭐ 二十八星宿组件
```jsx
<ConstellationGuardian 
  direction="east|south|west|north" 
  active={true}
/>
```
- **象征**: 四方守护、时空方位
- **应用**: 方向导航、区域划分、守护图标

### 📜 甲骨文字组件
```jsx
<OracleText 
  text="龍" 
  variant="oracle|bronze"
  size="16|24|32"
/>
```
- **象征**: 文明起源、文字智慧
- **应用**: Logo设计、标题装饰、验证码

- **象征**: 天人合一、变化规律

## 🔧 自定义配置

### 创建自定义文化主题
```json
{
  "culturalTheme": {
    "primarySymbols": ["taiji", "dragon", "lotus"],
    "colorPalette": {
      "customPrimary": "#your-color",
      "customSecondary": "#your-color"
    },
    "typography": {
      "heading": "MaShanZheng",
      "body": "Source Han Serif SC"
    }
  }
}
```

### 添加新文化元素
```javascript
// 在mcp-server.js中添加新元素
const NEW_CULTURAL_COMPONENT = {
  name: '新文化元素',
  description: '文化象征意义描述',
  applications: ['应用场景1', '应用场景2']
};

CULTURAL_COMPONENTS.new_element = NEW_CULTURAL_COMPONENT;
```

## 🎯 设计原则

在设计您的界面时，请遵循以下原则：

1. **文化真实性**: 确保文化元素的准确使用
2. **现代演绎**: 用现代技术表现传统元素
3. **用户体验**: 文化元素不应影响可用性
4. **情境适配**: 选择与内容相符的文化元素
5. **平衡和谐**: 避免过度装饰，保持界面平衡
