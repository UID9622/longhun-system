# 太极八卦易经甲骨文开源系统

## 项目简介

本项目融合了中国古代哲学智慧与现代Web技术，打造一个完全开源的文化传承与创新平台。

### 核心特色

- 🌓 **太极图可视化** - 动态交互式太极图生成
- ☯️ **八卦演算系统** - 完整的八卦、六十四卦推演算法
- 📜 **甲骨文字库** - 内置甲骨文数字化字库与查询系统
- 🔮 **易经占卜** - 传统易经占卜功能的现代化实现
- 🎨 **艺术化界面** - 融合传统美学与现代设计

## 技术栈

- **后端**: Node.js + Express + SQLite
- **前端**: HTML5 Canvas + JavaScript ES6+
- **数据**: JSON格式甲骨文字库 + SQLite数据库
- **部署**: 支持本地部署和云端部署

## 快速开始

```bash
# 克隆项目
git clone https://gitee.com/open-source/taichi-jiagua-oracle.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
├── src/
│   ├── core/           # 核心算法
│   │   ├── bagua.js   # 八卦演算
│   │   ├── yijing.js  # 易经逻辑
│   │   └── oracle.js  # 甲骨文处理
│   ├── visual/        # 可视化组件
│   │   ├── taichi.js  # 太极图绘制
│   │   └── bagua.js   # 八卦图绘制
│   └── data/          # 数据文件
│       ├── oracle.json # 甲骨文字库
│       └── yijing.json # 六十四卦数据
├── public/             # 前端资源
├── docs/              # 文档
└── examples/          # 示例代码
```
