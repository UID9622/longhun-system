# 🌟 太极八卦易经甲骨文开源系统 - 快速开始

## 🚀 一键启动

```bash
# 克隆项目
git clone https://gitee.com/open-source/taichi-jiagua-oracle.git
cd taichi-jiagua-oracle

# 运行启动脚本
./start.sh
```

或者手动启动：

```bash
npm install
npm start
```

然后在浏览器中访问：`http://localhost:3000`

### 1. 太极图演示 🌓
- **动态旋转**：点击"开始旋转"观看太极图旋转动画
- **装饰模式**：显示环绕八卦符号的装饰性太极图

- **先天八卦**：伏羲八卦，展示宇宙本源秩序
- **后天八卦**：文王八卦，体现万物变化规律

- **详细信息**：每个字符都有详细的历史背景和解释

## 📚 API 使用

### 基础请求示例

```javascript
// 获取甲骨文字符
fetch('/api/oracle/characters')
  .then(res => res.json())
  .then(data => console.log(data));

// 执行易经占卜
fetch('/api/yijing/divine', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ method: 'three_coins' })
})
  .then(res => res.json())
  .then(data => console.log(data));
```

### 主要接口

- `/api/oracle/characters` - 获取所有甲骨文字符
- `/api/oracle/search?keyword=关键词` - 搜索字符
- `/api/yijing/divine` - 执行占卜
- `/api/yijing/gua` - 获取六十四卦
- `/api/info` - 系统信息
