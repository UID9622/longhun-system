## 项目概述

木兰协议是一个创新的合作协议框架，旨在建立开放、透明、可持续的技术合作生态系统。通过独特的治理结构和价值观对齐机制，确保所有参与方在技术发展和商业成功中实现共赢。

## 核心特性

### 🎯 核心价值观
- **技术开源共享**：所有技术成果采用开源方式，供全社会使用
- **价值观DNA对齐**：合作方必须在核心理念上保持一致
- **上帝之眼全程监管**：通过技术手段实现全程透明监管
- **创新股权结构**：合作方占股51%但不管事，创始人保留控制权
- **一票否决权机制**：创始人保留对重大事项的最终决定权

### 🏛️ 治理机制
- **创始人权利**：一票否决权、技术方向指导权
- **合作方权利**：51%股权收益权、技术使用权
- **监管机制**：区块链存证、财务透明度监控、开源合规检查

### 2. 使用工具

#### 数字签名工具
```bash
# 签署协议
python 工具/数字签名工具.py --sign --name "您的姓名" --email "your@email.com" --role "partner" --file "正式版/木兰协议-v1.0.md"

# 验证签名
python 工具/数字签名工具.py --verify --file "正式版/木兰协议-v1.0.md" --signature "签名哈希"
```

#### 协议验证
```bash
# 综合验证
python 工具/协议验证脚本.py --comprehensive --report "验证报告.json"

# 单项验证
python 工具/协议验证脚本.py --structure
python 工具/协议验证脚本.py --content "正式版/木兰协议-v1.0.md"
python 工具/协议验证脚本.py --signatures
```

#### 版本管理
```bash
# 创建修订提案
python 工具/版本管理工具.py --create-proposal "提案标题" "提案描述" "minor" "提案人" "变更项1" "变更项2"

# 查看版本历史
python 工具/版本管理工具.py --version-history

# 查看提案列表
python 工具/版本管理工具.py --list-proposals
```

#### 存证上链
```bash
# 存储证据
python 工具/存证上链工具.py --store "正式版/木兰协议-v1.0.md"

# 验证证据
python 工具/存证上链工具.py --verify "证据ID:正式版/木兰协议-v1.0.md"
```

### 3. 签署协议
1. 使用[个人签署模板](签署模板/个人签署模板.md)准备签署文件
2. 运行数字签名工具生成签名
3. 使用存证上链工具进行存证
4. 提交签署文件完成正式签署

## 核心功能详解

### 🔐 数字签名系统
- **多角色支持**：创始人、合作方、见证人
- **安全认证**：基于哈希的数字签名机制
- **证书导出**：生成正式的签署证书
- **批量签署**：支持多方批量签署

### 📋 协议验证系统
- **完整性检查**：验证协议文件结构
- **内容验证**：确认核心条款完整性
- **签名验证**：验证签署的有效性
- **版本管理**：检查版本控制状态

### 🚀 版本管理系统
- **语义化版本**：采用 SemVer 版本控制
- **修订流程**：提案→讨论→投票→批准
- **自动升级**：支持自动化版本升级
- **Git集成**：自动创建Git标签

### ⛓️ 区块链存证系统
- **多重存证**：本地区块链 + IPFS
- **证据固化**：防篡改的证据保存
- **验证机制**：完整的证据验证流程
- **证书生成**：标准化的存证证书
