## 🔥 关键依赖和坑点
- **Python版本**：需要3.8+
- **必需库**：cryptography, sqlite3, requests
- **会炸的地方**：
  - 主权密钥丢失 → 系统变砖
  - DNA对齐失败 → 自动触发紧急否决
  - 区块链连接失败 → 存证无效
