## 生物特征提取器 API

### 类：`生物特征提取器`

#### 方法：`提取指纹特征(指纹图像路径: str) -> str`

提取指纹特征点并生成哈希值。

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 指纹图像路径 | str | 是 | 指纹图像文件路径（推荐500x500分辨率） |

**返回值**：

```python
"1a2b3c4d5e6f..."  # 128位十六进制哈希字符串
```

**示例**：

```python
from core.生物特征提取器 import 生物特征提取器

提取器 = 生物特征提取器()
哈希 = 提取器.提取指纹特征("fingerprint.jpg")
print(f"指纹哈希: {哈希}")
```

---

#### 方法：`提取面部特征(面部图像路径: str) -> str`

提取面部特征编码并生成哈希值。

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 面部图像路径 | str | 是 | 正面清晰面部照片 |

**返回值**：

```python
"1a2b3c4d5e6f..."  # 128位十六进制哈希字符串
```

**示例**：

```python
哈希 = 提取器.提取面部特征("face.jpg")
print(f"面部哈希: {哈希}")
```

---

#### 方法：`组合生物特征(指纹哈希: str, 面部哈希: str, 虹膜哈希: str = None) -> str`

组合多个生物特征哈希。

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 指纹哈希 | str | 是 | 指纹特征哈希 |
| 面部哈希 | str | 是 | 面部特征哈希 |
| 虹膜哈希 | str | 否 | 虹膜特征哈希（可选） |

**返回值**：

```python
"a1b2c3d4e5f6789012345678901234567890abcdef"  # 64位十六进制哈希
```

## 64卦映射器 API

### 类：`易经64卦映射器`

#### 方法：`哈希到卦象(哈希值: str, 卦数: int = 8) -> list`

将哈希值映射到卦象序列。

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 哈希值 | str | 是 | 十六进制哈希字符串 |
| 卦数 | int | 否 | 生成的卦象数量（默认8个） |

**返回值**：

```python
["乾", "坤", "屯", "蒙", "需", "讼", "师", "比"]
```

---

#### 方法：`生成卦象ID(生物特征哈希: str, 身份证号: str) -> dict`

生成完整的卦象身份ID。

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 生物特征哈希 | str | 是 | 组合生物特征哈希 |
| 身份证号 | str | 是 | 合法身份证件号 |

**返回值**：

```python
{
    "身份哈希": "a1b2c3d4e5f6...",
    "卦象序列": ["乾", "坤", "屯", ...],
    "卦象ID": "乾-坤-屯-蒙-...",
    "卦象数量": 8
}
```

```python
    "全球互认ID": "GLOBAL-CN-...",
    "本地身份系统": "网络身份证"
```


## 错误处理

所有API方法在遇到错误时会抛出异常。常见异常类型：

- `ValueError`：输入参数无效
- `FileNotFoundError`：文件不存在
- `RuntimeError`：运行时错误

**示例**：

```python
try:
    生成器 = 龍魂永世唯一ID生成器()
    结果 = 生成器.生成龍魂ID(
        身份证号="110101199001011234"
    )
except ValueError as e:
    print(f"参数错误: {e}")
except Exception as e:
    print(f"发生错误: {e}")
```
