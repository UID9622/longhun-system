## 生成龙魂ID

### 交互式生成

1. 运行主程序
2. 选择 `1. 生成新身份`
3. 输入身份证号
4. 输入国家代码（默认CN）
5. 系统生成龙魂ID
6. 导出证书（可选）

### 编程方式生成

```python
from 龙魂ID生成器 import 龙魂永世唯一ID生成器

生成器 = 龙魂永世唯一ID生成器()

# 生成ID
结果 = 生成器.生成龙魂ID(
    身份证号="110101199001011234",
    国家代码="CN",
    使用示例模式=True
)

print(f"龙魂ID: {结果['龙魂ID']}")
print(f"易经卦象: {' → '.join(结果['卦象序列'])}")
print(f"甲骨文编码: {结果['甲骨文编码']}")

# 导出证书
生成器.导出证书(结果)
```

## 评估委员会

### 提交案例

```python
from core.龙魂评估委员会 import 龙魂评估委员会

委员会 = 龙魂评估委员会()

# 创建案例
案例 = {
    "case_id": "ARB-2025-001",
    "type": "身份盗用",
    "龙魂ID": "LONGHUN-CN-乾-坤-屯-蒙-...",
    "卦象": "乾-坤-屯-蒙",
    "甲骨文": "𠂤𡈼𡱈𣎆",
    "哈希值": "a1b2c3d4e5f6...",
    "public_consent": "完全公开"
}

# 生成评估报告
报告 = 委员会.生成报告(案例)
print(报告)
```

### 公开等级

- **不公开**：案例仅内部存档
- **部分公开**：保留争议类型和解决方案，隐藏身份
- **完全公开**：全部公开，成为标杆案例

## 常见问题

### Q1: 系统需要联网吗？

**A**: 不需要。所有计算都在本地完成，完全离线可用。

### Q2: 会采集生物识别信息吗？

**A**: 不会。系统只生成哈希值，不存储原始生物数据。

### Q3: 生成的ID可以修改吗？

**A**: 不可以。ID基于确定性算法生成，相同输入永远产生相同ID。

### Q4: 如何保证永世唯一性？

**A**: 生物特征 + 合法身份 = 全球唯一组合，算法确保不可冲突。

### Q5: 支持哪些国家？

**A**: 支持200+国家/地区，包括中国、美国、欧盟、日本、印度等。

### Q6: 可以商业使用吗？

**A**: 可以。系统采用木兰宽松许可证 v2.0，可自由商业使用。
