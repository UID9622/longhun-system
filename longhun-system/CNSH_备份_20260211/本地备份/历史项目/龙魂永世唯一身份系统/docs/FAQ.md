## 安装与环境

### Q1：为什么运行时提示找不到Python3？

**A**：需要安装Python 3.8或更高版本。

**Mac**：
```bash
brew install python3
```

**Ubuntu/Debian**：
```bash
sudo apt-get update
sudo apt-get install python3
```

**Windows**：
访问 https://www.python.org/downloads/ 下载安装，安装时勾选 "Add Python to PATH"。

---

### Q2：为什么安装opencv-python失败？

**A**：OpenCV在某些系统上安装可能遇到问题。

**解决方案**：

1. 使用国内镜像：
```bash
pip3 install opencv-python numpy -i https://pypi.tuna.tsinghua.edu.cn/simple
```

2. 尝试安装headless版本（无GUI）：
```bash
pip3 install opencv-python-headless numpy
```

3. 如果仍失败，使用系统包管理器：
```bash
# Ubuntu/Debian
sudo apt-get install python3-opencv python3-numpy

# Mac
brew install opencv
```

## 功能使用

### Q5：可以在没有生物特征的情况下使用吗？

**A**：可以！系统提供**零生物识别模式**，只使用身份证号生成ID：

```python
from 龍魂ID生成器 import 龍魂永世唯一ID生成器

生成器 = 龍魂永世唯一ID生成器()

# 零生物识别模式（推荐）
结果 = 生成器.生成龍魂ID(
    身份证号="110101199001011234",
    国家代码="CN",
    使用示例模式=True  # 启用零生物识别模式
)

print(f"龍魂ID: {结果['龍魂ID']}")
```

这个模式符合《个人信息保护法》第28条，不采集生物特征。

### Q6：生成的ID可以修改吗？

**A**：不可以！龍魂ID的核心特性就是**永世唯一**：

- ✅ 相同输入 → 永远生成相同ID
- ✅ 不可篡改
- ✅ 不可伪造
- ✅ 一旦生成就确定不变

如果输入变化（比如换了身份证号），会生成新的ID。
