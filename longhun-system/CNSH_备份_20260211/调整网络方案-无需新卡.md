## 🎯 **调整后的网络方案**

### 方案1: Mate60双卡方案 ⭐推荐
```
柬埔寨流量卡 → 插入Mate60 → Mate60热点 → 苹果电脑
```

**优势：**
- 不需要购买新卡，直接使用现有流量卡
- Mate60双卡槽，可以同时插SIM和流量卡
- 手机热点性能稳定，连接可靠
- 省去额外设备，简化配置

**设置步骤：**
1. 将柬埔寨流量卡插入Mate60的第二个卡槽
2. 在Mate60设置中启用该卡的数据流量
3. 开启Mate60个人热点
4. 苹果电脑连接Mate60热点

### 方案2: 电脑直连流量卡方案
```
柬埔寨流量卡 → USB网卡/转接器 → 苹果电脑
```

**优势：**
- 电脑直连，稳定性最高
- 无需通过手机中转
- 可以配合WiFi网络使用

**需要准备的设备：**
- USB网卡或SIM卡转接器
- 确保MacOS兼容性

### 方案3: 混合网络方案
```
主网络：现有WiFi网络
备网络：Mate60热点（柬埔寨流量卡）
```

**优势：**
- 利用现有WiFi作为主要网络
- Mate60热点作为备用网络
- 自动故障切换

## 🔧 **具体操作指南**

### 方案1详细设置（推荐）

#### 步骤1: 安装柬埔寨流量卡
1. 关闭Mate60
2. 找到Mate60的第二个SIM卡槽
3. 按照指示插入柬埔寨流量卡
4. 开启Mate60

#### 步骤2: 配置移动数据
1. 进入Mate60设置 → 移动网络
2. 选择新插入的SIM卡
3. 启用"移动数据"
4. 设置APN（如果需要自动识别）
5. 设置"默认移动数据"为该卡

#### 步骤3: 开启热点
1. 设置 → 个人热点 → 开启
2. 设置热点名称（如：Lucky-Hotspot）
3. 设置强密码（8位以上，包含数字字母）
4. 选择"5GHz频段"（如果支持）
5. 设置"WPA2个人"加密

#### 步骤4: 电脑连接
1. 苹果电脑点击WiFi图标
2. 选择"Lucky-Hotspot"
3. 输入密码连接
4. 测试网络连接

## 📊 **性能监控**

### 网络状态检查
```python
# 创建网络监控脚本
import subprocess
import time

def check_connection():
    # 检查Mate60热点连接
    result = subprocess.run(['ping', '-c', '3', '192.168.43.1'], 
                         capture_output=True, text=True)
    return result.returncode == 0

def monitor_network():
    while True:
        if check_connection():
            print("✅ Mate60热点连接正常")
        else:
            print("❌ Mate60热点连接断开")
        time.sleep(30)

if __name__ == "__main__":
    monitor_network()
```
