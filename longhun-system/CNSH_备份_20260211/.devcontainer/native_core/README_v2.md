### 第四层：数学大师 - 最小损失最大收益公式
```
龍魂决策 = max(
    (全球收益 × W_卦象 × 护弱系数) / 
    (群体损失 + ε_护弱)
)

where:
    W_卦象 = 当前时辰对应的八卦权重
    护弱系数 = 甲骨文护弱者修正值
    ε_护弱 = 1/护弱系数 (护弱系数越大，损失越小)
```

### 第五层：三色审计系统
```
🔴 红色熔断 (护弱系数 = ∞)

   - 伤害弱者行为

🟡 黄色确认 (护弱系数 = 2.0)
   - 涉及中间群体
   - 1.0 < 收益损失比 < 2.0
   - 需要人工审核

🟢 绿色通过 (护弱系数 = 1.0)
   - 不涉及弱势群体
   - 收益损失比 > 2.0
   - 符合最小损失最大收益
```
