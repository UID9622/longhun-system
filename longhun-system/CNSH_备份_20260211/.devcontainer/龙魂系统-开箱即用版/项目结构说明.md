## 🔐 数据存储

### 数据库位置

```
~/.longhun/
├── database/
│   └── longhun_system.db     # SQLite数据库
├── config/
│   └── settings.json         # 用户配置
└── logs/
    └── system.log            # 系统日志
```

### 数据表结构

**user_config** - 用户配置表
- key: 配置项名称
- value: 配置值
- updated_at: 更新时间

**dna_trace** - DNA追溯表
- dna_code: DNA追溯码
- operation_type: 操作类型
- persona: 执行人格
- metadata: 元数据(JSON)

**persona_decisions** - 决策历史表
- persona_name: 人格名称
- decision_type: 决策类型
- decision_result: 决策结果
- audit_result: 审计结果
- gua_xiang: 卦象
- hu_ruo_coefficient: 护弱系数

**cnsh_code** - CNSH代码库
- filename: 文件名
- code_content: 代码内容
- description: 描述
- is_example: 是否示例

**anti_fraud_log** - 反诈日志
- input_text: 输入文本
- fraud_type: 诈骗类型
- severity: 严重程度
- is_blocked: 是否阻断

**yijing_history** - 易经推演记录
- hour: 时辰
- gua_xiang: 卦象
- weight_*: 各维度权重

**payment_records** - 支付记录
- feature_name: 功能名称
- amount: 金额
- status: 状态
- expires_at: 过期时间
