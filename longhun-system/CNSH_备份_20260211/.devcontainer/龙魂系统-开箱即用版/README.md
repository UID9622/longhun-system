## 🔐 安全与隐私

### 数据存储

```
~/.longhun/
├── database/
│   └── longhun_system.db     # 所有数据本地存储
├── config/
│   └── settings.json         # 本地配置
└── logs/
    └── system.log            # 本地日志
```

### 隐私承诺

1. **零上传**: 所有数据存储在本地，不上传任何服务器
2. **华为不收集**: 华为开发者功能仅提供接口，不收集用户数据
3. **完全开源**: 所有代码可审计，无后门
4. **用户主权**: 用户完全拥有自己数据的所有权
