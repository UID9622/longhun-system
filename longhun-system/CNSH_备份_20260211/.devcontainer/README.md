## 🚀 快速开始

```bash
python start_system.py start
```

### 2. 执行系统自检

```bash
python start_system.py check
```

### 3. 生成DNA追溯码

```bash
python start_system.py dna --topic "测试文档" --version v1.0
```

### 4. 执行三色审计

```bash
python start_system.py audit --ratio 2.5
```

```bash
python start_system.py font --text "中华龍魂" --output output.svg
```

## 🧩 子系统说明

### 字体系统 (font_system/)

CNSH中文原生字体引擎，支持：

- 汉字渲染为SVG
- 龍魂特殊符号映射（八卦、DNA分隔符等）
- DNA追溯码格式验证
- 多种字体风格（标准、力度、节奏、层级、棱角、组合）

```python
from font_system.cnsh_font_engine import CNSH字体引擎, 字体风格

引擎 = CNSH字体引擎()
svg = 引擎.渲染文本("中华龍魂", 字体风格.标准)
```

### 插件系统 (plugin_system/)

模块化扩展框架，支持：

- 动态加载/卸载插件
- 插件间事件通信
- OCSL合规检查
- 核心插件保护（不可卸载）

内置核心插件：
- 易经推演引擎
- DNA追溯系统
- 三色审计系统

```python
from plugin_system.plugin_manager import 插件管理器

管理器 = 插件管理器()
管理器.启用插件("插件名称")
```

### 原生系统核心 (native_core/)

龍魂系统核心，整合所有子系统：

```python
from native_core.longhun_system import 初始化系统

系统 = 初始化系统()
系统.获取系统信息()
系统.生成DNA("主题", "v1.0")
系统.执行三色审计(2.5)
```
