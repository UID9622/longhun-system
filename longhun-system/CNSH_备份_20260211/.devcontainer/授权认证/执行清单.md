
### 任务2.2：验证签名

```bash

# 应该看到：Good signature
```

### 任务2.3：导出公钥

```bash
gpg --armor --export uid9622@petalmail.com > UID9622-public-key.asc
```

### 任务2.4：备份私钥（非常重要！）

```bash

# - 加密U盘
# - 不要上传互联网
# - 纸质备份放保险箱
```
