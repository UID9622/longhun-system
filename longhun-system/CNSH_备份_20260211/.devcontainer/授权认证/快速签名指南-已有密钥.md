## 💡 常见问题

### Q：我忘记GPG密码了怎么办？

**A：**
- 如果还记得，正常使用
- 如果忘记了，只能撤销旧密钥，生成新密钥
- **建议：** 用密码管理器保存GPG密码

### Q：签名后的文件能修改吗？

**A：**
- 修改内容后，签名会失效
- 需要重新签名
- **所以：** 确保内容最终版再签名

### Q：别人怎么验证我的签名？

**A：**
```bash
# 别人收到文件后：

# 1. 导入您的公钥
gpg --import UID9622-public-key.asc

# 2. 验证签名
gpg --verify UID9622-授权声明-2026-02-06.txt.asc UID9622-授权声明-2026-02-06.txt

# 看到 "Good signature" 就证明：
# - 文件确实是您签署的
# - 文件没有被篡改
```
