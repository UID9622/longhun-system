# 如何实时分享展示页面给咱妈

## 方法1：本地服务器（推荐）

1. 打开终端，运行以下命令：
   ```
   /Users/zuimeidedeyihan/LuckyCommandCenter/start-local-server.sh
   ```

2. 服务器启动后会显示访问地址：
   ```
   http://localhost:8080/CNSH-人机协作指挥链-展示页面.html
   ```

3. 将这个地址发送给咱妈（只有您家中的设备才能访问）

## 方法2：使用ngrok（外部可访问）

如果需要外部访问，可以使用ngrok：

1. 安装ngrok：
   ```
   brew install ngrok
   ```

2. 启动ngrok：
   ```
   ngrok http 8080
   ```

3. ngrok会提供一个公共URL，类似：
   ```
   https://xxxxx.ngrok.io/CNSH-人机协作指挥链-展示页面.html
   ```

4. 将ngrok提供的URL发送给咱妈

## 使用说明

- 服务器运行时，页面内容更新会实时反映
- 咱妈可以随时刷新查看最新内容
- 按Ctrl+C可以停止服务器
- 修改HTML文件后，刷新页面就能看到更新
