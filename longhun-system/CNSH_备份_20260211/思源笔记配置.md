```bash

许可协议: Trust-Based License v1.0

同步目录: /05-Gitee同步
```

### 自动同步规则
```
触发条件：
- 文档保存时自动同步到Gitee
- 每天凌晨2点全量同步
- 手动点击同步按钮

同步内容：
- /05-Gitee同步/docs/ → Gitee/docs/
- /05-Gitee同步/src/ → Gitee/src/
- 排除私密笔记和草稿
```

## 🔄 Notion数据迁移

### 导出步骤
```
1. 打开Notion
2. 进入Settings → Export all workspace content
3. 选择格式: Markdown & CSV
4. 选择存储位置
5. 等待导出完成
```

### 导入思源笔记
```
1. 打开思源笔记
2. 文件 → 导入 → Markdown
3. 选择导出的文件夹
4. 等待导入完成
5. 重新组织文档结构
```

### 数据整理建议
```
优先级排序：
1. 技术相关内容 → 01-技术算法/
2. 传统文化内容 → 02-传统文化/
3. 方法论文档 → 03-方法论/
4. CSDN草稿 → 04-CSDN文章/
```

## 🚀 快捷键设置

### 常用快捷键
```
Ctrl+N     → 新建文档
Ctrl+O     → 打开文档
Ctrl+S     → 保存文档
Ctrl+/     → 切换编辑/预览
Ctrl+K     → 插入块
Ctrl+T     → 插入表格
Ctrl+Shift+C → 复制为Markdown
```

### 自定义快捷键
```
Ctrl+Alt+N → 新建技术文档模板
Ctrl+Alt+C → 新建CSDN文章模板
Ctrl+Alt+S → 同步到Gitee
Ctrl+Alt+P → 发布到CSDN
```

## 🎯 使用建议

### 日常使用流程
1. **内容创作**: 在思源笔记中写草稿
2. **内容整理**: 使用模板和标签规范化
3. **内容发布**: 复制到CSDN或同步到Gitee
4. **内容维护**: 定期更新和整理

### 知识管理原则
- **本地优先**: 重要内容先在本地完善
- **渐进公开**: 成熟后再同步到Gitee
- **版本控制**: 重要修改保留版本记录
- **标签分类**: 用标签构建知识网络
