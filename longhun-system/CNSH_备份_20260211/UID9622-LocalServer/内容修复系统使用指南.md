## 🚀 使用方式

### 方式1：命令行文件修复（推荐）

```bash
# 1. 准备文件
echo "有问题的内容" > test.txt

# 2. 运行修复
python3 ~/LuckyCommandCenter/UID9622-LocalServer/UID9622-Scripts/content_repair.py test.txt

# 3. 查看结果
cat ~/Desktop/UID9622-Logs/repaired_*.txt
```

### 方式2：剪贴板修复（需要安装pyperclip）

```bash
# 1. 安装依赖
cd ~/LuckyCommandCenter/UID9622-LocalServer
source venv/bin/activate
pip install pyperclip

# 2. 复制内容到剪贴板
# Command+C

# 3. 运行脚本
bash ~/LuckyCommandCenter/UID9622-LocalServer/UID9622-Scripts/repair_from_clipboard.sh

# 4. 修复后的内容已复制到剪贴板，直接粘贴即可
```

### 方式3：通过API调用

```bash
curl -X POST http://127.0.0.1:9622/api/repair_content \
  -H "Content-Type: application/json" \
  -d '{
    "content": "要修复的内容"
  }'
```
