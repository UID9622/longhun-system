## 🛠️ 技术架构（干货部分）

### 核心组件

CNSH编程语言包含完整的编译器架构：

#### 1️⃣ **词法分析器（Lexer）**

```javascript
// 支持中文关键字
const KEYWORDS = {
  '函数': 'FUNCTION',
  '返回类型': 'RETURN_TYPE',
  '整数': 'INT',
  '小数': 'FLOAT',
  '文本': 'STRING',
  '打印': 'PRINT',
  '返回': 'RETURN'
};

// 支持中文标点
const PUNCTUATION = {
  '「': 'LQUOTE',
  '」': 'RQUOTE',
  '【': 'LBRACKET',
  '】': 'RBRACKET'
};
```

#### 2️⃣ **语法分析器（Parser）**

递归下降解析，生成抽象语法树（AST）

#### 3️⃣ **代码生成器（Generator）**

将中文代码翻译成标准C代码，然后用gcc编译

## 🚀 快速上手（5分钟）

### 第1步：克隆仓库

```bash
git clone https://gitee.com/UID9622/cnsh-language.git
cd cnsh-language
```

### 第2步：写一个hello.cnsh

```cnsh
函数 主函数() 返回类型 整数 {
  打印「你好，CNSH！」
  返回 0
}
```

### 第3步：一键运行

```bash
./cnsh.sh hello.cnsh
```
