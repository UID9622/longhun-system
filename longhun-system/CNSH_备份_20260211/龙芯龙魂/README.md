# 🐉 龙芯龙魂数字永生系统

## 📋 系统概述

**龙芯龙魂**是一个基于龙芯3A5000国产硬件和CNSH哲学的数字永生系统，专注于普惠生活内容的生成和管理。

### 🎯 核心特性

- ✅ **国产自主**：基于龙芯3A5000硬件，完全自主可控
- ✅ **普惠使命**：为农民、教师、志愿者等群体提供生活帮助
- ✅ **安全可靠**：国密算法加密，无后门无依赖
- ✅ **智能过滤**：自动识别和过滤非普惠内容
- ✅ **DNA追溯**：每份内容都有唯一DNA追溯码

### 🧬 DNA追溯系统

每份生成的内容都有独特的DNA追溯码：
```
#龙-[平台]-[重要性]-[时间戳]-[哈希]
示例: #龙-普惠-8-20251214-143022-a1b2c3d4
```

## 🚀 快速开始

### 环境要求

- **硬件**: 龙芯3A5000 (LoongArch64) 或兼容x86/ARM
- **系统**: Linux/macOS/Windows
- **Python**: 3.8+
- **内存**: 至少2GB可用内存

### 安装步骤

1. **克隆项目**
```bash
cd /Users/zuimeidedeyihan/LuckyCommandCenter/龙芯龙魂
```

2. **安装依赖**
```bash
pip3 install -r requirements.txt
```

3. **配置环境**
```bash
# 编辑配置文件
nano config/loongson.env
```

4. **启动系统**
```bash
./启动龙芯龙魂.sh
```

编辑 `config/loongson.env`：

## 🔒 安全特性

### 内容过滤
系统自动识别和过滤以下内容：
- ❌ 技术相关内容（代码、算法、API等）
- ❌ 敏感内容（政治、军事、宗教等）

- 使用SHA256模拟国密SM3算法
