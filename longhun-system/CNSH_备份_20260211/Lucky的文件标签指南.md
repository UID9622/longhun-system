## 🛠️ 如何设置标签？（3种方法）

### 方法1：右键菜单（最简单）

**步骤**：
1. 在Finder中找到文件
2. 右键点击文件
3. 选择"标记"（或"Tags"）
4. 选择颜色标签

**Lucky您理解成**：就像在Word里给文字换颜色一样简单！

---

### 方法2：批量设置（效率高）

**步骤**：
1. 在Finder中选中多个文件（按住Cmd键点击）
2. 右键 → 标记
3. 选择颜色
4. 所有选中的文件都会被设置成这个颜色

**Lucky您理解成**：就像批量修改文件名一样，一次性给很多文件贴标签！

---

### 方法3：拖拽设置（最快）

**步骤**：
1. 打开Finder
2. 显示边栏（View → Show Sidebar）
3. 边栏上有"标记"（Tags）列表
4. 直接把文件拖到对应的颜色标签上

**Lucky您理解成**：就像拖文件到文件夹一样，拖到标签上就自动贴上了！

## 🔍 如何使用标签搜索文件？

### 方法1：Finder侧边栏

1. 打开Finder
2. 左侧边栏有"标记"（Tags）
3. 点击任意颜色
4. 显示所有该颜色的文件

**Lucky您理解成**：就像点击"收藏夹"，快速找到相关文件！

---

### 方法2：Spotlight搜索

1. 按 `Cmd + Space`（打开Spotlight）
2. 输入：`tag:绿色` 或 `tag:green`
3. 显示所有绿色标签的文件

**Lucky您理解成**：就像在网站上搜索"标签:技术"，找到所有技术文章！

---

### 方法3：智能文件夹

1. Finder → 文件 → 新建智能文件夹
2. 添加条件：标签 = 绿色
3. 保存这个智能文件夹
4. 以后点击就能看到所有绿色标签文件

**Lucky您理解成**：就像创建一个"自动更新的播放列表"，自动收集符合条件的文件！

## 💬 诸葛亮的小贴士

1. **不要过度分类**
   - 标签太多反而混乱
   - 建议最多用5-6种颜色
   - 每种颜色有明确含义

2. **定期整理**
   - 每周检查一次标签
   - 把完成的改成绿色
   - 把不重要的改成灰色

3. **结合文件夹使用**
   - 标签不能替代文件夹
   - 文件夹管理结构
   - 标签管理状态

4. **养成习惯**
   - 创建新文件时立即打标签
   - 就像"垃圾分类"一样
   - 一开始就做对，后面省事
