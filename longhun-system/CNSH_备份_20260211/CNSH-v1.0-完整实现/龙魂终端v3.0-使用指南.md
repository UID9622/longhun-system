## 🤖 AI模型配置

### 1. Ollama（推荐）

**安装Ollama：**

```bash
# Mac - 官方下载
curl -fsSL https://ollama.com/download/Ollama-darwin.zip -o /tmp/Ollama.zip
unzip -q /tmp/Ollama.zip -d /tmp/
sudo mv /tmp/Ollama.app /Applications/
open /Applications/Ollama.app

# 下载模型
ollama pull qwen2.5:7b
```

**验证运行：**

```bash
# 启动Ollama服务
ollama serve

# 测试对话
ollama run qwen2.5:7b
```

### 2. LM Studio

**下载安装：**
- 官网：https://lmstudio.ai
- 下载后安装，启动应用
- 下载模型（如 Qwen 7B）

**验证运行：**
- LM Studio会自动在 `http://localhost:1234` 启动服务

### 3. 自定义API

**配置步骤：**

1. 点击「🤖 选择模型」
2. 选择「🌐 自定义API」
3. 输入你的API端点地址
4. 开始对话

**API格式：**

```json
{
  "question": "你的问题"
}
```

**返回格式：**

```json
{
  "response": "AI的回答",
  "answer": "或这个字段",
  "message": "或这个字段"
}
```

## 🔧 故障排查

### 问题1：AI无法连接

**原因：** 本地AI模型未运行

**解决：**

```bash
# 检查Ollama
curl http://localhost:11434/api/tags

# 如果报错，启动服务
ollama serve
```

### 问题2：界面卡顿

**原因：** 动画效果过多

**解决：** 暂不支持关闭动画，建议使用性能较好的设备

### 问题3：命令无响应

**原因：** 输入框未聚焦

**解决：** 点击输入框后重新输入
