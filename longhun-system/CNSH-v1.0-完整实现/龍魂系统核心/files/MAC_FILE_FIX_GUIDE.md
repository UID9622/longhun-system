### 下载目录检查

```bash
# 检查下载目录
ls -la ~/Downloads/*.cnsh

# 如果看不到文件，尝试：
ls -la ~/Downloads/ | grep cnsh

# 或者查看所有下载文件
open ~/Downloads/
```

### 权限问题修复

```bash
# 如果文件无法打开，修复权限
chmod 644 ~/Downloads/*.cnsh
chmod 644 ~/Downloads/*.md

# 或者批量修复
find ~/Downloads -name "*.cnsh" -exec chmod 644 {} \;
find ~/Downloads -name "*.md" -exec chmod 644 {} \;
```
