---

## 🚀 最简单的方法（推荐）

### 方法1：右键设置（只需一次）

1. 在Finder中找到 `.cnsh` 文件
2. **右键点击**文件 → 选择"打开方式"
3. 选择"文本编辑"（TextEdit）
4. 文件会正常打开
5. 再次**右键点击**任何 `.cnsh` 文件 → 选择"打开方式" → 选择"文本编辑"
6. 点击"全部更改"按钮

✅ 以后所有 `.cnsh` 文件都会用文本编辑打开！

---

## 📝 方法2：直接拖拽

1. 打开"文本编辑"（在应用程序文件夹）
2. 把 `.cnsh` 文件拖拽到文本编辑窗口中
3. 文件会正常显示

---

## 🔧 方法3：使用终端查看

如果连文本编辑都打不开，可以用终端查看：

```bash
# 查看文件内容
cat "文件名.cnsh"

# 或者用编辑器打开
open -e "文件名.cnsh"

# 或者用VS Code（如果安装了）
code "文件名.cnsh"
```

## 💡 为什么显示问号？

macOS默认不认识 `.cnsh` 扩展名，所以显示为未知文件。但**内容是完全正常的文本**，只是系统不知道该用什么程序打开而已。
