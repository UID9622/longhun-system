## 📥 接入规则速查（老大接入外部API）

### 第一层：接入前审查 ✅

| 检查项 | 🟢 可接入 | 🟡 谨慎接入 | 🔴 禁止接入 |
|--------|----------|------------|-----------|
| **数据主权** | 不出境/合规 | 可能出境但有说明 | 强制出境且不透明 |
| **透明性** | 开源/合同清晰 | 闭源但有信誉 | 完全黑箱 |
| **P0检查** | 不违反 | 部分可能违反 | 明显违反 |
| **信誉** | 良好 | 一般 | 有劣迹 |

**快速决策：**
- 全🟢 → 直接接入
- 有🟡 → 老大确认后接入
- 任何🔴 → 立即拒绝

## 📤 接出规则速查（龍魂API对外）

### 六层防护体系

| 层次 | 名称 | 核心措施 | 防护目标 |
|-----|------|---------|---------|
| **第1层** | 入口设卡 | 动态签名+设备指纹+用量画像 | 区分正常用户和爬虫 |
| **第2层** | 输出锁死 | 拒绝logit+输出过滤+温度随机 | 不暴露模型内部决策 |
| **第3层** | 宪法保护 | 碎片化+哈希比对+自毁程序 | 保护核心价值观 |
| **第4层** | API设计 | 瘦API+速率限制+用量审计 | 减少攻击面 |
| **第5层** | 本地化 | 80%本地+20%云端 | 核心资产在本地 |
| **第6层** | 法律社区 | 实名认证+协议+举报机制 | 最后防线 |

---

### 第1层：入口设卡详细规则

**三层认证（缺一不可）：**

```yaml
1. API Key + 动态签名:
   - 格式: API Key + 时间戳 + SHA256签名
   - 有效期: 5分钟
   - 防重放: 每次签名不同
   
2. 设备指纹:
   - iOS: IDFV（合规）
   - 检测: 同Key多设备异常 → 封禁
   
3. 用量画像:
   - 正常: 一会儿问一句
   - 爬虫: 一秒问100句 → 封禁
```

**速率限制：**
- 免费用户：10次/分钟
- 付费用户：100次/分钟
- VIP用户：1000次/分钟

**想批量盗取？去充钱，充到肉疼！** 💰

---

### 第2层：输出锁死详细规则

**禁止输出的内容：**
- ❌ logits概率分布
- ❌ tokens列表
- ❌ 模型权重
- ❌ 训练数据
- ❌ 宪法全文

**只返回的内容：**
- ✅ 生成文本
- ✅ DNA追溯码
- ✅ 时间戳

**输出过滤：**
```python
# 敏感词替换（不删除，改表达）
"母协议" → "核心原则"
"28人格矩阵" → "多维决策系统"

# 语义扰动（相同问题每次措辞不同）
"我认为" → "我觉得" / "依我看" / "在我看来"
```

---

### 第3层：宪法保护详细规则

**宪法碎片化存储：**
```yaml
位置1：模型权重层（嵌入，无法单独提取）
位置2：Prompt模板（分散在多个模板）
位置3：运行时生成（不存硬盘）
位置4：加密存储（老大有密钥）
```

**敏感词哈希比对：**
```python
# 不存明文，存哈希
"母协议" → SHA256 → 存储哈希值
检查时 → 计算输入哈希 → 比对
```

**自毁程序（慎用）：**
```yaml
触发条件: 检测到高强度对抗攻击
触发次数: 100次异常请求
结果: 输出混沌数据，让攻击者拆出垃圾
```

---

### 第4层：API设计详细规则

**瘦API（只暴露必要接口）：**

✅ **允许的接口：**
```yaml
POST /api/v1/chat
  功能: 聊天对话
  输入: 用户消息
  输出: AI回复（仅文本）

POST /api/v1/constitution_check
  功能: 宪法合规检查
  输入: 待检查内容
  输出: 🟢🟡🔴 + 原因

GET /api/v1/status
  功能: 系统状态
  输出: 在线/离线
```

❌ **禁止的接口：**
```yaml
❌ POST /api/v1/batch_inference  # 批量推理
❌ POST /api/v1/fine_tune        # 微调
❌ GET /api/v1/weights            # 权重下载
❌ GET /api/v1/config             # 配置查询
```

**用量审计：**
- 每周生成报告
- 异常模式检测（深夜高频、单接口重复）
- 人工介入 + 自动限流

## 🛡️ 三色审计速查

| 场景 | 🟢 通过 | 🟡 待审 | 🔴 阻断 |
|------|--------|--------|--------|
| **接入外部API** | 开源+不出境 | 闭源但有信誉 | 黑箱+出境 |
| **API调用频率** | <10次/分钟 | 10-100次/分钟 | >100次/分钟 |

| **用户行为** | 正常对话 | 深夜高频 | 爬虫模式 |
| **数据上传** | 不包含隐私 | 可能包含隐私 | 明确违规 |
