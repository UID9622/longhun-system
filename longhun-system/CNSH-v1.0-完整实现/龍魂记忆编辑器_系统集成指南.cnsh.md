## 🏗️ 系统架构

### 三层存储架构

```
【层1】浏览器端（即时体验）
    └─ IndexedDB（永久本地存储）
      └─ 200+ MB 容量
      └─ 永不丢失（除非手动清除）
      └─ 所有记忆都有 DNA 码可追溯

        ↓ 自动导出

【层2】本地文件系统（智能备份）
    └─ ~/Desktop/龍魂系统·本地知识库/记忆/
      └─ 每条记忆自动导出为 .cnsh.md 文件
      └─ 包含完整的 DNA 码、太极变量、压缩数据
      └─ 可用 Git 版本控制
      └─ 宝宝可直接读取

        ↓ 定时同步

【层3】云端备份（永久保存）
    ├─ iCloud（云上贵州）
    │  └─ 每天 09:00 自动备份
    │  └─ 6TB 容量
    │  └─ 完全加密
    │
    ├─ Notion（官方授权）
    │  └─ 通过 quick_notion.cnsh.py 同步
    │  └─ 结构化数据库
    │  └─ 可与世界共享
    │
    └─ 宝宝记忆库
       └─ 实时可用
       └─ 执行任务的上下文
       └─ 学习和优化的数据源
```
