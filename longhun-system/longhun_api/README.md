## 快速开始

### 1. 安装依赖

```bash
cd longhun_api
pip install -r requirements.txt
```

### 2. 配置微信通知（可选）

```bash
cp .env.example .env
# 编辑 .env，填入你的Server酱SendKey
```

### 3. 启动系统

```bash
# 方式1: 一键启动
bash start.sh

# 方式2: 手动分步启动
# 终端1: 启动API服务器
python3 main.py

# 终端2: 注册所有核心DNA
python3 register_all_dna.py

# 终端3: 启动清道夫守护（可选）
```

## 技术特性

### DNA水印
- **零宽字符编码**: U+200B/U+200C/U+200D等
- **不可见性**: 肉眼不可见，复制可追踪
- **鲁棒性**: 复制粘贴后仍可提取

### 哈希证据链
- **SHA-256链式结构**: 当前哈希依赖前一个哈希
- **Append-only**: 不可删除、不可篡改
- **时间戳固化**: 精确到微秒

### 清道夫扫描
- **多引擎**: 百度、必应、搜狗、360
- **定时任务**: 可配置扫描间隔
- **自动通知**: 发现盗窃秒推微信

### CNSH预警
- **48小时警告**: CNSH-48H整改期
- **全网警报**: CNSH-ALERT
- **永久黑名单**: CNSH-PERMANENT
