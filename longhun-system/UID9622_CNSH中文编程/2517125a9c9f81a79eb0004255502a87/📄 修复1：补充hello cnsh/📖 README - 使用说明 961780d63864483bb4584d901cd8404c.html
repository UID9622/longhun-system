📖 CNSH编译器使用说明

DNA追溯码： #ZHUGEXIN⚡️2025-12-31-CNSH语言-V1.0

🚀 快速开始

第1步：安装依赖

# 需要Node.js环境
node --version  # 确认已安装

第2步：编写CNSH程序

创建 hello.cnsh：

函数 主函数() 返回类型 整数 {
  打印「你好，CNSH！」
  返回 0
}

第3步：编译

# CNSH → C
node cnsh-compiler.js hello.cnsh

# C → 可执行文件
gcc hello.c -o hello

# 运行
./hello

📝 语法参考

数据类型

  • 整数 - int
  • 小数 - double
  • 文本 - char*
  • 真假 - bool

控制流

  • 如果【条件】{}
  • 否则 {}
  • 循环【次数】{}
  • 返回 值

函数定义

函数 函数名(参数类型 参数名) 返回类型 返回类型 {
  # 函数体
  返回 值
}

🛡️ 三色审计

自动内容审核

  • 🔴 红色：暴力、违法、仇恨言论 → 拒绝编译
  • 🟡 黄色：敏感话题 → 警告继续
  • 🟢 绿色：安全内容 → 正常编译

📦 文件结构

cnsh/
├── cnsh-compiler.js    # 编译器
├── hello.cnsh          # CNSH源码
├── hello.c             # 生成的C代码
└── hello               # 可执行文件

🐛 常见问题

Q: 语法错误？

  • 使用中文方括号 【】
  • 使用中文引号 「」
  • 函数必须有 返回类型

Q: 编译失败？

# 查看详细错误
node cnsh-compiler.js hello.cnsh --verbose
