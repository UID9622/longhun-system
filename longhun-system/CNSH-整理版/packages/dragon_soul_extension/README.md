# 🐉 龙魂·思维主权守护者

龙魂浏览器扩展 - 自动净化输入，熔断风险内容，保障思维主权完整性。支持智能记忆压缩与快速还原。

## ✨ 功能特性

### 🌟 龙魂系统主控台（联动按键版）
- 🎛️ **统一界面** - 所有功能集成在一个页面，联动按键切换
- 🇨🇳 **数据主权** - 完全本地化，数据不上传，技术主权
- 🧠 **记忆压缩** - 智能压缩记忆，压缩率达 99%+
- 🧬 **DNA 存证** - 基于 SHA-256 的数字存证系统
- 🔍 **资产扫描** - 识别原创内容，支持 40+ 关键词
- 📊 **统计分析** - 实时统计系统运行状态
- 💾 **本地存储** - IndexedDB 永久存储，安全可靠

### 核心功能
- 🔍 **智能净化** - 自动识别并净化网页输入框中的危险内容
- 🛡️ **熔断机制** - 检测到高风险内容时自动熔断，保护系统安全
- 🧬 **DNA 校验** - 自动校验和修正 CNSH DNA 格式
- 📊 **实时统计** - 查看拦截、隔离、修正等统计数据
- 📝 **审计日志** - 记录所有操作，支持导出分析
- 🎨 **现代界面** - 简洁美观的弹窗界面

### 🆕 记忆压缩系统
- 🧠 **智能压缩** - 将大量文本压缩为简短的 DNA 格式
- 📦 **多级压缩** - 支持 light/medium/heavy 三种压缩级别
- 🔑 **关键词提取** - 自动提取 Top 5 关键词
- 💭 **情感分析** - 识别积极/消极/平静三种情感
- ☯️ **太极算法** - 提取字数、行数、重要程度等变量
- 💾 **永久存储** - 使用 IndexedDB 本地存储
- 🔄 **快速还原** - 通过 DNA 追溯码快速恢复原文
- 🗄️ **记忆库管理** - 查看、搜索、删除、导出记忆

### 🆕 DNA 存证系统
- 🔐 **数字指纹** - 基于 SHA-256 算法生成数字指纹
- 🧬 **DNA 追溯码** - 唯一的 DNA 追溯码格式
- ✅ **存证验证** - 随时验证存证真伪
- 📥 **文件导出** - 导出存证为 JSON 文件
- 📥 **文件导入** - 从 JSON 文件导入存证
- 📋 **存证管理** - 查询、导出、删除存证

### 🆕 资产扫描器
- 🔍 **智能扫描** - 识别文件中的原创标记
- ✅ **核心原创** - 识别 UID9622、宝宝、#龍芯 等核心标记
- 🟢 **原创分类** - 支持 5 种分类（核心原创、原创、混合、外部、未标记）
- 📊 **实时统计** - 实时显示扫描统计
- 📄 **报告导出** - 导出扫描报告为 JSON

## 🚀 安装指南

### 前置条件

1. 准备图标文件（16x16、48x48、128x128 的 PNG 图片）
2. 将图标文件放入 `assets/icons/` 目录并命名为：
   - `icon-16.png`
   - `icon-48.png`
   - `icon-128.png`

### 安装步骤（Chrome）

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `dragon_soul_extension` 文件夹
6. 扩展安装完成！

### 安装步骤（Edge）

1. 打开 Edge 浏览器
2. 访问 `edge://extensions/`
3. 开启左下角的"开发人员模式"
4. 点击"加载解压缩的扩展"
5. 选择 `dragon_soul_extension` 文件夹
6. 扩展安装完成！

## 📖 使用说明

### 🌟 龙魂系统主控台（推荐使用）

**打开方式：**

1. 在浏览器中直接打开 `main-console.html` 文件
2. 或者在扩展目录中找到该文件并双击打开

**功能模块：**

- **🧠 记忆压缩系统** - 智能压缩记忆，压缩率达 99%+
- **🧬 DNA 存证系统** - 生成和验证数字存证
- **🔍 资产扫描器** - 识别原创内容
- **📊 统计分析** - 查看系统运行状态

**使用流程：**

1. 点击顶部的导航按钮切换功能模块
2. 在对应的功能区域输入内容或上传文件
3. 点击处理按钮执行操作
4. 查看处理结果和统计数据
5. 保存或导出结果

### 基本使用

1. 扩展安装后自动运行
2. 在任何网页的输入框中输入内容
3. 扩展会自动检测并净化内容
4. 净化结果会通过右上角通知显示

### 弹窗功能

点击浏览器工具栏中的扩展图标，可以：

- 查看今日统计数据（拦截、隔离、修正次数）
- 清空隔离区
- 导出审计日志（JSON 格式）
- 查看系统信息（GPG 指纹、UID、DNA 前缀）

### 净化规则

扩展会自动检测以下内容：

**威胁特征（直接拦截）：**
- 脚本注入：`<script>`, `javascript:`, `vbscript:`
- 命令执行：`eval()`, `exec()`, `system()`, `popen()`
- 路径遍历：`../`, `..%2F`, `%2e%2e`
- SQL 注入：`' OR 1=1`, `'; DROP TABLE`, `UNION SELECT`
- 系统命令：`; rm -rf`, `| curl `, `| wget `

**可疑特征（隔离待审）：**
- 编码内容：`base64`, `0x`, `\x`
- 危险协议：`data:text`, `file://`, `ldap://`

### DNA 格式修正

扩展会自动：

- 迁移旧版 DNA 前缀：
  - `#龍芯⚡️` → `#龍芯⚡️`
  - `#龍芯⚡️` → `#龍芯⚡️`
  - `#UID9622⚡️` → `#龍芯⚡️`
- 标记外来 UID：`UID1234` → `UID1234[🟡BRANCH]`
- 校验 DNA 格式并标记不规范内容

## 🛠️ 开发说明

### 核心模块

1. **DragonPurifier** - 净化器核心类
   - `purify(input, source)` - 净化输入内容
   - `migrateDNAPrefix(input, mutations)` - DNA 前缀迁移
   - `tagForeignUID(input, mutations)` - 外来 UID 标记
   - `validateDNAFormat(content)` - DNA 格式校验

2. **DNAValidator** - DNA 校验器
   - `validate(dnaString)` - 验证 DNA 格式
   - `extractDate(dnaString)` - 提取日期
   - `extractIdentifier(dnaString)` - 提取标识符

3. **FuseEngine** - 熔断引擎
   - `shouldFuse()` - 判断是否应该熔断
   - `recordFuse()` - 记录熔断事件
   - `getFuseStatus()` - 获取熔断状态

4. **QuantumLogger** - 量子日志记录器
   - `log(level, message, context)` - 记录日志
   - `getLogs()` - 获取日志
   - `clearLogs()` - 清空日志

### 消息传递

扩展使用 Chrome Runtime API 进行消息传递：

```javascript
// 发送消息
chrome.runtime.sendMessage({
  action: 'recordFuse',
  data: { /* 数据 */ }
});

// 接收消息（在 service-worker.js 中）
chrome.runtime.onMessage.addListener((request, sender, sendResponse) => {
  if (request.action === 'recordFuse') {
    // 处理消息
  }
});
```

## ⚠️ 注意事项

1. **隐私保护** - 所有数据仅存储在本地，不会上传到任何服务器
2. **性能影响** - 扩展会对每个输入事件进行处理，可能会有轻微的性能影响
3. **兼容性** - 扩展兼容 Chrome、Edge 等基于 Chromium 的浏览器
4. **权限说明** - 扩展需要以下权限：
   - `storage` - 存储审计日志和配置
   - `activeTab` - 访问当前标签页的输入
   - `scripting` - 注入内容脚本
   - `alarms` - 定时清理任务

## 🧠 记忆压缩系统使用指南

### 快速开始

1. **打开记忆编辑器**
   - 在浏览器中直接打开 `memory-editor.html` 文件
   - 或者在扩展目录中找到该文件并双击打开

2. **输入记忆内容**
   - 在左侧输入框中输入你的记忆、想法、笔记
   - 支持任意长度的文本（几十万字都可以）

3. **处理记忆**
   - 点击"🧠 处理记忆"按钮
   - 系统会自动：
     - 生成 DNA 追溯码
     - 提取太极算法变量（字数、关键词、情感等）
     - 压缩记忆内容

4. **保存记忆**
   - 点击"💾 保存到系统"按钮
   - 记忆会保存到 IndexedDB 本地存储
   - 可以随时查看、搜索、删除

5. **还原记忆**
   - 在记忆库中找到要还原的记忆
   - 点击"🔄 还原"按钮
   - 系统会根据 DNA 追溯码恢复原文

### 核心功能说明

#### DNA 追溯码生成
- 格式：`#龍芯⚡️YYYY-MM-DD-MEMORY-XXXX-UID9622`
- 每条记忆都有唯一的 DNA 追溯码
- 可用于快速定位和还原记忆

#### 太极算法变量
- **字数**：记忆的字符数量
- **行数**：记忆的行数
- **关键词**：Top 5 关键词，按词频排序
- **情感**：积极/消极/平静
- **重要程度**：1-10 分，基于字数和关键词计算

#### 记忆压缩
- **Light 级别**：保留 200 字摘要 + 3 个关键词
- **Medium 级别**：保留 100 字摘要 + 5 个关键词（默认）
- **Heavy 级别**：保留 50 字摘要 + 7 个关键词

#### 记忆还原
- **完整还原**：从 IndexedDB 恢复完整原文
- **摘要还原**：基于压缩数据生成摘要版本
- **还原质量**：0-100%，表示还原的完整度

### 高级功能

#### 批量操作
- 支持批量压缩记忆
- 支持批量还原记忆
- 支持导出所有记忆为 JSON

#### 搜索功能
- 按关键词搜索
- 按情感过滤
- 按重要程度过滤
- 按日期范围过滤

#### 数据管理
- 查看存储统计信息
- 清理过期记忆
- 导出/导入记忆数据
