# 🚀 快速安装指南

## 第一步：准备图标文件

在安装扩展之前，你需要准备三个图标文件：

1. **icon-16.png** (16x16 像素)
2. **icon-48.png** (48x48 像素)
3. **icon-128.png** (128x128 像素)

### 如何获取图标

**选项 1：使用临时图标（快速测试）**
- 从网上下载任何 PNG 图片
- 使用图片编辑器调整尺寸
- 重命名为上述文件名
- 放入 `assets/icons/` 目录

**选项 2：创建自定义图标**
- 访问 https://www.favicon-generator.org/
- 上传你的 logo 或使用在线工具创建
- 下载生成的图标文件
- 放入 `assets/icons/` 目录

**选项 3：使用 AI 生成**
- 访问 https://www.canva.com/
- 搜索"dragon icon"或"shield icon"
- 选择合适的模板并导出
- 调整尺寸并放入 `assets/icons/` 目录

## 第二步：加载扩展到浏览器

### Chrome 浏览器

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的**"开发者模式"**开关
4. 点击左上角的**"加载已解压的扩展程序"**按钮
5. 在文件选择器中，导航到并选择 `dragon_soul_extension` 文件夹
6. 点击"选择文件夹"
7. ✅ 扩展安装成功！

### Microsoft Edge 浏览器

1. 打开 Edge 浏览器
2. 在地址栏输入：`edge://extensions/`
3. 打开左下角的**"开发人员模式"**开关
4. 点击左上角的**"加载解压缩的扩展"**按钮
5. 在文件选择器中，导航到并选择 `dragon_soul_extension` 文件夹
6. 点击"选择文件夹"
7. ✅ 扩展安装成功！

## 第三步：验证安装

1. 在浏览器工具栏中找到扩展图标（🐉）
2. 点击图标，应该会看到弹窗界面
3. 弹窗中显示的统计数据应该都是 0
4. 查看系统信息，确认配置正确

## 常见问题

### Q: 扩展加载失败，提示"清单文件无效"

**A:** 检查 `manifest.json` 文件格式是否正确，确保所有引号和逗号都正确。

### Q: 扩展加载了但没有效果

**A:**
1. 刷新当前网页
2. 检查浏览器控制台是否有错误信息
3. 确认扩展已启用（在 `chrome://extensions/` 中查看）

### Q: 图标显示不正确

**A:** 确保图标文件：
- 文件名正确（icon-16.png, icon-48.png, icon-128.png）
- 格式为 PNG
- 尺寸匹配或接近指定尺寸

### Q: 如何卸载扩展

**A:**
1. 访问 `chrome://extensions/`
2. 找到龍魂扩展
3. 点击"移除"按钮

### Q: 如何更新扩展

**A:**
1. 修改代码文件
2. 在 `chrome://extensions/` 中找到扩展
3. 点击"重新加载"按钮（刷新图标）
