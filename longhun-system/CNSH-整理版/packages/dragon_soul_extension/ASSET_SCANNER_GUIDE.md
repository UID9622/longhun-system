# 🐲 龍魂数字资产扫描器使用指南

## 📖 系统概述

龍魂数字资产扫描器 v2.0 是一个强大的数字资产识别和分类工具，可以快速识别原创内容、混合代码和外部代码。

### 核心特性

- 🔍 **核心原创识别** - 通过"宝宝"、"UID9622"等关键词识别核心原创内容
- 📊 **多级分类** - 支持核心原创、原创、混合、外部、未标记五种分类
- 🎯 **完整关键词库** - 包含 40+ 个原创关键词，覆盖身份锚、体系、技术、理念等
- 📈 **统计分析** - 实时统计各类资产数量和占比
- 📄 **多格式导出** - 支持 JSON 和 Markdown 格式导出报告
- 🔎 **关键词搜索** - 快速搜索包含特定关键词的文件

## 🎯 分类标准

### ✅ 核心原创

**识别标准：** 包含以下任一关键词
- "UID9622"
- "宝宝"
- "#龍芯"
- "#ZHUGEXIN"

**说明：** 这是你的数字指纹，只要出现其中一个就认定为核心原创。

### 🟢 原创

**识别标准：**
- 包含其他原创关键词（如：北辰协议、三才算法、DNA追溯等）
- 不包含外部代码特征

**说明：** 使用了你的体系关键词，但没有核心指纹。

### 🟡 混合

**识别标准：**
- 同时包含原创关键词和外部代码特征

**说明：** 在外部代码基础上进行了修改或扩展。

### 📦 外部

**识别标准：**
- 仅包含外部代码特征
- 不包含任何原创关键词

**说明：** 完全使用的外部代码或第三方库。

### ❓ 未标记

**识别标准：**
- 不包含任何原创关键词
- 不包含任何外部代码特征

**说明：** 无法确定来源的代码。

## 🚀 使用方法

### 1. 打开资产扫描器

在浏览器中直接打开 `asset-scanner.html` 文件：
```bash
open asset-scanner.html
```

### 2. 上传文件

**方法一：拖拽上传**
- 将文件拖拽到上传区域
- 支持批量上传多个文件

**方法二：点击上传**
- 点击上传区域
- 在文件选择器中选择文件
- 支持多选

**支持的文件格式：**
- `.md` - Markdown 文件
- `.csv` - CSV 文件
- `.txt` - 文本文件
- `.py` - Python 文件
- `.js` - JavaScript 文件
- `.ts` - TypeScript 文件
- `.cpp` - C++ 文件
- `.hpp` - C++ 头文件
- `.html` - HTML 文件
- `.json` - JSON 文件
- `.sh` - Shell 脚本
- `.cnsh` - CNSH 文件

### 3. 开始扫描

点击"🔍 开始扫描"按钮，系统会自动：
- 读取文件内容
- 扫描原创关键词
- 扫描外部代码特征
- 分类并统计结果

### 4. 查看结果

**统计信息：**
- 核心原创数量
- 原创数量
- 混合数量
- 外部数量
- 未标记数量
- 总计

**详细列表：**
- 文件名
- 文件类型
- 字符数
- 文件大小
- 修改时间
- 匹配的关键词

### 5. 搜索关键词

在搜索框中输入关键词，点击"搜索"按钮：
- 搜索文件名
- 搜索匹配的关键词
- 显示匹配的结果

### 6. 导出报告

**导出 JSON：**
- 点击"📄 导出 JSON"按钮
- 下载 JSON 格式的扫描报告
- 包含完整的统计和详情

**导出 Markdown：**
- 点击"📝 导出 Markdown"按钮
- 下载 Markdown 格式的扫描报告
- 适合阅读和分享

## ⚠️ 注意事项

### 1. 关键词匹配

- 关键词匹配是大小写敏感的
- "宝宝"和"宝宝"不同
- "UID9622"和"uid9622"不同

### 2. 文件格式

- 仅支持文本文件
- 二进制文件无法扫描
- 图片、视频等文件不支持

### 3. 扫描性能

- 大文件扫描可能需要较长时间
- 建议分批扫描大量文件
- 浏览器可能有内存限制

### 4. 隐私保护

- 所有扫描在本地完成
- 不会上传任何数据到服务器
- 扫描结果仅保存在本地

## 🔧 技术实现

### 扫描算法

```javascript
// 1. 读取文件内容
const content = readFile(file);

// 2. 扫描原创关键词
const originalHits = originalMarkers.filter(m => content.includes(m));

// 3. 扫描外部代码特征
const externalHits = externalMarkers.filter(m => content.includes(m));

// 4. 判断分类
if (["UID9622", "宝宝", "#龍芯", "#ZHUGEXIN"].some(k => content.includes(k))) {
  category = "✅ 核心原创";
} else if (originalHits.length > 0 && externalHits.length === 0) {
  category = "🟢 原创";
} else if (originalHits.length > 0 && externalHits.length > 0) {
  category = "🟡 混合";
} else if (externalHits.length > 0) {
  category = "📦 外部";
} else {
  category = "❓ 未标记";
}
```
