## 🎯 三大核心系统

### 1. 智能净化系统

#### 核心功能
- 🔍 **智能净化** - 自动识别并净化危险内容
- 🛡️ **熔断机制** - 检测高风险内容并自动熔断
- 🧬 **DNA 校验** - 自动校验和修正 CNSH DNA 格式
- 📊 **实时统计** - 查看拦截、隔离、修正等统计数据
- 📝 **审计日志** - 记录所有操作，支持导出分析

#### 核心模块
- `core/purifier.js` - 净化器模块
- `core/dna-validator.js` - DNA 校验器
- `core/fuse-engine.js` - 熔断引擎
- `core/quantum-logger.js` - 量子日志记录器

### 2. 记忆压缩系统

#### 核心功能
- 🧠 **智能压缩** - 将大量文本压缩为简短的 DNA 格式
- 📦 **多级压缩** - 支持 light/medium/heavy 三种压缩级别
- 🔑 **关键词提取** - 自动提取 Top 5 关键词
- 💭 **情感分析** - 识别积极/消极/平静三种情感
- ☯️ **太极算法** - 提取字数、行数、重要程度等变量
- 💾 **永久存储** - 使用 IndexedDB 本地存储
- 🔄 **快速还原** - 通过 DNA 追溯码快速恢复原文
- 🗄️ **记忆库管理** - 查看、搜索、删除、导出记忆

#### 核心模块
- `core/memory-compressor.js` - 记忆压缩器
- `core/dna-generator.js` - DNA 追溯码生成器
- `core/taiji-extractor.js` - 太极算法变量提取器
- `core/storage-manager.js` - IndexedDB 存储管理器
- `core/memory-restorer.js` - 记忆还原器

#### 压缩效果

| 原始长度 | Medium 压缩后 | 压缩率 |
|---------|--------------|--------|
| 1,000 字 | 150 字 | 85% |
| 10,000 字 | 150 字 | 98.5% |
| 100,000 字 | 150 字 | 99.85% |

### 3. 资产扫描系统

#### 核心功能
- 🔍 **核心原创识别** - 通过"宝宝"、"UID9622"识别核心原创
- 📊 **多级分类** - 支持核心原创、原创、混合、外部、未标记五种分类
- 🎯 **完整关键词库** - 包含 40+ 个原创关键词
- 📈 **统计分析** - 实时统计各类资产数量和占比
- 📄 **多格式导出** - 支持 JSON 和 Markdown 格式导出报告
- 🔎 **关键词搜索** - 快速搜索包含特定关键词的文件

#### 核心模块
- `core/asset-scanner.js` - 资产扫描器

#### 分类标准

| 分类 | 识别标准 | 说明 |
|------|---------|------|
| ✅ 核心原创 | 包含"宝宝"、"UID9622"等 | 你的数字指纹 |
| 🟢 原创 | 包含其他原创关键词 | 使用了体系关键词 |
| 🟡 混合 | 同时包含原创和外部特征 | 在外部代码基础上修改 |
| 📦 外部 | 仅包含外部代码特征 | 完全使用外部代码 |
| ❓ 未标记 | 不包含任何关键词 | 无法确定来源 |

## 🚀 使用方法

### 1. 安装扩展

```bash
# 1. 准备图标文件
# 创建三个 PNG 图标（16x16、48x48、128x128）
# 放入 assets/icons/ 目录

# 2. 加载扩展
# 打开 chrome://extensions/
# 启用"开发者模式"
# 点击"加载已解压的扩展程序"
# 选择 dragon_soul_extension 文件夹
```

### 2. 使用记忆编辑器

```bash
# 打开记忆编辑器
open memory-editor.html

# 输入记忆内容
# 点击"处理记忆"
# 点击"保存到系统"
# 点击"还原"恢复原文
```

### 3. 使用资产扫描器

```bash
# 打开资产扫描器
open asset-scanner.html

# 拖拽文件到上传区域
# 点击"开始扫描"
# 查看扫描结果
# 导出报告
```
