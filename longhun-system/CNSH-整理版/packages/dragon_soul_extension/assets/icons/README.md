# 图标文件说明

1. **icon-16.png** - 16x16 像素
2. **icon-48.png** - 48x48 像素
3. **icon-128.png** - 128x128 像素

## 图标设计建议

- 主题：🐉 龙魂守护者
- 风格：简洁、现代、科技感
- 颜色：建议使用紫色/蓝色渐变
- 格式：PNG（支持透明背景）

### 方法 1：使用在线工具

- https://www.favicon-generator.org/

使用 Figma、Sketch、Photoshop 等软件创建

## 临时解决方案

在开发阶段，你可以：
1. 使用任何现有的 PNG 图片作为占位符
2. 重命名图片文件为上述名称
3. 确保尺寸匹配或接近

- 确保图标清晰可见
- 避免过于复杂的细节
- 在小尺寸下仍然可识别
