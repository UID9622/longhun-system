# 🚀 快速开始指南

## 三步上手

### 第一步：打开编辑器
双击 `编辑器/快速启动.html`

### 第二步：创作字元
1. 在画布上拖动鼠标绘制笔画
2. 调整力度、侵蚀、纹理等参数
3. 实时预览效果

### 第三步：保存作品
- 点击"保存作品"下载 .cnsh 文件
- 点击"导出 SVG"下载矢量图

## 推荐创作流程

1. **从简单字开始**：一、人、大
2. **多尝试参数**：不同力度和纹理效果差异很大
3. **保存中间版本**：用 .cnsh 格式保存可重新编辑
4. **导出最终作品**：用 SVG 格式分享或打印

## 进阶技巧

- **侵蚀效果**：调高侵蚀强度（0.5-1.0）可模拟古碑风化感
- **石刻纹理**：选择"石刻"纹理 + 高力度（20-30）更有质感
- **墨色浓淡**：调低墨色浓度（0.4-0.6）可做淡墨效果
