# CNSH Core 项目总结

## 🎯 项目概述

CNSH Core 是一个结合 Obsidian 和 Ollama 的国产本地AI知识管理系统，旨在为中国用户提供高效的知识管理和智能交互体验。项目符合国家数字主权战略，支持国产化部署和数字人民币集成方案。

## 🌟 核心特性

### 1. 知识管理
- **笔记同步**: 自动同步 Obsidian 笔记到 CNSH 系统
- **智能分类**: 基于内容的自动分类和标签系统
- **知识图谱**: 可视化笔记之间的关系
- **内容摘要**: 自动生成笔记摘要

### 2. 智能助手
- **本地对话**: 在 Obsidian 中与本地大模型对话
- **知识问答**: 基于知识库的问答系统
- **内容扩展**: 基于现有内容生成扩展信息
- **语义搜索**: 基于向量嵌入的文档相似性搜索

### 3. 国产化支持
- **鸿蒙兼容**: 支持华为鸿蒙操作系统
- **昇腾适配**: 兼容华为昇腾AI芯片
- **数字人民币**: 集成e-CNY支付场景
- **国产生态**: 完整的国产技术栈支持

### 4. 安全与隐私
- **本地部署**: 完全本地化，保障数据安全
- **端到端加密**: 数据传输全程加密
- **权限控制**: 细粒度的访问控制
- **合规保障**: 符合《数据安全法》要求

## 🚀 部署方案

### 1. 一键安装
- 运行 `./install.sh` 自动完成所有安装步骤
- 包括依赖安装、模型下载、配置设置等

### 2. 手动部署
- 详细的分步骤部署指南
- 支持多种操作系统和环境配置
- 灵活的自定义选项

### 3. 容器化部署
- Docker 支持
- 便于企业级部署和扩展
- 环境隔离和一致性保障

## 📊 技术架构

### 1. 前端架构
- **Obsidian 插件**: 基于 JavaScript 的插件系统
- **Web 界面**: React/Vue 支持的现代化界面
- **移动端**: 鸿蒙OS 原生应用支持

### 2. 后端架构
- **Node.js 服务器**: 基于 Express 的 API 服务
- **Python 集成**: 机器学习模型处理
- **数据库**: SQLite + 向量数据库

### 3. AI 集成
- **Ollama 本地模型**: 支持 qwen:7b-chat, chatglm3:6b
- **向量嵌入**: 文档向量化处理
- **智能推理**: 本地化推理引擎
