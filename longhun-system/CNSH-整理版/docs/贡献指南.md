## 🚀 快速开始

### 1. Fork 仓库

```bash
# 访问 https://github.com/UID9622/CNSH
# 点击 Fork 按钮
```

### 2. 克隆你的 Fork

```bash
git clone https://github.com/YOUR_USERNAME/CNSH.git
cd CNSH
```

### 3. 安装依赖

```bash
./scripts/install.sh
```

### 4. 创建分支

```bash
git checkout -b feature/你的功能名称
# 或
git checkout -b fix/修复的问题
```

### 代码规范

```javascript
// ✅ 推荐：使用中文语义变量（业务逻辑层）

    const 笔画粗细 = 渲染参数.力度;
    // ...
}

// ✅ 允许：英文变量（技术底层）
function renderStroke(strokeData, renderParams) {
    const strokeWidth = renderParams.force;
    // ...
}

// ❌ 不推荐：拼音命名
function xuanRanBiHua(shuJu, canShu) {
    // ...
}
```

function 计算贝塞尔曲线(点集, 张力) {

#### 类型说明

| 类型 | 用途 | 示例 |
|------|------|------|
| `feat` | 新功能 | `feat: 添加撤销/重做功能` |
| `fix` | 修复Bug | `fix: 修复笔画渲染偏移问题` |
| `docs` | 文档 | `docs: 更新安装指南` |
| `style` | 格式 | `style: 统一代码缩进` |
| `refactor` | 重构 | `refactor: 优化渲染管线` |
| `test` | 测试 | `test: 添加笔画测试用例` |
| `chore` | 构建 | `chore: 更新依赖版本` |

#### 示例

```
feat: 添加字元导出为PNG格式功能

- 支持透明背景
- 支持自定义分辨率
- 添加导出进度提示

DNA: #龍芯⚡️-CNSH-EDITOR-0023
```

## 🧪 测试要求

### 运行测试

```bash
# 运行所有测试
npm test

# 运行特定测试
npm test -- --grep "笔画渲染"

# 生成覆盖率报告
npm run test:coverage
```

### 测试覆盖率要求

| 模块 | 最低覆盖率 |
|------|-----------|
| 字元编辑器 | 70% |
| AI核心系统 | 80% |
| 工具函数 | 90% |

## 📝 PR 提交规范

### PR 标题格式

```
[类型] 简短描述

示例：
[feat] 添加撤销/重做功能
[fix] 修复笔画渲染偏移问题
[docs] 更新README安装说明
```

### PR 描述模板

```markdown
## 描述
简要描述这个PR做了什么

## 变更类型
- [ ] Bug修复
- [ ] 新功能
- [ ] 文档更新
- [ ] 性能优化
- [ ] 代码重构

## 测试
- [ ] 已通过本地测试
- [ ] 已添加新测试用例
- [ ] 所有测试通过

## 检查清单
- [ ] 代码遵循项目规范
- [ ] 已更新相关文档
- [ ] 已分配DNA编码

## DNA编码
#龍芯⚡️-CNSH-XXX-XXXX
```

## 🎨 设计贡献

### UI/UX 设计

1. **Fork 仓库**
2. **在 `design/` 目录下添加设计稿**
3. **提交 PR**

### 字元创作

1. **使用字元编辑器创作汉字**
2. **导出 .cnsh 和 SVG 文件**
3. **提交到 `examples/glyphs/` 目录**
4. **在 PR 中说明创作意图**
