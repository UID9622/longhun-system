## 📖 **详细导出步骤**

### 1️⃣ **ChatGPT 数据导出**

**步骤**：
1. 访问 https://chat.openai.com
2. 点击左下角账户图标 → Settings
3. 进入 Data controls 选项卡
4. 点击 "Export data"
5. 等待邮件通知（通常24小时内）
6. 点击邮件中的下载链接
7. 下载 ZIP 文件并解压
8. 找到 `conversations.json` 文件

**文件位置**：
```
chatgpt_export/
├── conversations.json  # 主要对话记录
├── message_feedback.json
├── user.json
└── ...
```

**使用工具**：
```bash
python ai_chat_extractor.py
# 选择 1. 提取对话记录
# 选择 1. ChatGPT
# 输入: /path/to/chatgpt_export/conversations.json
```

---

### 2️⃣ **智谱清言 数据导出**

**步骤**：
1. 打开智谱清言App或网页版
2. 进入"我的" → 设置
3. 找到"数据导出"或"对话记录导出"
4. 选择导出格式为JSON
5. 填写邮箱接收导出文件
6. 下载导出文件

**文件格式**：
```json
{
  "conversations": [
    {
      "title": "对话标题",
      "messages": [
        {
          "id": "消息ID",
          "content": "用户消息",
          "createTime": "2024-01-01T12:00:00Z"
        },
        {
          "id": "消息ID",
          "content": "AI回复",
          "createTime": "2024-01-01T12:01:00Z"
        }
      ]
    }
  ]
}
```

---

### 3️⃣ **千问 数据导出**

**步骤**：
1. 访问 https://qianwen.aliyun.com
2. 进入个人中心 → 数据管理
3. 选择"对话记录" → "导出"
4. 选择导出格式为JSON
5. 设置时间范围（可选）
6. 确认导出并等待邮件
7. 下载导出文件

**注意事项**：
- 千问可能限制导出时间范围
- 建议每季度导出一次
- 大量数据可能分批导出

---

### 4️⃣ **腾讯元宝 数据导出**

**步骤**：
1. 打开腾讯元宝App
2. 进入"我的" → 设置
3. 选择"隐私设置" → "数据管理"
4. 点击"导出对话记录"
5. 填写接收邮箱
6. 选择导出时间范围
7. 等待审核和发送

**特别说明**：
- 腾讯元宝可能需要实名认证才能导出
- 导出申请可能需要人工审核
- 建议提前准备，避免急需时无法导出

---

### 5️⃣ **DeepSeek 数据导出**

**步骤**：
1. 访问 https://chat.deepseek.com
2. 进入个人设置
3. 选择"数据导出"
4. 选择包含对话记录
5. 设置导出格式为JSON
6. 确认导出并等待邮件
7. 下载导出文件

**文件结构**：
```
deepseek_export/
├── conversations.json
├── user_settings.json
└── export_metadata.json
```

## 🚨 **常见问题**

### ❓ **导出失败怎么办？**
1. 检查文件路径是否正确
2. 确认文件格式是否为JSON
3. 尝试重新导出原始数据

### ❓ **DNA码生成失败？**
1. 检查DNA生成器是否正常
2. 确认内容描述是否合规
3. 尝试手动生成DNA码

### ❓ **数据量太大？**
1. 分批处理，每次处理一个月
2. 使用过滤功能，只提取重要对话
3. 增加磁盘空间
