# CNSH Core 更新日志

## [1.0.0] - 2024-12-10

### 🎉 初始发布

- **核心功能**：
  - ✅ Obsidian 与 Ollama 无缝集成
  - ✅ 本地大模型知识管理系统
  - ✅ 向量数据库支持
  - ✅ RESTful API 服务
  - ✅ 智能助手功能

- **国产化支持**：
  - ✅ 华为鸿蒙系统兼容
  - ✅ 昇腾芯片适配
  - ✅ 数字人民币集成方案
  - ✅ 国产化技术栈支持

- **部署工具**：
  - ✅ 一键安装脚本
  - ✅ 详细部署文档
  - ✅ 故障排除指南
  - ✅ GVP 申请指南

- **安全特性**：
  - ✅ 本地化部署保障数据安全
  - ✅ 符合《数据安全法》要求
  - ✅ 端到端加密
  - ✅ 国产加密算法集成

### 📋 已知问题

1. 首次安装时可能需要手动配置某些环境变量
2. 大型知识库向量化处理可能需要较长时间
3. 部分国产芯片适配仍在优化中

### 🔧 技术规格

- **前端**：Node.js + Express
- **AI引擎**：Ollama (支持 qwen:7b-chat, chatglm3:6b)
- **数据库**：SQLite + 向量数据库
- **插件**：Obsidian 插件生态
- **兼容平台**：Windows, macOS, Linux, 鸿蒙OS
