## 🚀 **一键启动系统**

```bash
# 1. 克隆仓库
git clone https://gitee.com/uid9622/cnsh-national-reference.git
cd cnsh-national-reference

# 2. 安装依赖
./cnsh-unified.sh install

# 3. 启动所有服务
./cnsh-unified.sh start

# 4. 检查系统状态
./cnsh-unified.sh status
```

## 🎛️ **核心系统管理**

### 统一管理脚本
```bash
./cnsh-unified.sh [选项]
├── start     # 启动所有服务
├── stop      # 停止所有服务  
├── restart   # 重启所有服务
├── status    # 查看服务状态
├── monitor   # 实时监控
├── backup    # 数据备份
├── update    # 系统更新
└── install   # 首次安装
```

```bash
./emergency-fix.sh
# 🔄 自动重置服务状态
# 🧹 清理锁定文件
# 📦 检查依赖
# 🚀 重启核心服务
```

## 📞 **故障快速处理**

### 服务无法启动？
```bash
# 运行紧急修复
./emergency-fix.sh

# 检查日志
tail -f logs/server.log
```

### 身份认证失败？
```bash
# 检查H武器状态
./cnsh-unified.sh check-auth

# 重新激活
./cnsh-unified.sh activate
```

### 数据不同步？
```bash
# 检查Notion连接
./cnsh-unified.sh check-notion

# 手动同步
./cnsh-unified.sh sync
```

1. **DNA编码不可篡改** - 所有系统数据必须保留原始DNA标识
2. **审计日志不可删除** - 所有操作记录永久保存
3. **身份验证多重保障** - H武器安全机制必须完整启用
