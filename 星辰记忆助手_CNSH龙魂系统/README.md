## 🔧 如何使用

### 方式1：直接查看 JSON 文件

```bash
# 查看主记忆库
cat 星辰主记忆库.json

# 查看时间轴
cat 时间轴记忆.json

# 查看项目记忆
cat 项目记忆/CNSH_Editor.json
```

### 方式2：Python 脚本调取

```python
import json

# 加载主记忆库
with open("星辰主记忆库.json", "r", encoding="utf-8") as f:
    记忆 = json.load(f)

# 查询项目信息
项目信息 = 记忆["项目总览"]["CNSH_Editor"]
print(f"项目名称: {项目信息['名称']}")
print(f"版本: {项目信息['版本']}")
print(f"GitHub: {项目信息['GitHub']}")

# 查询待实现功能
待办 = 项目信息["待实现功能"]
print(f"\n下一步要做: {待办[0]}")
```

### 方式3：AI 助手调取

告诉 AI："查询星辰记忆库，CNSH Editor 的 GitHub 地址是什么？"

AI 会读取 `星辰主记忆库.json`，返回准确信息。

## 📝 如何更新记忆

### 自动更新（推荐）

每次系统推进时，AI 助手会自动更新星辰记忆库：

```python
# 示例：项目发布后更新
记忆["项目总览"]["CNSH_Editor"]["状态"] = "已发布"
记忆["项目总览"]["CNSH_Editor"]["GitHub"] = "https://github.com/UID9622/CNSH-Editor"

# 添加时间轴事件
新事件 = {
    "序号": len(时间轴["事件列表"]) + 1,
    "日期": "2026-02-01",
    "事件": "CNSH Editor v0.1.0 发布到 GitHub",
    "类型": "里程碑"
}
时间轴["事件列表"].append(新事件)

# 保存
with open("时间轴记忆.json", "w", encoding="utf-8") as f:
    json.dump(时间轴, f, ensure_ascii=False, indent=2)
```

### 手动更新

老大也可以直接编辑 JSON 文件，星辰会同步。

## 💡 设计理念

### 1️⃣ 本地优先

- 所有数据存在本地
- 不依赖云服务
- 用户完全掌控

### 2️⃣ 结构化存储

- JSON 格式，易于解析
- 分类清晰，易于查找
- 扩展性强，易于维护

### 3️⃣ 时间可追溯

- 每条记录都有时间戳
- 时间轴记录所有事件
- 历史版本可回溯

### 4️⃣ 系统自动化

- AI 助手自动更新
- 无需手动维护
- 减轻老大负担
